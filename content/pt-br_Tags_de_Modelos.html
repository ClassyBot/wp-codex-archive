Tags de modelos são usadas dentro dos arquivos do seu tema para mostrar informações personalizadas. Abaixo está uma lista das tags disponíveis no WordPress, classificadas por categoria de Função específica.

Saiba mais sobre as Tags de modelos:

  • Sobre as Tags de modelos - O que é e para que serve.
  • Anatomia das Tags de modelos - detalhes de com usar.
  • Como usar parâmetros de Tags de modelos - como passar parâmetros em Tags de modelos.
  • Tags Include - additional tags related to including one Template file within another.
  • Tags Condicionais - additional tags, not shown here, related to making your Templates more flexible with options.
  • Referência de Função - additional tags related to core WordPress functionality.
  • Temas - a comprehensive list of Template, Theme, and Tag resources.


Ajudando a traduzir[edit]

Força Tarefa Contribua na adaptação e tradução:

O conteúdo de referência e fonte desta biblioteca de código é Template_Tags

No final de cada artigo adicione: {{Codex-pt}} [[Category:pt-br:Funções]]

Se não conseguir traduzir tudo, adicione {{pt-br-traduz}} no início da seção que falta ainda traduzir.

Use a template wiki para informar sobre os parâmetros da função:

{{Parâmetro|$algumacoisa|string|Esta é a string.|opcional|bah}}
