Опис[edit]

Ја вклучува датотеката footer.php од вашата тема. Доколку постои посебно име за подножјето, пример footer-{name}.php, истото ќе биде вклучено.

Доколку темата не содржи footer.php датотека, тогаш ќе биде вметнато подножјето од стандардната тема кое се наоѓа во wp-includes/theme-compat/footer.php.

Употреба[edit]

<?php get_footer( $name ); ?>

Параметри[edit]

$name
(string) (optional) Повикува footer-name.php.
Default: None

Примери[edit]

Едноставна 404 страници[edit]

Кодот подолу е едноставен пример за нацрт за 404 грешка кој треба да биде вметнат во 404.php датотеката на вашата тема. 

<?php get_header(); ?>
<h2>Грешка 404 - Страницата не е пронајдена</h2>
<?php get_sidebar(); ?>
<?php get_footer(); ?>

Повеќе подножја[edit]

Различно подножје за различни страници. 

<?php
if ( is_home() ) :
	get_footer( 'home' );
elseif ( is_404() ) :
	get_footer( '404' );
else :
	get_footer();
endif;
?>

Имињата на заглавјата за почетната и 404 страницата треба да се наоѓаат во footer-home.php и footer-404.php, соодветно.

Забелешки[edit]

  • Користи: locate_template()
  • Користи: do_action() Ја повикува 'get_footer' акцијата.

Лог на промени[edit]

  • Од: 1.5.0
  • 2.7.0 : Додаден параметарот 'name'.

Изворна датотека[edit]

get_footer() се наоѓа во wp-includes/general-template.php.
