Server vereisten[edit]

Hieronder staan de vereisten voor een server om WordPress te kunnen hosten. Deze worden automatisch bijgewerkt en zijn daarom in het Engels.

  • WordPress server requirements for Version 3.2+:
    • PHP version 5.2.4 or greater
    • MySQL version 5.0.15 or greater or any version of MariaDB
    • (Optional) (Required for Multisite) Apache mod_rewrite module (for clean URIs known as Permalinks)

» Requirements for Older Versions

Aanbevolen configuratie[edit]

Het is aanbevolen om een robuust platform bestaande uit het Linux besturingssysteem en de Apache webserver of de NGINX webserver. In feite werkt elke server die PHP en MySQL/MariaDB ondersteunt. Wanneer je host één van deze systemen en mod_rewrite niet ondersteunt, dan is het misschien beter om naar een andere hosting provider over te schakelen die de vereisten wel aanbieden.

Het is daarnaast ook belangrijk dat jouw host verbindingen van andere sites toestaat zodat alle WordPress functies naar behoren werken. Als jouw host uitgaande HTTP verbindingen blokkeert zullen een aantal functies van WordPress niet werken.

Aanbevolen serverconfiguratie:

  • PHP versie 5.6 of hoger
  • MySQL versie 5.6 / MariaDB versie 10.0 of hoger

Er kunnen nog andere zaken meetellen, bekijk voor de technische details de technical Installation Checklist (Engels).

Tegenwoordig zijn er veel hosting providers, met een beetje moeite is het gemakkelijk om een webhost te vinden die aan de vereisten voldoet. Als je suggesties zoekt, bekijk dit eens:

  • WordPress Recommended Hosts (Engels) - Deze webhosts ondersteunen WordPress op meerdere manieren.
  • Doorzoek de WordPress fora for aanbevelingen.
