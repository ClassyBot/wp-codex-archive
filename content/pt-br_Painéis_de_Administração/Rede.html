O Painel de Administração oferece acesso aos recursos do controle de sua instalação do WordPress. Cada painel de administração é apresentado em seções, o cabeçalho, a navegação principal, a área de trabalho e o rodapé. A parte superior de todos os os painéis, o cabeçalho, estám em destaque na tonalidade clara. O cabeçalho mostra o nome do seu blog que é também um link para o seu blog, pode mostrar a mensagem de motores de busca bloqueados se as configurações de privacidade assim estiver definida, links para Seu perfil (mostrada como seu nome de usuário) e link Sair. Logo abaixo do topo da área sombreada há duas abas com Opções de Tela e Ajuda contextual Ajuda, que podem ser clicadas para expandi-las.

Do lado esquerdo da tela há o Menu de Navegação logo mais, há detalhes de uma das funções administrativas que você pode executar. Duas setas expandir/recolher logo abaixo De Painel e Comentários permitem que o menu de navegação se recolha em um conjunto de ícones, enquanto se expandi-los aparece o ícone e rótulo para cada uma das principais funções administrativas. Dentro de cada uma das funções importantes, como Postagens, uma seta para baixo é apresentada se a parar o cursor do mouse sobre a barra de título. Clique sobre a seta que se expande para mostrar cada uma das opções de sub-menu. Clicando nessa seta, o sub-menu é recolhido.

A grande área no centro da tela é Área de trabalho. Aqui as informações específicas relativas a uma escolha particular de navegação, como a adição de uma nova postagem, é apresentada.

Finalmente, no Rodapé, na parte inferior de cada painel de administração, há links para o WordPress, Documentação e Feedback. Além disso, você pode ver a Versão do WordPress que está usando. Se sua versão não for a masi recente, aparecerá uma mensagem Uma atualização do WordPress está disponível. Atualize agora. Clique no link para ir até o painelAtualizações.

Cada painel é são acessado através do Menu Principal de navegação. Veja sobre cada um deles no quadro abaixo. Os links nessas caixas vão leva-lo para as seções deste artigo descrevendo brevemente cada painel.
