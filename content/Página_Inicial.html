O software WordPress é uma poderosa plataforma de publicação pessoal que vem com um grande conjunto de recursos para tornar sua experiência de publicação, fácil, agradável e o mais atrativa possível.

Este é o mesmo software usado no serviço de blogs wordpress.com e sua prinpicial diferença entre o serviço de blogs é que você é livre para modificar o software e todos os seus componentes livremente.

Para usar o software WordPress, você precisa de um serviço de hospedagem que tenha suporte a ele, alguns até têm recursos de instalação automática, mas se não tiver, não há o que se preocupar: é muito fácil intalá-lo.

Após ter seu WordPress instalado e funcionando, é hora de criar o conteúdo do seu site. Você vai descobrir como a maioria dos recursos de edição já estão disponível no WordPress.

Você pode inserir imagens e ajustá-las em seu texto, mostrar uma galeria de miniaturas dentro do post ou página, incorporar vídeos, criar páginas estáticas, usar widgets para inserir componentes extras ao site e muitos outros recursos.

Você pode deixar seu site com o visual e os recursos que quiser através de Temas e Plugins.

O Diretório de Plugins WordPress contém mais de 10.000 plugins gratuitos para adicionar os mais diferentes recursos ao seu site como sistemas de votação, avaliação, slider de imagens, chat em tempo real, mecanismos de busca avançado e tudo o mais que imaginar.

O Diretório de Temas WordPress contém somente temas gratuitos testados e aprovados por um time de especialistas, para garantir que seu site não só seja bonito mas também funcional e seguro.

E se você já está a passo alé, poderá criar seus próprios temas e plugins, estendo o uso do WordPress para o que quiser.

A maioria dos problemas encontrados no WordPress são relacionados ao servidor, é comum os servidores limitarem o uso de memória ao PHP ou não possuírem versões mais recentes de seus sistemas.

Para estes e outros casos ocasionais, consulte os artigos com Perguntas e Respostas comuns e se isso não ajudar, entre no Fórum de Suporte Oficial, sempre tem alguém disponível para ajudá-lo, não se esqueça de ler as Regras do Fórum.
