Descrição[edit]

Inclui o arquivo de template header.php do diretório do seu tema atual. Se um nome for definido, então um cabeçalho especial header-{name}.php será incluído.

Se não existir o arquivo header.php no diretório do seu tema, então um cabeçalho do tema padrão wp-includes/theme-compat/header.php será incluído..

Uso[edit]

<?php get_header( $name ); ?>

Parâmetros[edit]

$name
(string) (opcional) Chama o header-name.php.
Default: None

Exemplos[edit]

Página de erro 404[edit]

O código a seguir é um exemplo de template para uma página de erro 404 "HTTP 404: Página não encontrada" (a qual você pode incluir em seu Tema como 404.php). 

<?php get_header(); ?>
<h2>Error 404 - Página não encontrada</h2>
<?php get_sidebar(); ?>
<?php get_footer(); ?>

Múltiplos cabeçalhos[edit]

Diferentes cabeçalhos para diferentes páginas. 

<?php
if ( is_home() ) :
	get_header( 'home' );
elseif ( is_404() ) :
	get_header( '404' );
else :
	get_header();
endif;
?>


O nome dos arquivos para a Home (página inicial) e 404 (página de erro) devem ser header-home.php e header-404.php respectivamente.

Notas[edit]

  • Utiliza: locate_template()
  • Utiliza: do_action() Chama a ação 'get_header'.
