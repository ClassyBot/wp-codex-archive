Artigo ou Seção com explicações avançadas
A seguir, estão explicações ou instruções para usuários avançados, o que pode exigir o conhecimento de termos e ferramentas não comuns a todas as pessoas.
Adicione {{Avançado}} para usar esta caixa.

A API XML-RPC API que agora substitui o legado as APIs do Blogger, MovableType e metaWeblog.

Componentes[edit]

  • Posts (para posts, páginas e tipos de posts) - Adicionado no WordPress 3.4
    • wp.getPost
    • wp.getPosts
    • wp.newPost
    • wp.editPost
    • wp.deletePost
    • wp.getPostType
    • wp.getPostTypes
    • wp.getPostFormats
    • wp.getPostStatusList
  • Taxonomias (para categorias, tags e taxonomias personalizadas) - Adicionado no WordPress 3.4
    • wp.getTaxonomy
    • wp.getTaxonomies
    • wp.getTerm
    • wp.getTerms
    • wp.newTerm
    • wp.editTerm
    • wp.deleteTerm
  • Mídia - Adicionado no WordPress 3.1
    • wp.getMediaItem
    • wp.getMediaLibrary
    • wp.uploadFile
  • Comentários - Adicionado no WordPress 2.7
    • wp.getCommentCount
    • wp.getComment
    • wp.getComments
    • wp.newComment
    • wp.editComment
    • wp.deleteComment
    • wp.getCommentStatusList
  • Options - Adicionado no WordPress 2.6
    • wp.getOptions
    • wp.setOptions
  • Usuários
    • wp.getUsersBlogs
    • wp.getAuthors

Componentes Obsoletos[edit]

  • Categorias - agora usa-se Taxonomias, with taxonomy_name='category'
    • wp.getCategories
    • wp.suggestCategories
    • wp.newCategory
    • wp.deleteCategory
  • Tags - agora usa-se Taxonomies, com taxonomy_name='post_tag'
    • wp.getTags
  • Pages - agora usa-sePosts, compost_type='page'
    • wp.getPage
    • wp.getPages
    • wp.getPageList
    • wp.newPage
    • wp.editPage
    • wp.deletePage
    • wp.getPageStatusList
    • wp.getPageTemplates
