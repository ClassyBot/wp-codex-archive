Resumo ou Excertos no WordPress é um sumário opcional que você mesmo cria, para cada postagem.

Os resumos tem utilidade, além de introduzir o assunto da postagem:

  1. Substitua o contéudo completo nos feeds RSS quando marcada a opção de mostrar resumos estiver marcada em Configurações > Leitura.
  2. Dependendo do tema, pode ser mostrado em diversos contextos, como consultas específicas (queries) e outras listagens como:
    • Resultados da Procura
    • Arquivo de Tags
    • Arquivo de Categorias
    • Arquivos Mensal
    • Arquivos do Autor

Adicionando Resumos às Postagens[edit]

Para adicionar um resumo de uma postagem, basta escrever uma no campo Resumo após a caixa de edição. Um trecho pode ser curto ou longo, como você deseja. Normalmente, dada a sua finalidade, algumas frases é suficiente.

Curiosamente, desde os resumos no WordPress tem um propósito semelhante à descrição META de documentos (X)HTML, resumos podem ainda ser utilizado como meta descrições também. Alguns temas fazem isso por padrão. Também pode ser feito por meio de um plugin SEO ou um plugin para gerenciar dados no cabeçalho de páginas (X)HTML.

Só é possível adicionar resumos da postagem na tela de edição, ou seja aquela quando vamos em Posts > Adicionar Novo, ou quando clicamos para re-editar uma postagem.

Logo abaixo da área de edição, por padrão aparece a caixa Resumo:

A caixa para o resumo


Informe um resumo da postagem, por exemplo, sobre o que fala. Você pode usar tags HTML como negrito, itálico, inserir imagens etc.

Mostrando Resumos[edit]

No arquivo do tema, você deve adicionar: 

<?php the_excerpt(); ?>

Opcionalmente, pode adicionar condição para quando não haver resumos: 


<?php  if (has_excerpt() ) { ?>
<?php the_excerpt(); ?>
<?php } ?>

Saiba mais sobre a função the_excerpt e has_excerpt.

Resumo, Resumo automático, Teaser e Trecho[edit]

O resumo manual é muitas vezes confundido com resumo automático ou Teaser (a parte de um post que aparece na primeira página quando você usar a tag <!--more-->). Embora ambos sejam relacionados com o trecho manual, eles são diferentes entre si.

A relação entre Resumo, Resumo automático e Teaser é a seguinte: Quando um post não tem nenhum resumo manual e o modelo usa a tag Template Tags/the_excerpt, o WordPress gera um resumo automaticamente, selecionando as primeiras 55 palavras da postagem. Quando o modelo usa a tag Template Tags/the_content, o WordPress irá procurar a marca e criar mais teaser do conteúdo que precede a tag <!--more-->

Há plugins que podem obter um trecho da postagem para usá-lo como resumo, trecho, é justamente isso: um trecho extraído da postagem.

Outras diferenças

Você já deve ter reparado que o resumo manual pode ter qualquer tipo de texto, imagens etc, ou seja, é um texto introdutório livre, para chamar a atenção e introduzir o assunto da postagem ao visitante. Veja os exemplos:

Um exemplo de resumo manual e resumo automático: 

Esta é uma história muito conhecida entre as crianças de todo o mundo.
 Era uma vez uma menina que se chamava Chapeuzinho Vermelho, um dia Leia mais...

Um exemplo de resumo usando the_excerpt(): 

Era uma vez uma menina que se chamava Chapeuzinho Vermelho, um dia [...]

Um exemplo de Teaser (avançado): 

 Era uma vez uma menina que se chamava Chapeuzinho Vermelho, um dia...

Um exemplo de Trecho (através de plugins): 

 Chapeuzinho Vermelho foi pela floresta, cantando feliz pela floresta(...)
