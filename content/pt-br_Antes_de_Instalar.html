Visite o site da Comunidade brasileira WordPress: http://wp-brasil.org/

Antes de instalar WordPress, é preciso verificar se o seu servidor oferece todas as condições necessárias (a maioria oferece mas é bom verificar!) e que você também tenha alguns conhecimentos básicos necessários.

Para uma lista de verificação do que é necessário ser oferecido pelo servidor, veja Hospedando o WordPress. Outra lista é Página de Requerimentos Oficial.

Com foi dito, você também precisa ter algum conhecimento. Isso quer dizer, um conhecimento básico de como usar um editor de texto para editar o arquivo de configuração, por exemplo. Se você é um usuário Windows, O Bloco de Notas já é suficiente.

Mais tarde, você pode precisar editar os arquivos do tema do seu site o que pode ser através do editor do WordPress Painéis de Administração, mas pode ser melhor usar um editor com mais funcionalidades. Veja uma lista de editores de texto que pode usar.

Você precisa saber como usar um Cliente FTPpara enviar arquivos e mudar suas permissões de acesso. Um aplicativo gratuito para isso é o FileZilla.

E é isso! Agora que está tudo devidamente verificado, é só partir para a Instalação.
