Les marqueurs de modèle sont utilisés par les Modèles (en anglais) de votre blog pour afficher dynamiquement des informations ou pour le personnaliser, fournissant les outils pour le rendre aussi original et intéressant que vous l'êtes. Ci-dessous figure la liste des marqueurs généralistes disponibles pour les utilisateurs dans WordPress, triés par type d'utilisation.

Pour plus d'informations sur les marqueurs de modèle et les modèles en général, consultez les articles suivants :

  • Découvrir les Marqueurs de Modèle (une introduction aux Marqueurs de Modèle) (en anglais)
  • Anatomie d'un Marqueur de Modèle (détails sur la manière d'insérer des marqueurs dans vos fichiers de Modèle) (en anglais)
  • Comment Passer des Paramètres à un Marqueur ? (détails sur la manière d'utiliser des Paramètres avec les Marqueurs) (en anglais)
  • Marqueurs d'Inclusion (Marqueurs additionnels non listés ici, en relation avec l'inclusion d'un fichier de Modèle avec un autre) (en anglais)
  • Marqueurs Conditionnels (d'autres Marqueurs additionnels non listés ici, permettant de rendre les Modèles plus flexibles avec des options) (en anglais)
  • Modèles (une liste très fournie de ressources sur les Modèles, Thèmes et Marqueurs) (en anglais)
  • Découvrir les Modèles (une introduction aux fichiers de Modèle) (en anglais)

Note : Si vous connaissez une méthode intéressante concernant l'intégration d'un ou plusieurs des marqueurs, n'hésitez pas à ajouter des exemples pour que les autres utilisateurs WordPress puissent en profiter.

Note : Certains de ces marqueurs sont apparus avec des versions spécifiques de WordPress (et des annotations comme, par exemple, "Version 2.1" sont présentes dans la liste ci-dessous). Certains marqueurs ont été rendus obsolètes dans les versions 2.0.x (et la mention "Obsolète" figure ci-dessous). Certains marqueurs ont été rendus obsolètes dans la version 2.1 (et sont donc annotés avec la mention "Obsolète dans la version 2.1" ci-dessous).
