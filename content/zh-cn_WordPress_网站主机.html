大多数主机服务商和建立在Unix/Linux系统之上的个人服务器，在最一般的配置下就可以作为WordPress主机。

服务器要求[edit]

Version 3.2 (Spring 2011) WordPress服务器要求:

  • PHP 5.2 或更高版本
  • MySQL 5.0.15 或更高版本
  • (可选 (多站点要求)) Apache mod_rewrite 模块 (用于段网址，如众所周知的 Permalinks)

Version 2.9, 3.0, 3.1 WordPress服务器要求:

  • PHP 4.3 或更高版本
  • MySQL 4.1.2 或更高版本
  • (可选) Apache mod_rewrite module (作为我们所熟知的简洁链接Permalinks使用)

Version 2.5, 2.6, 2.7, 2.8 WordPress服务器要求:

  • PHP 4.3 或更高版本
  • MySQL 4.0 或更高版本
  • (可选) Apache mod_rewrite module (作为我们所熟知的简洁链接Permalinks使用)

Versions 2.1, 2.2, 2.3 WordPress服务器要求:

  • PHP 4.2 或更高版本
  • MySQL 4.0 或更高版本
  • (可选) Apache mod_rewrite module (作为我们所熟知的简洁链接Permalinks使用)

Version 2.0 WordPress服务器要求:

  • PHP 4.2 或更高版本
  • MySQL 3.23.23 或更高版本
  • (可选) Apache mod_rewrite module (作为我们所熟知的简洁链接Permalinks使用)



推荐的设置[edit]

我们推荐在Linux上搭配Apache或者NGINX的Web服务器作为强大的平台来运行WordPress，但是其他任何支持 PHP 和MySQL 的服务器也是可以的。如果你的主机不支持以上所列要求之一和mod_rewrite, 你最好还是换一个支持以上选项的主机服务商吧

还有一项基本要求是你的主机应该支持远程连接。

推荐的服务器设置:

  • PHP 5.2 或更高版本
  • MySQL 5.0.15 或更高版本

其他的要求，详情请看安装技术清单（英文版）.

主机服务商[edit]

主机现在很普遍了，你可以很轻易的找到支持以上要求的主机。如果需要给你推荐主机，那可以试试：

  • WordPress 推荐主机 - 这些主机支持WordPress。
  • 在WordPress 支持论坛 中搜索推荐主机
