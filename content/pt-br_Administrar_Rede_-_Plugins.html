Administrar Rede → Plugins[edit]

O painel Plugins na administração da rede mostra quais os plugins disponíveis na rede.

Os sites de uma rede WordPress não podem instalar plugins e temas sozinhos, todo plugin e tema usado pelos sites da rede devem ser instalados no site principal e então ativos para uso.


Opções de Tela[edit]

Clique na aba Opções de Tela para definir o número de plugins a se listar na tabela além de mostrar ou não a descrição.

Ajuda[edit]

Clique nesta aba para uma descrição deste painel.

Plugins da Rede[edit]

Esta tabela lista todos os plugins, um por linha. Os plugins são listados por ordem alfabética do nome da pasta usada pelo plugin. A tabela de plugins contém as seguintes colunas:

Plugins
O nome do plugin.
Descrição
Uma descrição do plugin, como descrito pelo autor.

Ao mover o cursor do mouse sobre um item, os seguintes links de ação aparecem:

  • Ativar/Desativar na Rede - Ative ou desative o plugin para ser usado em toda a rede.
  • Editar - Leva para uma área de texto onde se pode editar o código plugin.
  • Excluir - Exclui o plugin. Se estiver em uso, desative antes para excluir.
