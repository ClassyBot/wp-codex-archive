设置 → 常规[edit]

此设置 常规页面为管理面板的默认界面。一些您站点的基本属性都设置于此，如：您站点的标题和位置、允许何人注册、日期和时间如何计算并显示等。

返回管理页面。

常规设置[edit]

站点标题
在此输入您站点（或博客）的标题。绝大多数主题将在页面上方或读者浏览器标题栏显示该标题。 WordPress 还会将此标题用作您 Syndication feeds 的识别名称（identifying name）。
标语
用简短的话语解释您站点的主题。您站点的口号或标语可输入于此。标语是简短的词组或句子，它可传达站点的主旨，且常常很有趣、很显眼。
WordPress 地址 (URL)
输入包含您 WordPress 核心程序文件（例如：wp-config.php、wp-admin、wp-content 以及 wp-includes）目录的完整 URL。例如，若您将 WordPress 装在一个叫 "blog"的目录下，那么 WordPress 地址就是 http://example.net/blog （其中 example.net 是您的域名）。如果您将 WordPress 安装在网站的根目录，这个地址将会是根目录 URL http://example.net。WordPress 将会从末尾删掉一个斜线(/)。
站点地址 (URL)
输入您希望人们从浏览器中访问您 WordPress 站点的地址。此地址应是 WordPress index.php 文件的安装目录。除非您指定 WordPress 到单独目录，站点 (URL) 应与 WordPress 地址 (URL) （见上）相同。WordPress 将会从末尾删掉一个斜线(/)。
E-mail 地址
输入您希望 WordPress 发送给您有关管理维护消息的 e-mail 地址。例如，若您允许新用户注册成为您的站点成员（下见 Membership ）, then a notification will be sent through e-mail to this address. In addition, if the option, An administrator must always approve the comment, is set in Administration > Settings > Discussion, this e-mail address will receive notification that the comment is being held for moderation. Please note this is different than the address you supplied for the admin user account; the admin account e-mail address is sent an e-mail only when someone submits a comment to a post by admin. The address you enter here will never be displayed on the site.
成员身份
  • 任何人均可注册 - 如果您想让任何人都可以在您的站点上注册一个账号，请选中该复选框。
新添用户默认角色
此下拉菜单允许您为新用户选择一个默认的角色，此默认角色将会分配给新注册的成员以及通过管理 > 用户 > 用户页添加的用户。有效地选择是管理员, 编写者, 作者, 投稿者, 或者订阅用户.
时区 
From the pulldown box, choose a city in the same timezone as you. For example, under America, select New York if you reside in the Eastern Timezone of the United States that honors daylight savings times. If you can't identify a city in your timezone, select one of the Etc GMT settings that represents the number of hours by which your time differs from Greenwich Mean Time. Click the Save Changes button and the UTC time and "Local time" will display to confirm the correct Timezone was selected.
日期格式
The format in which to display dates on your site. The Date Format setting is intended to be used by theme designers in displaying dates on your site, but does not control how the date is displayed in the Administrative Screens (e.g. Manage Posts). Click the Save Changes button and the "Output" example below will show the current date in the format entered. See Formatting Date and Time for some of the formats available.
时间格式
此选项将影响到您的网站以何种方式显示显示时间. 时间格式将会通过您所使用的主题显示在您的网站上，but does not control how the time is displayed in the Administrative Screens (e.g. Write Post edit of timestamp). Click the Save Changes button and the "Output" example below will show the current time in the format entered. See Formatting Date and Time for some of the formats available.
一周开始于
在WordPress的日历下拉框中，选择您喜欢的起始日期. 默认情况下系统定义为星期一,这意味着日历表中的第一列将会显示星期一. 如果您想让日历的第一列显示为星期日,请在下拉菜单中选择星期日.

保存更改[edit]

请点击保存更改，以确保设置选项均已存入数据库。保存完成后，页面顶端会出现确认框。
