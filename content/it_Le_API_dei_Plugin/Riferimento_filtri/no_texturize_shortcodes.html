Descrizione[edit]

Il filtro 'no_texturize_shortcodes' permette di specificare quali shortcode non devono essere eseguiti attraverso la funzione wptexturize().

Per impostazione predefinita, WordPress esegue automaticamente la funzione wptexturize() per tutti i contenuti di articoli e pagine, anche per il contenuto negli shortcode. Questo processo sostituisce le virgolette "normali" con virgolette "tipografiche". Non sempre è ciò che si vuole... in particolare se lo shortcode deve contenere testo grezzo, pre-elaborato.

Utilizzo[edit]

Questo filtro espone un array che contiene i nomi degli shortcode esenti dall'applicazione della funzione wptexturize(). Per usare questo filtro, basta aggiungere un nuovo elemento all'array per ogni shortcode che si intende preservare.

Parametri[edit]

$shortcodes
(array) (richiesto) Un array di shortcode esenti dal processo descritto
Default: Nessuno

Esempi[edit]

add_filter( 'no_texturize_shortcodes', 'shortcode_da_preservare' );
function shortcode_da_preservare($shortcodes){
    $shortcodes[] = 'myshortcode';
    return $shortcodes;
}
