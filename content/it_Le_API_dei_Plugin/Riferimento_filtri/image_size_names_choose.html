Descrizione[edit]

Il filtro image_size_names_choose permette di modificare della lista di dimensioni per le immagini disponibili agli amministratori nella Media Library di WordPress.

Si usa per rendere disponibili e selezionare dimensioni personalizzate per le immagini nel pannello di amministrazione di WordPress.

Utilizzo[edit]

Il codice seguente aggiunge una nuova opzione alla lista di dimensioni selezionabili nella Media Library. 

add_filter( 'image_size_names_choose', 'my_custom_sizes' );

function my_custom_sizes( $sizes ) {
    return array_merge( $sizes, array(
        'dimensione-personalizzata' => __('Nome della dimensione personalizzata'),
    ) );
}

Trigger[edit]

A partire da WordPress 3.6 questo filtro si applica nei seguenti posti:

  • wp-admin/custom-background.php - in wp_set_background_image() riga 413
  • wp-admin/includes/media.php - in image_size_input_fields() riga 732
  • wp-includes/media-template.php - in wp_print_media_templates() riga 318
  • wp-includes/media.php - in wp_prepare_attachment_for_js() riga 1373
