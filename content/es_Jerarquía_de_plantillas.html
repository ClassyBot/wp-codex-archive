Introducción[edit]

Las plantillas de Wordpress encajan juntas como las piezas de un rompecabezas para generar las páginas web de tu sitio WordPress. Algunas plantillas (el Header(cabecera) y el Footer(pie de página como archivos por ejemplo) son usados en todas las páginas web del sitio, mientras que otras son usadas solamente bajo condiciones específicas.

De que se trata este artículo[edit]

Este artículo trata de responder la siguiente pregunta:

¿Cuál(es) archivo(s) de la plantilla usará Wordpress cuando muestra cierto tipo de página?

Quién podría encontrar esto útil[edit]

Desde la introduccion de Plantillas en WordPress 1.5, las Plantillas se han vuelto más y más configurables. En cuanto a desarrollo de plantillas para wordpress, el apropiado entendimiento de la manera en que Wordpress selecciona los archivos de la plantilla para mostrar varias páginas en tu blog, es esencial.si lo que buscas es personalizar una plantilla de Wordpress que ya existe, este artículo tiene como objetivo ayudarte a decidir cual archivo de plantilla necesitas editar.

Usando etiquetas condicionales[edit]

Wordpress provee mas de una forma de hacer coincidir las plantillas con lo que buscas. Los desarrolladores de Plantillas de WordPress también pueden usar etiquetas condicionales para controlar cual plantilla será usada para generar cierta página. Algunas plantillas de Wordpress puede que no implementen todos los archivos de plantilla aquí descritos. Algunas plantillas usan etiquetas condicionales para cargar otros archivos de plantilla. Mira las paginas de etiquetas condicionales y las de "Basadas en busqueda" en Desarrollo de plantillas para más información.
