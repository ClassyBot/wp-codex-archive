WPMU follows Wordpress Development. Currently, only Donncha O'Caoimh has commit rights. Check-ins can been seen on the WPMU SVN Changeset Log

  • Discussion is generally held on the WPMU Forums and to a lesser extent on the WPMU IRC Channel
  • Bug reports are accepted via WordPress MU Trac site
  • Code contributions should be submitted via attachments to WordPress MU Trac site

This has the advantage that one person is held accountable for getting changes into the code base. It also has the disadvantage that one person is held accountable for getting changes into the code base!

  • "and you'll see that these bugs were reported last week"

A lot of code is submitted and discussed in the WPMU Forums. The forums do not allow editing of other's posting entries, so contributions tend to pile up as subsequent postings, and there is little or no deletion of obsolete content.
