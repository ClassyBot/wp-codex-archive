Väčšina hostingových poskytovateľov za použitia osobnej inštalácie na Unix/Linux systémoch by vám mala umožniť hostiť váš WordPress s najčastejšou konfiguráciou.

Serverové požiadavky[edit]

Požiadavky serveru pre WordPress Verziu 2.9:

  • PHP verzia 4.3 alebo novšia
  • MySQL verzia 4.1.2 alebo novšia
  • (Nepovinné) Apache mod_rewrite modul (pre čisté URI známe ako Trvalé odkazy)

Požiadavky serveru pre WordPress Verzie 2.5, 2.6, 2.7, 2.8:

  • PHP verzia 4.3 alebo novšia
  • MySQL verzia 4.0 alebo novšia
  • (Nepovinné) Apache mod_rewrite modul (pre čisté URI známe ako Trvalé odkazy)

Požiadavky serveru pre WordPress Verzie 2.1, 2.2, 2.3:

  • PHP verzia 4.2 alebo novšia
  • MySQL verzia 4.0 alebo novšia
  • (Nepovinné) Apache mod_rewrite modul (pre čisté URI známe ako Trvalé odkazy)

Požiadavky serveru pre WordPress Verziu 2.0:

  • PHP verzia 4.2 alebo novšia
  • MySQL verzia 3.23.23 alebo novšia
  • (Nepovinné) Apache mod_rewrite modul (pre čisté URI známe ako Trvalé odkazy)


Odporúčané nastavenia[edit]

Odporúčame vám použiť Linux a to buď s Apache, alebo NGINX web-serverom ako najmohutnejšie platformy na beh WordPress, ale aj akýkoľvek server ktorý podporuje PHP a MySQL to zvládne. Ak váš host nepodporuje tieto platformy, a mod_rewrite, bude pre vás lepšie ak zmeníte váš hosting za jedného z mnohých poskytovateľov hostingu ktorý podporujú tieto možnosti.

Je tiež dôležité, aby váš hostiteľ umožňoval vzdialené pripojenie.

Poskytovatelia Hostingu[edit]

Hosting je dnes komoditou a s málinko prehrabávaním ľahko nájdete hosting ktorý podporuje hore uvedené požiadavky. Ak hľadáte návrhy hostingov, prosím vyskúšajte:

  • Odporúčaný poskytovatelia WordPress - Poskytovatelia čo podporujú WordPress viacerými cestami ako len jednou.
  • Hľadanie poskytovateľa pre WordPress na fórach podpory na odporúčanie
