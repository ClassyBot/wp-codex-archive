Опис[edit]

Перетворює строку дати з формата MySQL/MariaDB в інший формат.

Приклад[edit]

<?php $date = mysql2date( $format, $date, $translate ); ?>

Параметри[edit]

$format
(string) (required) Запит на формат виводу. Повинен бути рядком в форматі PHP date format. 'U' для мітки часу Unix, або 'G' для мітки часу Unix, при умові що $date у форматі GMT.
Default: None
$date
(string) (required) вхідний рядок, який не може бути необробленою міткою часу. Поперше, мітку часу перетворюємо у формат: 'Y-m-d H:i:s' , бо це єдиний шлях, як mysql2date() розпізнає дату.
Default: None
$translate
(boolean) (optional) Якщо true, дата та формат рядка буде передано функції date_i18n() для перекладу.
Default: true

Повертає Значення[edit]

(string|integer) 
Відформатований рядок часу, або мітка часу Unix timestamp.

Приклади[edit]

Конвертуємо a MySQL/MariaDB дату в мітку часу Unix timestamp: 

echo mysql2date( 'U', '2012-02-23 06:12:45' ); // 1329977565

Конвертуємо MySQL/MariaDB дату в інший формат дати: 

echo mysql2date( 'l, F j, Y', '2012-02-23 06:12:45' ) // Thursday, February 23, 2012

Початковий Файл[edit]

mysql2date() знаходиться у wp-includes/functions.php
