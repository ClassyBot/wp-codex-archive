Template Description

This Template is uses to link to the Thai Codex page. In most cases it'll be used inside {{Languages}} template for Multilingual Codex.

Usage

{{th|Article|Separator|Label}}

Parameters

Article
(text) (optional) The name on the article in Thai codex.
Default: หน้าแรก
Separator
(text) (optional) The separator charecter.
Default: •
Label
(text) (optional) The link label.
Default: ไทย

Example

Linking to Thai "Main Page" use {{th}}
ไทย •
Linking to Thai "Some Page" without Separator use {{th|Some Page|}}
ไทย
