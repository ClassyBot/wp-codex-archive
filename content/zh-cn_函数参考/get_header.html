说明[edit]

从你当前的主题目录包含（加载）模板文件header.php。 如果指定了名称（name），那么将包含头部文件header-{名称}.php。

如果主题没有header.php文件，那么将包含默认的wp-includes/theme-compat/header.php。

用法[edit]

<?php get_header( $name ); ?>

参数[edit]

$name
(string) (可选) 用来调用header-name.php。
Default: None

示例[edit]

简单的404页面[edit]

下面的代码是一个简单的例子，一个展示“HTTP 404：未找到”错误的模板。你可以把它作为文件 404.php 包含在你的主题中。 

<?php get_header(); ?>
<h2>错误404 - 未找到</h2>
<?php get_sidebar(); ?>
<?php get_footer(); ?>

多个头部[edit]

为不同的页面包含不同的头部。 

<?php
if ( is_home() ) :
	get_header( 'home' );
elseif ( is_404() ) :
	get_header( '404' );
else :
	get_header();
endif;
?>

home和404头部对应的文件名分别是header-home.php和header-404.php。

Notes[edit]

  • Uses: locate_template()
  • Uses: do_action() Calls 'get_header' action.
