自 2.7 版本以来，WordPress 就加入了快捷键的支持，允许用户使用快捷键来浏览、审核评论，以此节省时间。特别是在您的站点有着大量评论的时候，您会体验到快捷键的高效。

启用键盘快捷键[edit]

键盘快捷键的开启、关闭，并不是全局设定的，而是用户的个人选项 —— 用户可在其个人资料编辑页面（管理页面 > 用户 > 我的个人资料）进行启用。选中键盘快捷键的复选框来启用评论审核的快捷键。

请注意，快捷键的使用需要两只手，且必须使用 QWERTY 布局的键盘。

评论的选定[edit]

键盘快捷键可以用来浏览评论和控制（操作）评论，当然，先要 选定 才能做出这些操作。被 选定 的评论有淡蓝色的背景，默认情况下，没有评论是被 选定 的，所以你需要按下键盘上的字母 j，或者字母 k 来开始浏览和 选定

使用键盘快捷键来浏览[edit]

使用键盘快捷键来进行评论浏览是通过字母 j 和 k来实现的。当你第一次进入评论管理页面时，没有一条评论会是被 选定 的状态，所以按下字母 j 或者 k 来 选定 第一条评论，之后可以按下字母 j 来移动到下一条评论。

  • 按下 j 将当前 选定 光标（淡蓝色背景）向下移动
  • 按下 k 将当前 选定 光标（淡蓝色背景）向上移动

如果你把 选定 光标移动到了评论管理页面中最后一条，然后按下 j，页面会跳转到下一页并自动 选定 第一条评论。以此类推，选定 光标在评论管理页面的第一条时按下 k，会返回之前的页面并自动 选定 最后一条评论。这样您就可以很方便地浏览大量的评论和对评论做出相应的操作。

使用键盘快捷键审核评论[edit]

评论必须处于 选定 状态（参见前一段）时以下操作才能生效。

  • 按下 a 设置当前选定的评论状态为“获准”。
  • 按下 s 设置当前选定的评论状态为“垃圾评论”。
  • 按下 d 设置当前选定的评论状态为“回收站” (WordPress 2.9以上版本) 或者删除当前选定评论。
  • 按下 z 从回收站中恢复当前选定的评论，或当选定多条时撤销选定。

上述三个快捷键行为（最为常用）是双手放在QWERTY键盘时左手按着的三个键，为了更加方便地使用，建议您将左手左平移一个键位，即将食指从 f 移动到 d 。

  • 按下 u 设置当前选定的评论状态为“未获准”, 并置为“未审核”。
  • 按下 r 回复当前选定的评论 (按 Esc 取消回复).
  • 按下 q 激活 "快速编辑" 功能，让您快速地编辑当前选定的评论。

批量操作[edit]

您可以在选定多条评论后对它们进行批量操作。首先要选定多条评论以便进行批量操作，对当前 选定 的评论按下 x ，评论前的选择框将 勾选 。按组合键 Shift-x 将反选已 勾选 的评论:

  • 按下 Shift-a “获准”已 勾选 的评论。
  • 按下 Shift-s “垃圾评论”已 勾选 的评论。
  • 按下 Shift-d “删除”已 勾选 的评论。
  • 按下 Shift-u “未获准”已 勾选 的评论。
  • 按下 Shift-t 将已 勾选 的评论放入“回收站”。
  • 按下 Shift-z 从“回收站”恢复已 勾选 的评论。
