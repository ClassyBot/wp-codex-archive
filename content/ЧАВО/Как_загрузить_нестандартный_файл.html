При загрузке файлов *.rar, *.swf и других выдаётся сообщение: «Файл не удалось загрузить из-за ошибки. Тип файла не подходит по соображениям безопасности».

Можно добавить нужный MIME-тип в список разрешённых с помощью подобного фрагмента в functions.php активной темы: 

function additional_mime_types( $mimes ) {
	$mimes['rar'] = 'application/x-rar-compressed';
	$mimes['swf'] = 'application/x-shockwave-flash';

	return $mimes;
}
add_filter( 'upload_mimes', 'additional_mime_types' );

Или воспользоваться для тех же целей плагином PJW Mime Config.
