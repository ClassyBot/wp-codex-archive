Поддержка WordPress MU осуществляется на сайте WordPress MU и форумах WordPress MU. Поддержка блогов на WordPress.COM осуществляется на форумах поддержки WordPress.com.

WordPress MU или WordPressµ — многопользовательская версия известного приложения для ведения блогов в WordPress. Она идеально подойдёт тем, кто хочет открыть крупную сеть блогов. Обсуждение можно почитать на форуме, который расположен на сайте MU.WordPress.

WordPress MU широко используется в блогах на WordPress.com, различными изданиями, университетами и крупными компаниями, блог-системы которых работают за межсетевыми экранами.

Используя многопользовательскую версию WordPress, вы сможете дать пользователям возможность создавать и настраивать их блоги, не мешая остальным. Нет ограничений ни на число пользователей, ни на число блогов. Пользователи могут выполнять различные роли в различных блогах (авторы, редакторы и т.д.)

Возможности[edit]

В отличие от однопользовательской версии WordPress, модулями расширения и темами оформления в WordPress MU управляет основной системный администратор. Сами пользователи таких полномочий не имеют.

Возможности хоста[edit]

  • Создание «сообществ» пользователей.
  • Содержание тысяч пользователей на одном сайте.
  • Opportunity to provide free hosting as an incentive to paid hosting services.
  • Множество параметров для добавления модулей и функций администрирования.
  • Настройка панелей администрирования.
  • Управление темами оформления.
  • Обеспечение безопасности «прямо на месте».
  • Добавление модулей защиты от спама в комментариях.

Возможности пользователей[edit]

Пользователи, ведущие свои блоги в WordPressMU, фактически работают с ограниченной версией WordPress и, соответственно, ограничены в настройке. Тем не менее, у них остаётся масса возможностей и преимуществ.

  • Никакой установки. Достаточно ввести имя пользователя и пароль, чтобы вести свой блог.
  • Возможность попробовать WordPress в деле с минимальным риском или вовсе без него.
  • Право выбирать проверенные и готовые к использованию темы оформления.
  • Участие в сообществах.
  • Защита от спама для пользователей предустановлена.

WordPress MU разработан для предоставления услуг хостинга блогов на WordPress пользователям. Полная версия WordPress до сих пор бесплатна и доступна для скачивания, она разработана для предоставления возможности полной настройки одно- или мультипользовательских блогов и/или множества блогов одним администратором.

Установка и поддержка[edit]

Installing WPMU содержит документацию по установке.

Русские языковые файлы для текущей версии доступны на l10n-ru.

За поддержкой и информацией по установке и использованию WordPress MU, посетите сайт WordPress MU и форум поддержки. WordPress MU в работе и сопровождении сложнее однопользовательского WordPress, он не для робких или неподготовленных пользователей.

Если вы пользователь WordPress MU кликните по кнопке "Обратная связь" в вашей "Консоли". Еще немного информации можно найти внутри WordPress Codex в темах, указанных ниже или тут - Donncha's FAQ.

  • There are some special WPMU Plugins and not all WP plugins work with WPMU, see WPMU Plugin Compatibility
  • Not all themes work with WPMU, see WPMU Theme Compatibility
  • WPMUDEV.org is a repository for uploading special WPMU related contributions
  • There are WPMU Functions and WPMU Global Variables that should use to interact with WPMU's multiuser features
  • There are some special WPMU Widgets in development
  • The code can be checked out from the WPMU SVN Repository. Don't ask for check-in rights - like WordPress, WPMU is closed for contributions. Instead please submit your contributions to the WPMU Trac as a ticket.
  • If you don't want to submit a change to the core, you can share a WPMU Hack here on Codex.

There is the WPMU IRC Channel for real time help and chat.
