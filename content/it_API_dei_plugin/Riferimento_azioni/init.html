Viene eseguita dopo che WordPress ha finito di caricare ma prima che qualunque header venga inviato. Utile per intercettare $_GET o $_POST.

Ad esempio, per agire sui dati $_POST:
add_action('init', 'process_post');

function process_post(){
 if(isset($_POST['unique_hidden_field'])) {
   // elaborare i dati $_POST qui
 }
}

Anche le chiamate a load_plugin_textdomain dovrebbero essere effettuare durante init, altrimenti gli utenti non possono agganciarvisi.
