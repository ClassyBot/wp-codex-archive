설명[edit]

당신의 웹사이트의 front side 에 보여질 Toolbar 의 상태를 설정합니다. (워드프레스 알림판에서 툴바를 끌 순 없습니다.)

알림: Admin Bar는 워드프레스 버전 3.3 이래로 Toolbar 와 함께 교체되었습니다.

사용법[edit]

<?php show_admin_bar( $bool ); ?>

파라메터[edit]

$bool
(boolean) (required) 툴바를 보일 것인지 아닌지 결정합니다.
Default: None

반환값[edit]

none

추가 정보[edit]

이 함수는 플러그인 로드 혹은 테마안에 위치된 functions.php 파일에서 즉각적으로 불려져야만 합니다.

이 함수는 워드프레스 3.3버전 이전의 버전에서 보여지는 대쉬보드(알림판) 안의 툴바의 보여짐에 대하여 영향을 미칩니다. 3.3 버전 이후로는 대쉬보드 페이지에서는 영향을 미치지 않습니다.

예제[edit]

show_admin_bar( false );


function.php 파일 테마안에 코드를 위치시키는 것은 당신의 웹사이트에서 프론트엔드 위에서 툴바가 보여지는 것을 막을 것 입니다.

당신은 또한 admin 바가 보여질 유저들에 대해서 결정할 수 있습니다. 어드민 바가 관리적인 특권들과 함께 일부 유저들에게만 보여져야만 한다면, 예제를 따르세요. 

if ( ! current_user_can( 'manage_options' ) ) {
    show_admin_bar( false );
}

워드프레스의 새로운 버젼과 함께 대쉬보드에서 툴바의 사용가능을 버리기 위해서는 앞의 예제를 사용해야만 합니다. 그러나 직접적으로 직면하는 프론트 페이지들에서만 숨겨집니다. ( 이말인 즉슨 관리자 페이지에서는 결국엔 보여집니다. ) 

add_filter('show_admin_bar', '__return_false');

변경 기록[edit]

  • Since: 3.1.0

소스 파일 (근원지)[edit]

show_admin_bar() 는 wp-includes/admin-bar.php에 위치되어 있습니다.
