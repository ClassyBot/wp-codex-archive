Como recupero o acesso ao site depois de mudar URL do site?[edit]

Após mudar endereço do site pelo painel do WordPress, você pode deixar o site inacessível, para reveter isso, basta corrigir/reverter os dados através do phpMyAdmin, um aplicativo em seu painel de serviço de hospedagem.

  • Na table usada pelo seu WordPress, procure pelo item chamado (prefixo)_options (o prefixo geralmente é wp mas pode variar).
  • Na linha do item, clique no ícone Browse ou Visualizar.
  • No item siteurl clique no ícone Edit ouEditar, na tela seguinte, use o URL original/anterior ao problema de acesso e salve.
  • Novamente acesse (prefixo)_options, procure agora pelo item Home, clique no ícone Edit ouEditar, na tela seguinte, use o URL original/anterior ao problema de acesso e salve.

Como obter URL's sem "cruft" para resultados de busca?[edit]

Para um resposta limpa URIs na busca no WordPress vindo do formulário de busca (www.example.com/search/searchterms instead of www.example.com/?s=searchterms)

Primeiro crie um arquivo chamado search.php contendo: 

<?php header('Location: http://www.example.com/search/' . $_GET['s']); ?>

Coloque esse arquivo na raiz da sua instalação do WordPress e altere a action para o formulário de busca como seguinte código: 

action="<?php bloginfo('url'); ?>/search.php"

Como eu consigo ter os arquivos .phps para mostrar em um texto colorido em meu servidor?[edit]

Se você deseja ter o seu servidor exibindo os arquivos .phps desta forma updatelinkroll.phps at carthik.net, adicione as seguintes linhas em seu .htaccess na pasta raiz de seu servidor. Isto presume que seu servidor é Apache e tem a habilidade de adicionar tipos em seu .htaccess 

AddType application/x-httpd-php-source .phps

Como eu posso fazer uma edição em massa no campo Status para todos os posts em meu Banco de dados?[edit]

Utilize o seguinte comando SQL para mudar o post_status em todos os posts em sua wp_posts tabela no banco. Este Comando irá fazer uma edição e excluir as páginas que foram modificadas -- Lembre de substituir o STATUS com draft, private, ou publish. 

UPDATE wp_posts  SET post_status = 'STATUS' WHERE post_status != 'static';

Se você tem multiplos autores e quer somente realizar uma edição em massa nos posts de um autor autor específico, você pode usar o seguinte comando, mas lembre de substituir NUMBER com o ID correto no post_author. 

 UPDATE wp_posts SET post_status='STATUS' WHERE post_author='NUMBER';

Como criar um link em meu header que vincula a uma URL[edit]

Esta solução assume a criar uma Page faz o link ser posto em seu header.

Install and activate O plugin chamado Redirectifiy. Crie uma Pagina chamada Wiki e salve esta página. Em seguida vá em Manage->Pages e edite essa página e adicione Custom field com uma 'key' de redirecionamento e um valor da URL para sua Wiki. (Essa última edição da página é necessária para superar o bug onde você pode atribuir um Campo Customizado para uma nova página, mas você pode isso quando edita uma página.)
