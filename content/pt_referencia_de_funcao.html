Os arquivos do wordpress definem muitas funções uteis do PHP. Algumas das funções, conhecidas como Template Tags, são definidas especialmente para uso nos temas do Wordpress. Existem também algumas funções relacionadas a ações e filtros (A API de Plugins), que são usadas primariamente para o desenvolvimento de Plugins. O resto é usado para criar o nucleo da funcionalidade do Wordpress.

Muitas das funções do nucleo do Wordpress são uteis para desenvolvedores de Plugins e Temas. Então este artigo lista a maioria das funções do nucleo, exluindo Template Tags. No fim da pagina tem uma seção listando outras fontes para encontrar informações sobre funções do Wordpress.

Nota: Essa referencia se aplica a versão 2.1 e versões posteriores do Wordpress, e não inclui funções que foram descontinuadas como as da versão do Wordpress. Um pagina incompleta para as versões 2.0.x do Wordpress pode ser encontrada em Function Reference 2.0.x.
