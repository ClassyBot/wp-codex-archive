Painel → Início[edit]

O 'Painel Início é a área para acesso às áreas de Administração mais usadas do blog além de oferecer informações da Comunidade WordPress.

Repare que há blocos de informações chamados de módulos. O WordPress mostra oito módulos como Agora, Comentários, Links de Entrada, Plugins, QuickPress, Rascunhos Recentes, WordPress Blog e Outras Notícias WordPress. Além dos módulos, você eventualmente pode ver o painel Bem-Vindo e o Aviso de Segurança.

Opções de Tela[edit]

Mostrado logo abaixo do cabeçalho, à esquerda, Opções de Tela permite escolher quais módulos são mostrados ou não e quantas colunas usar para mostrar estes módulos. Clicando na aba Opções de Tela você verá duas seções:

Mostrar na Tela
Lista vários módulos com uma caixa de seleção ao lado de cada item.Marque os itens que quer mostrar e desmarque para ocultar.
Layout da Tela
Nesta seção, clique nos números de 1 a 4 correspondente ao número de colunas que quer usar para mostrar os módulos. Ao aumentar o número de colunas, você deve clicar e arrastar segurando na barra de título do módulo.

Clique na aba Opções de Tela para esconder as opções.


Organizando o Painel[edit]

Módulos podem ser movidos para cima e para baixo, de uma coluna para outra. Apenas mova o cursor no mouse sobre a barra de título, clique, arraste para o novo lugar no painel e então solte o botão do mouse. Módulos podem ser mostrados em até 4 colunas como visto acima.

Desenvolvedores de plugin ou de temas adicionar novos módulos ao Painel através de código para plugins (ou arquivo functions.php) com muita facilidade.

Módulos Padrão do Painel[edit]

Os Módulos do Painel a seguir são incluídos por padrão no WordPress e devem estar disponíveis em todas as instalações.

Agora[edit]

O módulo Agora oferece um sumário de várias estatísticas do blog, incluindo o número de postagens, páginas, categorias, comentários, tags, comentários aprovados, comentários pendentes e Spams. Na frente de cada tipo de conteúdo, uma estatística (número) é apresentado sob a forma de um link. Clique sobre esse link para ir para a área específica e administrar tal item.

Além disso, o tema atual é descrito junto com o número de Widgets em uso. Um botão para mudar temas também é oferecido.

Na parte inferior do módulo Agora, um comunicado diz qual versão do WordPress você está usando.

Comentários Recentes[edit]

A lista dos mais recentes comentários no seu blog é detalhado neste módulo. Cada comentário dessa lista possui um link para o título da postagem relacionada, ao clicar nesse link para editar a postagem. Ao passar o mouse sobre cada comentário ativa um menu de opções para aprovar (ou Desaprovar) o comentário, editar, responder para o autor do comentário, marcar como spam ou excluir o comentário.

Ver Todos
leva você para editar comentários no Painel Comentários.

QuickPress[edit]

O módulo QuickPress permite que você escreva novos post (ou rascunhos) de forma rápida e fácil. Entre com um título para o post, atualize/insira media, digite o conteúdo do post, adicione tags, click no botão Publicar ou click no botão Salvar Rascunho. É muito fácil!!!

Nota: Categorias serão adicionadas ao QuickPress em uma próxima atualização.

Rascunhos Recentes[edit]

Este módulo permite um fácil acesso à seus rascunhos de posts recente.s Clicando sobre o título do post o leva ao painel de edição onde o post pode ser editada.

Ver tudo
Clique neste link para ir até o painel listando apenas os rascunhos.

Links de Entrada[edit]

Este módulo faz consultas ao [Google Blog Search] para que, quando são criados links para o seu site. Se não houver links de entrada aparece a mensagem ". Ele não encontrou ainda ligações recebidas ... Está tudo bem - não há pressa." será exibida.

No canto superior direito da caixa, um link "Configurar" permite que você edite as informações do feed RSS que fornece as ligações recebidas, e um link para o feed RSS atual.

Configurar
Quando clicado , você pode controlar as informações exibidas na caixa. Clique no botão Cancelar para sair da configuração
Digite a URL do feed RSS aqui: - digite a fonte de publicações para receber suas novas ligações
Quantos itens você gostaria de exibir? - quantas ligações recebidas devem ser exibidas ::Mostrar data do item? - Marque esta caixa para exibir a data em cada link novo
Enviar - clique neste botão para salvar suas alterações.

Aviso de Segurança[edit]

Este aviso eventualmente pode aparecer se o navegador que estiver usando for inseguro ou de uma versão anterior.

Aviso sobre o navegador Chrome


Você tanto pode clicar no link oferecido para atualizar seu navegador atual, quanto clicar no link Browse Happy, onde conhecerá outros navegadores que pode experimentar.

Plugins[edit]

O módulo de Plugins exibe os plugins mais populares , mais novos e plugins atualizados mais recentemente, como visto em WordPress Extend Plugins site. Clique no Título do Plugin para visitar a página do plugin que no site WordPress Extend Plugins. Clique no link Install para iniciar o processo de instalação desse plugin .

Blog do WordPress[edit]

Este módulo do Painel de Administração lista "as últimas notícias do blog oficial do WordPress." Aqui você pode descobrir o que os desenvolvedores do WordPress fizeram recentemente e manter-se atualizado com as últimas notícias relacionadas ao WordPress. Além de desenvolvimentos de software, tais como anúncios de versão e avisos de segurança, as notícias sobre a comunidade WordPress, em geral, são publicadas periodicamente.

Configurar 
Ao clicar você pode configurar as informações que serão exibidas no box. Click the Cancel button to cancel the configuration edit.
Coloque a URL do feed RSS aqui: - URL do feed, a URL padrão é http://wordpress.org/development/feed/
Give the feed a title (optional): - description (title of the box) of this feed, defaults to WordPress Blog.
How many items would you like to display? - use the pulldown box to select the number of items to display.
Display item content? - check this box to display an excerpt of the content.
Display item author if available? - check this box to display the post author
Display item date?- check this box to display the date on the post
Submit - click this button to save your configuration changes.
