Les Raccourcis Clavier de la Gestion des Commentaires[edit]

Depuis la version WordPress Version 2.7, il est possible de gérer les commentaires à l'aide de raccourcis clavier. Ces raccourcis clavier sont conçus pour gagner du temps en vous permettant de naviguer rapidement et d'effectuer des actions sur les commentaires. Si votre blog reçoit un grand nombre de commentaires, vous trouverez ces raccourcis particulièrement utiles.

Activer les Raccourcis Clavier[edit]

Ces raccourcis sont activables pour chaque utilisateur dans son profil Administration > Utilisateurs > Votre Profil en cochant la case Raccourcis clavier (en anglais).

S'il vous plaît, veuillez noter que les raccourcis clavier sont conçus pour être utilisés à deux mains sur une disposition de clavier de style QWERTY.

La Sélection d'un Commentaire[edit]

Les raccourcis clavier sont utilisés pour la navigation et pour les actions. Avant d'effectuer une action sur un commentaire, celui-ci doit être sélectionné. Le commentaire qui est sélectionné est indiqué par un fond bleu clair. Par défaut, aucun commentaire n'est sélectionné, vous devrez appuyer sur les lettres j ou k pour lancer la navigation/sélection entre les commentaires.

Utilisation des Raccourcis Clavier pour Naviguer[edit]

La navigation entre les commentaires s'effectue à l'aide des touches j et k. À l'ouverture de la page des Commentaires, aucun commentaire n'est sélectionné ; il faut d'abord utiliser les touches de navigation (j ou k) pour sélectionner le premier commentaire. Ensuite, il suffit d'appuyer sur la touche j pour sélectionner le commentaire suivant.

  • Appuyer sur j déplace la sélection actuelle (fond bleu clair) vers le bas.
  • Appuyer sur k déplace la sélection actuelle (fond bleu clair) vers le haut.

Notez que si vous avez sélectionné le dernier commentaire d'une page et que vous appuyez sur j à nouveau, vous serez redirigé automatiquement vers la page suivante et le premier commentaire de la page suivante est sélectionné. De même, en appuyant sur k après avoir sélectionné le premier commentaire de la page, vous serez redirigé sur la page précédente et le dernier commentaire de la page précédente est sélectionné. Il est donc très facile de naviguer à travers une longue liste de commentaires pour effectuer les actions nécessaires.

Utilisation des Raccourcis Clavier pour Exécuter des Actions[edit]

Assurez-vous que le commentaire est d'abord sélectionné (voir ci-dessus) avant d'effectuer l'une de ces actions.

  • Appuyer sur a : Approuve le commentaire sélectionné.
  • Appuyer sur u : Désapprouve le commentaire sélectionné (le replace dans les commentaires en attente).
  • Appuyer sur s : marque le commentaire sélectionné comme Indésirable.
  • Appuyer sur d : déplace le commentaire sélectionné dans la Corbeille (WordPress 2.9) ou le Supprime définitivement (Indésirables et Corbeille).
  • Appuyer sur z : Annule la mise à la corbeille du commentaire sélectionné.
  • Appuyer sur r : Répond au commentaire sélectionné (Echap pour annuler la saisie de la réponse).
  • Appuyer sur q : Modifie rapidement le commentaire sélectionné.
  • Appuyer sur e : Modifie le commentaire sélectionné dans l'écran d'édition des commentaires.

Actions Groupées[edit]

Vous pouvez également effectuer une action sur plusieurs commentaires à la fois. D'abord, vous devez sélectionner les commentaires en utilisant la touche x pour cocher les cases des commentaires à sélectionner. Shift + x peut être utilisé pour inverser l'état de sélection des commentaires (tous les commentaires non sélectionnés le sont ensuite, tous les commentaires sélectionnés ne le sont plus).

Une fois que vous avez sélectionné les commentaires, vous pouvez effectuer une action groupée sur ces commentaires dont la case sélectionnée est cochée en :

  • Appuyer sur Shift + a : Approuve les commentaires.
  • Appuyer sur Shift + a : Désapprouve les commentaires (les replace dans les commentaires en attente).
  • Appuyer sur Shift + s : marque les commentaires comme Indésirables.
  • Appuyer sur Shift + t : déplace les commentaires dans la Corbeille.
  • Appuyer sur Shift + z : Annule la mise à la corbeille des commentaires sélectionnés (Corbeille).
  • Appuyer sur Shift + d : Supprime définitivement les commentaires (Indésirables et Corbeille).

Les Raccourcis de l’Éditeur[edit]

L'éditeur intégré de WordPress a de nombreux raccourcis utiles. Plutôt que d’attraper votre souris pour cliquer sur la barre d’outils, utilisez les raccourcis clavier. Sous Windows et Linux, utilisez Ctrl + <lettre>, sous Macintosh, utilisez la touche Commande + <lettre>. Ces raccourcis fonctionnent uniquement en mode Visuel.
