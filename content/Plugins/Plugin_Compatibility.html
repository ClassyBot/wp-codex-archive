Plugin Compatibility on WordPress Extend[edit]

For the most up-to-date plugin compatibility data, please visit individual plugin pages on the WordPress Plugin Directory and look for the Compatibility widget (see an example on the Hello Dolly plugin page). Compatibility information is crowd-sourced and contributed by users like yourself, so if you find any missing information, please help the community by testing the plugin and indicating whether it works or not.
