Descrizione[edit]

the_title è un filtro che viene applicato al titolo del post recuperato dal database prima che venga stampato a video. In alcuni casi (come quando si usa the_title), il titolo può essere soppresso facendo restituire alla funzione che gestisce il filtro un valore fasullo (es. NULL, FALSE o la stringa vuota).

Parametri[edit]

$title
(string) (richiesto) Il titolo del post
Default: Nessuno
$id
(int) (richiesto) L'ID del post
Default: Nessuno

Esempi[edit]

Sopprimere il titolo nei template per tutti i post della categoria "blurb": 

function suppress_if_blurb($title, $id) {
    if (in_category('blurb', $id)) {
        return '';
    }
    return $title;
}
add_filter('the_title', 'suppress_if_blurb', 10, 2);

File sorgente[edit]

Questo filtro è applicato da:

  • do_trackbacks in wp-includes/comment.php
  • wp_get_archives e get_calendar in wp-includes/general-template.php
  • get_adjacent_post_rel_link, get_boundary_post_rel_link, get_parent_post_rel_link e adjacent_post_link in wp-includes/link-template.php
  • Walker_Nav_Menu->start_el in wp-includes/nav-menu-template.php
  • get_the_title e Walker_Page->start_el in wp-includes/post-template.php
