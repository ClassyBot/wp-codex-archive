WPMU 是 WordPress MultiUser 或 WordPress_MU 的简写形式

当您从支持论坛来到这里时，本页将是您的向导页

  • 首先，您可以查看 Readme 文件。在遇到问题并想在论坛发贴求助之前，请确保已经通读了 Readme 文件。
  • Codex 是关于 WordPress 和 WordPress MU 的信息集合。WPMU 索引页面列出了所有相关的分类，并且非常易懂。WPMU 最近更新可以让您更方便的了解您感兴趣的项目动向。
  • Trac 是我们的一个问题跟踪系统。如果您觉得您发现了一个 Bug，您可以在 Trac 上发表一个描述详细的帖子。Trac 也是一个寻找问题是否被解决的好去处。并且在那里，您可以查看程序的全部代码。
  • WPMU 聊天室在 IRC 上 Freenode 中的 #wpmu 频道。这里是链接聊天室的相关介绍，如果您对 IRC 不熟悉，可以进行查询。
