Page d'accueil du Codex en français - Télécharger WordPress en français
Les utilisateurs francophones se retrouvent sur le site WordPress-Francophone, notamment sur son forum d'entraide.

Il existe deux méthodes pour déposer les fichiers sur votre site, et une fois en place, les modifier.

  1. En utilisant le gestionnaire de fichiers fourni par votre hébergeur.
  2. En utilisant un client FTP ou SFTP. Ce guide vous expliquera comment utiliser FileZilla (en anglais).

Le FTP (ou "File Transfer Protocol") est le protocole le plus répandu pour la transmission de fichiers depuis plus d'une trentaine d'années. Toutefois il ne sécurise pas la connexion. Le SFTP (ou "Secure File Transfer Protocol") transfère les fichiers et le mot de passe par une connexion sécurisée, et sera donc utilisé en priorité s'il est disponible. La majorité des hébergeurs offrent le SFTP, mais vous devrez peut-être contacter votre fournisseur pour activer ce service.

Pourquoi utiliser FileZilla ? Parce que, comme WordPress, il est publié sous la license GPL. De ce fait, ce logiciel n'est pas seulement gratuit, il le restera. FileZilla est également un client stable qui marche sur toutes les versions de Windows. La page d'accueil du projet FileZilla est http://filezilla.sourceforge.net/.

Vous avez besoin d'un client FTP pour votre Mac ? Essayez donc Cyberduck, qui est également gratuit et sous licence GPL.
