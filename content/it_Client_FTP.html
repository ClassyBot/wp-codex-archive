Ci sono due metodi per mettere dei file sul tuo sito e, una volta lì, cambiarli.

  1. Usando il gestore file fornito dal pannello di controllo del tuo server di hosting. Gestori file noti: cPanel, DirectAdmin, Plesk
  1. Usando un client FTP o SFTP. Questa guida ti spiegherà come usare FileZilla.

L'FTP o "File Transfer Protocol" è il protocollo di trasferimento più usato negli ultimi trent'anni, ma non offre una connessione sicura. L'SFTP o "Secure File Transfer Protocol" trasferisce i tuoi file e la tua password con una connessione protetta, e dovrebbe, quindi, essere usato al posto dell'FTP quando possibile. La maggior parte dei provider offre supporto per le connessioni SFTP, anche se potresti doverli contattare per abilitare l'SFTP sul tuo account.

Perché usare FileZilla? Perché, come WordPress, è rilasciato sotto licenza GPL. Quindi non è solo libero, ma è anche destinato a restarlo. FileZilla è anche un client stabile che funziona su tutte le versioni di Windows.

Hai bisogno di un client FTP o SFTP per il tuo Mac? Prova Cyberduck, anch'esso libero e rilasciato sotto licenza GPL.
