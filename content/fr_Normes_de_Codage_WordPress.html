Les Standards PHP, HTML, CSS and JavaScript pour WordPress ont été déplacés[edit]

Les normes de codage de WordPress ont été déplacées vers le Manuel de Contributeur du Cœur

Les normes de codage PHP WordPress sont désormais disponibles dans le Manuel de Contributeur du Cœur (en anglais).

Les normes de codage HTML WordPress sont désormais disponibles dans le Manuel de Contributeur du Cœur (en anglais).

Les normes de codage CSS WordPress sont désormais disponibles dans le Manuel de Contributeur du Cœur (en anglais).

Les normes de codage JavaScript WordPress sont désormais disponibles dans le Manuel de Contributeur du Cœur (en anglais).
