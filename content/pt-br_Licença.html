Licença do WordPress[edit]

Como a maioria dos softwares, o WordPress é distribuído sob uma licença, o que significa que há certas coisas que você está legalmente permitido (e não é permitido) para fazer com o software WordPress e seu código-fonte. O WordPress é distribuído sob uma licença chamada de GNU General Public License, uma licença muito popular na indústria de código aberto. Se isso não soar familiar, então continue a ler...

Sobre o GPL[edit]

A Licença Pública Geral GNU (GNU General Public License), ou GPL, é uma licença código-aberto. Código aberto não significa apenas que você pode ver o código-fonte - tem implicações políticas e filosóficas. Código aberto, ou "Software Livre", significa que você é livre para modificar e redistribuir o código-fonte sob certas condições. Free não se refere ao preço, refere-se à liberdade.

Porquê escolhemos GPL[edit]

As razões para o WordPress ser sob GPL são práticas e idealistas. O WordPress nasceu da própria liberdade mencionada anteriormente. O antecessor ao projeto WordPress, b2/Cafelog, também foi um projeto código aberto.
