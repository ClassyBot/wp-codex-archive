I file di WordPress definiscono molteplici e utili funzioni PHP. Alcune di queste, conosciute come Tag dei template, sono progettate per essere utilizzate specialmente nei Temi WordPress. Vi sono altre funzioni collegate alle azioni e ai filtri (le API dei Plugin), che sono utilizzate principalmente per lo sviluppo di Plugin. Tutte le altre sono utilizzate per creare il "cuore" delle funzionalità di WordPress.

Molte di queste funzioni di base di WordPress sono utili agli sviluppatori di Plugin e Temi. In questo articolo sono elencate quasi tutte le funzioni di base con l'esclusione dei Tag dei template. In fondo a questa pagina vi è una sezione che elenca altre risorse dove trovare informazioni sulle funzioni di WordPress. In aggiunta a queste informazioni, il sito WordPress phpdoc dettaglia tutte le funzioni di WordPress dalla versione 2.6.1 in poi.

Aiutaci a rendere questa pagina sempre più completa!

Ecco alcune delle cose che puoi fare per aiutare a migliorare la pagina:

  • Aggiungere la documentazione a funzioni non ancora documentate, creando sottopagine o aggiungendo un breve commento alle liste sottostanti. Se si crea una sottopagina per una funzione, inserire informazioni ed esempi sull'uso di tale funzione con la medesima struttura delle altre sottopagine di questi elenchi.
  • Elencare ulteriori funzioni, seguendo la struttura delle categorie presenti.
  • Correggere gli errori spostando le funzioni in categorie più opportune e ovviamente correggendo errori di battitura. Nota: una funzione può comparire in più di una categoria.

Si legga Contributing to WordPress (in inglese) per scoprire come contribuire agli sforzi per migliorare il codex!

Riferimenti ufficiali e non[edit]

  • WordPress.org PHPDoc - Quasi tutte le funzioni in WordPress hanno intestazioni in stile "PHPDoc". Questo sito mostra tutti i PHPDoc.
  • Trac Browser - Il codice sorgente di WordPress. Scegliere 'Tags' per trovare il codice sorgente di una specifica versione, o 'Trunk' per vedere la versione più recente.
  • PHPXref for WordPress - Riferimenti incrociati di file, funzioni e variabili di WordPress, che sembrano riferirsi alla versione rilasciata più di recente.
  • WordPress Function History Table - Elenca ogni funzione di WordPress con l'indicazione della versione in cui è stata introdotta o rimossa. PlanetOzh raccomanda: "si veda WordPress Functions Implementation History Tool per maggiori informazioni (e per uno strumento più usabile)"
  • WordPress Variable Report - Una lista di funzioni di WordPress, che elenca quale versione di PHP ogni funzione richiede (e se c'è una funzione compatibile disponibile), quali variabili/argomenti nelle funzioni sono inutilizzati, e quali funzioni/variabili ogni classe contiene (e i loro valori predefiniti).
  • WordPress Hooks Table - Una lista di azioni e filtri di WordPress, con riferimenti ai file in cui si trovano
