Pred namestitvijo WordPress-a morate preveriti če vaš ponudnik gostovanje izpolnjuje potrebne pogoje (brez panike! večina jih) in da imate sami dovolj znanja (brez panike! ni tako zapleteno). Nobeden od teh dveh pogojev ne bi smel biti problem, toda pametno se je najprej prepričati.

Za seznam zahtev vašega spletnega strežnika si poglejte ta razdelek - Hosting WordPress. Drugi kraj za preverjanje je tudi the official requirement page.

Vedeti boste morali kako uporabljati urejevalnik besedila za urejanje glavne konfiguracijske datoteke. Če ste uporabnik sistema Windows bo Beležnica ( NotePad ) dovolj. Pozneje boste verjetno želeli urediti vaše predloge (glej Templates za nekatere primere). To lahko storite s pomočjo WordPress Administration Panels, vendar je uporaba dobrega urejevalnika besedil zelo priporočljiva. Za več informacij na to temo si poglejte Editing Files.

You will need to be able to use an FTP program to upload files and set file permissions (optional). You could choose FileZilla for this task. For help with FTP programs, see Podz' tutorial.

That's it! Now you are all set to go on to Installation.
