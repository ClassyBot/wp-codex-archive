Cos’è phpMyAdmin?[edit]

Strumento di amministrazione, phpMyAdmin è uno script PHP ideato per dare agli utenti la capacità di interagire con i loro database MySQL. WordPress memorizza tutte le sue informazioni nel database MySQL ed interagisce con esso per generare le informazioni presenti all’interno del sito. Una vista «grezza» dei dati, delle tabelle e dei campi memorizzati nel database MySQL è accessibile attraverso phpMyAdmin.

A cosa serve?[edit]

Il programma phpMyAdmin è comodo per eseguire operazioni di manutenzione sulle tabelle, effettuare il backup delle informazioni e modificare i dati direttamente nell’eventualità che WordPress non funzioni. A volte, nel forum di supporto, qualcuno scrive una query SQL di qualche beneficio o altre operazioni che possono essere eseguite utilizzando phpMyAdmin. Sebbene molte di queste azioni possono essere compiute dalla linea di comando di MySQL, fare ciò non è alla portata di tutti.

Dove lo posso ottenere?[edit]

Molti pannelli di controllo, come cPanel e Plesk, hanno phpMyAdmin preinstallato, per cui non c’è nulla di speciale che si debba fare per utilizzarlo. Normalmente c'è un link nella pagina relativa ai database. Chiedi al tuo provider di hosting se è disponibile.

Puoi anche scaricare phpMyAdmin ed installarlo dalla pagina principale del progetto.

Attenzione[edit]

Da un grande potere derivano grandi responsabilità. phpMyAdmin ti permette di interagire direttamente con il database, ma allo stesso tempo ti lascia anche combinare pasticci. Non esiste un’opzione «annulla» o «ripristina» nel database. Poni sempre molta attenzione quando lavori con il database.
