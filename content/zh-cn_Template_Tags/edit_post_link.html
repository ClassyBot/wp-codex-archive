概述[edit]

用户登录成功且具有编辑日志的权限，则通过该函数来调用当前日志编辑的页面。 该函数必须在The Loop内使用。

通过get_edit_post_link来获取URL.

用法[edit]

 <?php edit_post_link( $link, $before, $after, $id ); ?> 

传入参数[edit]

link
(string) (optional) The link text.
Default: __('Edit This')
before
(string) (optional) Text to put before the link text.
Default: None
after
(string) (optional) Text to put after the link text.
Default: None
id
(integer) (optional) Post ID (added in WordPress 2.8).
Default: None

使用样例[edit]

默认用法[edit]

<?php edit_post_link(); ?>

Display Edit in Paragraph Tag[edit]

Displays edit 日志 link, with link text "edit", in a paragraph (<p>) tag. 

<?php edit_post_link('edit', '<p>', '</p>'); ?>

Changelog[edit]
