플러그인은 워드프레스의 기능을 확장할 수 있는 도구입니다. 워드프레스의 핵심은 유연성을 극대화하고 덩치큰 코드를 최소화할 수 있도록 설계되어 있습니다. 플러그인은 사용자정의 함수와 같이 자신만의 특정한 요구 사항들을 사용자에게 맞춤 제공합니다.

이 문서는 플러그인과 플러그인 개발에 관련된 포괄적인 참고자료(레퍼런스)의 목록입니다.

플러그인 레퍼런스[edit]

  • 플러그인 관리 - 검색, 설치, 관리 및 문제를 해결하는 방법
  • 플러그인 - 워드프레스 플러그인 목록, 기타 플러그인 링크
  • 플러그인/플러그인 호환성 - 워드프레스 버전에 맞는 호환되는 플러그인 목록

플러그인 개발의 기초[edit]

  • 플러그인 작성 - 플러그인을 개발하는 최상의 학습법
  • 워드프레스 개발자를 위한 I18n - 국제화, 당신의 플러그인을 국제화하는 방법
  • 플러그인 API - 당신의 플러그인에서 후크(액션과 필터)와, 워드프레스 함수를 오버라이딩하여 사용하는 방법.
  • 관리 메뉴 추가 - 당신의 플러그인에 관리 메뉴를 추가하는 방법
  • 플러그인 배포 및 프로모션 - 당신이 만든 플러그인을 널리 배포하기 위한 몇가지 팁
  • 플러그인 및 테마의 마이그레이션 - 새로운 워드프레스 버전에서도 작동할 수 있도록 업그레이드 하는 방법
  • 플러그인 및 컨텐츠의 디렉토리 - 서버, 워드프레스에서 플러그인 파일/폴더가 설치될 위치와 예제코드

외부 리소스:

  • 당신의 첫 워드프레스 플러그인(비디오 데모)
  • NEW 플러그인 해부학 (이탈리아어 번역)
  • 워드프레스 플러그인 작성 PART1 및 PART2
  • 워드프레스 플러그인의 개발이라오 (포르투칼-브라질어)

플러그인 개발 - 레퍼런스[edit]

  • 플러그인 API/필터 레퍼런스 - 워드프레스에서 사용할 수 있는 필터 후크 레퍼런스
  • 플러그인 API/액션 레퍼런스 - 워드프레스에서 제공하는 액션 후크 레퍼런스
  • 함수 레퍼런스 - 워드프레스를 위한 완벽한 PHP 함수 레퍼런스(2.1버전, 이전 버전 링크 포함)
  • 워드프레스 공식 PHPDoc - 워드프레스의 거의 모든 기능이 여기 "PHPDoc" 에 나와있습니다. 이 사이트는 PHPDoc 의 모든 것을 보여줍니다.
  • 옵션 레퍼런스 - 워드프레스에서 기본적으로 설치하는 옵션 목록(2.3.1버전, 이전 버전 링크 없음)
  • 데이터베이스 설명 - 워드프레스의 데이터베이스에 대한 설명(2.1버전; 이전 버전 링크 포함)
  • 쿼리 개요 - 게시물을 검색하고 표시하는 워드프레스 쿼리 프로세스에 대한 설명
  • 워드프레스 표준 코딩 - 워드프레스 개발을 위한 표준코딩에 대한 일반적인 정보
  • 워드프레스 함수 사용 금지 후크 - 워드프레스 2.4에 추가된 함수사용을 금지하는 두가지 후크와 그것의 사용 방법.
  • 단축 API - 단축API 를 위한 강의와 레퍼런스(새로운 2.5버전)
  • 대쉬보드 위젯 API - 관리자 대쉬보드에 새로운 위젯을 추가하기 위한 예제와 레퍼런스
  • API 설정 - 기존의 설정화면에 새로운 설정기능을 추가하기 위한 예제와 레퍼런스
