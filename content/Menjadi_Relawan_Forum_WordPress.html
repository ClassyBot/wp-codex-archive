Selamat Datang[edit]

Dukungan Relawan Forum adalah anggota dari WordPress Support Forum memberikan jawaban atas masalah WordPress Anda dan isu-isu.

Jika Anda tertarik untuk menjadi sukarelawan dan membantu di forum, tidak ada kriteria. Hanya diterjunkan di mana Anda inginkan. Dianjurkan bahwa Anda memiliki yang berikut:

  • Keakraban dengan fungsi dan fitur WordPress
  • Keakraban dengan terminologi WordPress
  • Kemampuan untuk memahami maksud di balik pertanyaan itu, bukan hanya pertanyaan (tidak semua orang bisa menulis bahasa Inggris dengan baik sehingga kadang-kadang kita harus menafsirkan atau mengajukan pertanyaan lebih)
  • Kesabaran
  • Kesediaan untuk pergi langkah ekstra

Pedoman untuk semua orang menggunakan forum dapat ditemukan di Menggunakan Support Forum. Untuk menjaga dengan berita dan informasi tentang kegiatan di Forum, dan bertanya tentang relawan, bergabung #forums pada Slack dan / atau mendaftar untuk [http: / /lists.automattic.com/mailman/listinfo/wp-forums wp-forum mailing list].

Untuk pertanyaan, diskusi dan informasi lainnya, klik Diskusi Tab di bagian atas halaman ini.

Menghubungkan ke Docs[edit]

Hal ini sering nyaman untuk link ke halaman dari bagian Docs saat membalas di forum. Berikut adalah cara pintas kecil yang menyenangkan: Bungkus alamat Page dalam tanda kurung ganda. Tidak URL lengkap, hanya Page.

Misalnya, jalan pintas cepat ke halaman Permalinks Menggunakan akan diketik seperti ini: < pre> Using_Permalinks </ pre> Ketika Anda menyimpan pos itu akan dikonversi ke link yang tepat.

Catatan: Ini hanya bekerja dengan halaman tanpa para di link mereka.

Moderasi Bantuan[edit]

Jika Anda melihat topik yang Anda merasa perlu perhatian Moderator, tambahkan tag "modlook" untuk itu. Secara khusus, perhatikan beberapa hal berikut:

  • Duplikat posting
  • Berlebihan (atau prematur) benjolan
  • Spam & SEO upaya
  • Sandi disengaja atau kunci berbagi data
  • Non-WordPress terkait permintaan

Perhatikan bahwa tidak ada kebutuhan untuk menambahkan tag tambahan atau untuk mencoba untuk menjelaskan masalah. Moderator cerdas; mereka akan melihat masalah ini untuk diri mereka sendiri. Menambahkan penjelasan hanya lebih banyak pekerjaan bagi mereka untuk membersihkan juga.

Moderator: setelah menyerahkan topik modlook, pastikan untuk 'menghapus' tag!
