Обзор[edit]

На этой странице перечислены все API, связанные с базой данных. Каждый раздел покрывает функции, связанные с названной функциональностью. Все вместе они образуют то, что можно назвать API доступа к базе данных WordPress, который является интерфейсом к плагинам / темам / дополнениям, созданными проектом WordPress и обеспечивает доступ к данным, как к именованным значениям, сохранённым в слое базе данных.

Если вы прочитали всё это, вы должны иметь хорошее представление, как расширить WordPress посредством его плагинов, используя простой доступ к базе данных.
