설명[edit]

데이터베이스에서 회원 정보를 수정 한다.

가능하도록 세팅이 되어있다면 'user_pass'를 $userdata에 사용하여 비밀번호를 변경할 수 있다.

$userdata가 'ID' 키를 가지지 않는다면, 새로운 회원이 생성되며 생성된 회원의 'ID'를 리턴한다.

현재 회원의 비밀번호가 수정되면 쿠키는 지워진다.

참고: wp_insert_user()함수로 새로운 회원을 만들 수 있다.

사용법[edit]

<?php wp_update_user( $userdata ) ?>

파라메터(인자)[edit]

$userdata
(array) (required) 회원 데이터의 배열.
Default: None

결과값[edit]

UserID (integer) 
수정 된 회원의 ID 값.

예[edit]

<?php
require_once( ABSPATH . WPINC . '/registration.php');
$user_id = 1;
$website = 'http://wordpress.org';
wp_update_user( array ('ID' => $user_id, 'user_url' => $website) ) ;
?>

참고[edit]

  • Uses: get_userdata()
  • Uses: wp_hash_password()
  • Uses: wp_insert_user() Used to update existing user or add new one if user doesn't exist already
  • Uses: wp_get_current_user()
  • Uses: wp_clear_auth_cookie()
  • Uses: wp_set_auth_cookie()
