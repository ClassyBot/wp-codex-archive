함수 설명[edit]

카테고리 계층 구조에 따라 부모 카테고리의 목록을 반환합니다.

사용 방법[edit]

 <?php get_category_parents( $id, $link, $separator, $nicename, $visited ); ?> 

Parameters[edit]

$id
(integer) (required) 카테고리 ID.
Default: None
$link
(boolean) (optional) 각 카테고리의 링크를 표시할지 여부
Default: false
$separator
(string) (optional) 각 카테고리를 구분하는 구분자(Seperator).
Default: '/'
$nicename
(boolean) (optional) nice name을 표시할지 여부
Default: false
$visited
(boolean) (optional) Already linked-to categories. This parameter is used internally by the function, which makes recursive calls to itself, to prevent duplicates in the returned list.
Default: array()

반환값[edit]

(string|WP_Error) 
구분자 $separator로 나눠진 카테고리 문자열. 실패 시 WP_Error 반환.

예제[edit]

Returns the parent categories of the current category with links separated by '»' 

<?php echo get_category_parents( $cat, true, ' &raquo; ' ); ?>

will output:

Internet » Blogging » WordPress »

변경 이력[edit]

  • Since: 1.2.0

Source File[edit]

get_category_parents() is located in wp-includes/category-template.php
