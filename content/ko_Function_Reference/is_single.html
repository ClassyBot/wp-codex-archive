설명[edit]

이 상황 태크는 현재 보고있는 페이지가 싱글 페이지(포스트 하나만을 보여주는 페이지)인지 파악하며, 싱글일 때에는 TRUE를 아닐 때에는 FALSE를 리턴한다.

사용법[edit]

<?php is_single($post); ?>

인자[edit]

$post
(mixed) (optional) 포스트의 ID 또는 타이틀, 슬러그
Default: None

결과 값[edit]

(boolean) 
싱글일 때에는 True, 아닐 때에는 false.
