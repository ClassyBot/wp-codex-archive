설명[edit]

현재 포스트(글) 또는 포스트(글) ID를 가지고 해당 포스트(글)의 전체 주소를 가져옵니다.

사용법[edit]

<?php get_permalink( $id, $leavename ); ?>

인자값[edit]

$id
(int) (optional) 옵션. 포스트(글) ID.
Default: 0
$leavename
(boolean) (optional) 옵션, 기본값은 false(거짓) 입니다. 게시물 이름 또는 페이지 이름을 유지할지 여부를 선택합니다.
Default: false

반환값[edit]

(string) 

변경 로그[edit]

Since: 1.0

소스 파일[edit]

get_permalink() 는 wp-includes/link-template.php 에 있습니다.
