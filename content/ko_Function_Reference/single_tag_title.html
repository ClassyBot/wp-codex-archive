설명[edit]

현재 아카이브(archive) 페이지를 위한 태그 타이틀(제목)을 보거나 반환합니다.

사용법[edit]

 <?php single_tag_title( $prefix, $display ); ?> 

기본사용법[edit]

 <?php single_tag_title(); ?> 

인자값[edit]

$prefix
(string) (optional) 타이틀(제목)앞에 붙을 텍스트
Default: None
$display
(boolean) (optional) 타이틀을 보려면, (TRUE), PHP에서 사용하기 위해서 값을 반환받으려면 (FALSE).
Default: TRUE

예제[edit]

tag title을 이용하여 다음과 같이 따라하면 이 예제와 같이 "Currently browsing " 글자를 반환하는 예제를 볼 수 있습니다. 

<p><?php single_tag_title('Currently browsing '); ?>.</p>
Currently browsing WordPress.


이 예제는 현재 태그 타이틀(제목)을 $current_tag 변수에 지정하여 PHP에서 사용할수 있도록 합니다. 

<?php $current_tag = single_tag_title("", false); ?>

변경로그[edit]

  • 3.1.0: uses single_term_title()
  • Since: 2.3.0

소스파일[edit]

single_tag_title()은(는) wp-includes/general-template.php에 있습니다.
