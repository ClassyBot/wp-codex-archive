설명[edit]

이메일의 유효성 확인

사용방법[edit]

<?php is_email( $email ) ?>

파라메터(인자)[edit]

$email
(string) (required) 체크할 이메일 주소
Default: None

결과값[edit]

(string|bool) 
이메일이 없다면 false, 있다면 해당 이메일.

예[edit]

<?php
if (is_email('email@domain.com')) {
      echo 'email address is valid.';
}
?>
