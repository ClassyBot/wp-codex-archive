설명[edit]

포스트가 가진 태그의 집합을 리턴해줘요. 태그의 ID를 키로 하는 배열 안에 오브젝트로 정보가 들어가 있어요.

The Loop안에서 사용하면 루핑되는 포스트에서 자동으로 뽑아주죠. 루프 밖에서 사용하면 반드시 파라메터를 넣어주세요.

사용법[edit]

get_the_tags($post_id);

파라메터[edit]

$id
(int) (필수항목이 아니에요.) 포스트의 아이디.
Default: None

리턴 값[edit]

(array) 
태그 오브젝트의 배열

멤버 값[edit]

term_id 
태그의 아이디
name 
태그의 이름
slug 
태그의 슬러그
term_group 
태그의 그룹
taxonomy 
이 경우에는 'post_tag'만 나온답니다. get_the_terms()를 사용하면 다른 값이 나오겠죠.
description 
태그의 설명
count 
이 태그가 몇 개의 포스트에서 사용되는지

사용 예[edit]

태그를 드롭다운으로 보여줘요[edit]

function drop_tags()
{
    echo "<select onChange=\"document.location.href=this.options[this.selectedIndex].value;\">";
    echo "<option>Tags</option>\n";
    foreach (get_the_tags() as $tag)
    {
        echo "<option value=\"";
        echo get_tag_link($tag->term_id);
        echo "\">".$tag->name."</option>\n";
    }
    echo "</select>";
}
