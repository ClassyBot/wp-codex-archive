설명[edit]

WordPress가 생성한 웹페이지에 자바스크립트를 삽입합니다. 같은 스크립트를 여러번 삽입해도 한 개의 스크립트만을 로딩합니다.

사용법[edit]

wp_enqueue_script( 
     $handle
    ,$src
    ,$deps
    ,$ver
    ,$in_footer 
);

이 함수를 호출하기 위해 wp_enqueue_scripts 액션을 사용합니다. 혹은 Admin 모드에서 호출하기 원한다면 admin_enqueue_scripts를 사용합니다.
