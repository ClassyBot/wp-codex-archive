함수 설명[edit]

카테고리의 이름을 가지고 ID를 얻습니다.

사용 방법[edit]

<?php get_cat_ID( $cat_name ) ?>

매개변수[edit]

$cat_name
(string) (optional) 기본값은 'General' 이며, 다른 어떤 카테고리 이름이든지 사용할 수 있습니다.
Default: 'General'

반환값[edit]

(0이면 integer, ID이면 string) 
실패하면 0, 성공하면 카테고리의 ID를 반환합니다.

기초[edit]

이 예제는 루프 안에서 이 함수를 사용하는 방법을 소개하는 아주 간단한 예제입니다. "Category Name" 자리에 당신이 사용할 카테고리 이름을 입력하세요. 

<?php
$category_id = get_cat_ID('Category Name');
 <!-- Start the Loop. -->
 <?php if ( have_posts() ) : while ( have_posts() ) : the_post(); ?>

 <!-- Test if the current post is in category "Category Name". -->
 <!-- If it is, the div box is given the CSS class "post-cat-special". -->
 <!-- Otherwise, the div box is given the CSS class "post". -->

 <?php if ( in_category($category_id) ) { ?>
           <div class="post-cat-special">
 <?php } else { ?>
           <div class="post">
 <?php } ?>
</div>
 <!-- Stop The Loop (but note the "else:" - see next line). -->

 <?php endwhile; else: ?>


 <!-- The very first "if" tested to see if there were any Posts to -->
 <!-- display.  This "else" part tells what do if there weren't any. -->
 <p>Sorry, no posts matched your criteria.</p>


 <!-- REALLY stop The Loop. -->
 <?php endif; ?>
