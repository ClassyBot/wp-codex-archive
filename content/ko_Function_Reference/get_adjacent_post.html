함수 설명[edit]

인접한 포스트를 반환합니다. 이전 포스트 일 수도 있고 다음 포스트 일 수도 있습니다.

사용 방법[edit]

<?php get_adjacent_post( $in_same_term, $excluded_terms, $previous, $taxonomy ) ?>

파라미터[edit]

$in_same_term
(boolean) (선택) 같은 taxonomy term인가를 정합니다. 기본 값은 'false'입니다.
Default: false
$excluded_terms
(array or string) (선택) Array 값이나 쉼표로 나누어진 term ID 값을 제외합니다.기본 값은 비어있습니다.
Default: ''
$previous
(boolean) (선택) 이전 포스트를 반환할지 정합니다. 기본 값은 'true'입니다.
Default: true
$taxonomy
(string) (선택) $in_same_term 값이 true 라면, taxonomy 값을 정해 원하는 값을 불러옵니다. 기본 값은 'category' 입니다.
Default: 'category'

반환 값[edit]

  • 만약 반환할 포스트가 있다면, 그 포스트를 반환합니다.
  • 글로발(global) $post 값이 지정되어 있지않다면 null 값을 반환합니다.
  • 해당하는 포스트가 없다면 비어있는 String 값을 반환합니다.

예시[edit]

같은 taxonomy 중 이전 포스트의 제목과 링크를 호출하는 방법 

 <?php $prev_post = get_adjacent_post( true, '', true, 'category_name' ); ?>
 <?php if ( !empty( $prev_post ) ): ?>
 	<a href="<?php echo $prev_post->guid; ?>"><?php echo $prev_post->post_title; ?></a>
 <?php endif; ?>

같은 taxonomy 중 다음 포스트의 제목과 링크를 호출하는 방법 

 <?php $next_post = get_adjacent_post( true, '', false, 'category_name' ); ?>
 <?php if ( !empty( $next_post ) ): ?>
 	<a href="<?php echo $next_post->guid; ?>"><?php echo $next_post->post_title; ?></a>
 <?php endif; ?>

메모[edit]

  • 이 글로발(global) 값을 사용합니다: (object) $post
  • 이 글로발(global) 값을 사용합니다: (object) $wpdb

필터[edit]

$adjacent 값은 '이전' 이나 '다음' 입니다.

  • "get_{$adjacent}_post_join":
    $join, $in_same_cat, $excluded_categories
  • "get_{$adjacent}_post_where":
    $wpdb->prepare("WHERE p.post_date $op %s AND p.post_type = %s AND p.post_status = 'publish' $posts_in_ex_cats_sql", $current_post_date, $post->post_type), $in_same_cat, $excluded_categories
  • "get_{$adjacent}_post_sort":
    "ORDER BY p.post_date $order LIMIT 1"
