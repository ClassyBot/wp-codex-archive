설명[edit]

유저 데이터 필드와 값이 일치하는 유저의 정보를 얻는다.

사용법[edit]

<?php get_user_by($field, $value) ?>

파라메터 (인자)[edit]

$field
(string) (required) 'id', 'slug', 'email', 'login' 등 유저 데이터 필드
Default: None
$value
(string|integer) (required) 값
Default: None

리턴되는 결과값[edit]

(mixed) 
유저 오브젝트. $field가 존재하지 않는다면 false를 리턴한다.
