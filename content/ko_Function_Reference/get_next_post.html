함수 설명[edit]

현재 포스트를 기준으로, 다음 포스트를 반환합니다.

사용 방법[edit]

<?php get_next_post( $in_same_term, $excluded_categories, $taxonomy ) ?>

파라미터[edit]

$in_same_term
(boolean) (optional) Whether post should be in same category.
Default: false
$excluded_categories
(string) (optional) 제외할 카테고리(복수).
Default: ''
$taxonomy
(string) (optional) Taxonomy, if $in_same_term is true. Added in WordPress 3.8.
Default: 'category'

반환 값[edit]

  • 성공적으로 수행하면 Post 객체 를 반환합니다.
  • 전역 변수 $post 가 설정되어 있지 않으면 Null 값을 반환합니다.
  • 적합한 포스트가 존재하지 않으면 빈 문자열을 반환합니다.

예제[edit]

<?php
$next_post = get_next_post();
if (!empty( $next_post )): ?>
  <a href="<?php echo get_permalink( $next_post->ID ); ?>"><?php echo $next_post->post_title; ?></a>
<?php endif; ?>

소스 파일[edit]

get_next_post() 함수는 wp-includes/link-template.php 파일에 위치합니다.
