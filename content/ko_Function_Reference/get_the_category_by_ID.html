설명[edit]

카테고리 ID 부터 카테고리명 을 가져옵니다.

사용예[edit]

<?php get_the_category_by_ID( $cat_ID ); ?>

Parameters[edit]

$cat_ID
(int) (필수) Category ID.
Default: None

Return Values[edit]

(string) 
카테고리명. cat_ID가 없을경우 WP_Error를 반환합니다. 자세한 내용은 is_wp_error() 을 참고합니다.

변경이력[edit]

Since: 0.71

Source File[edit]

get_the_category_by_ID() 는 wp-includes/category-template.php 에 있습니다.
