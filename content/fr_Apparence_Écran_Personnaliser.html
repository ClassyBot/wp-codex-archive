Apparence →Personnaliser[edit]

Identité du site[edit]

Ce réglage vous permet de définir l'identité du site. Voici les options disponibles:

  • Logo - permet d'ajouter ou de changer le logo (affiché selon le thème, souvent dans l'en-tête de la page).
  • Titre du site - permet de changer le Titre du site.
  • Slogan - permet de changer le Slogan.
  • Icône du site - permet d'ajouter une icône (utilisée pour représenter votre site en tant qu’icône de navigateur (favicon) et icône d’application).

Couleurs[edit]

Permet de sélectionner un jeu de couleurs (si le thème en propose), de sélectionner une couleur d'arrière plan pour l'ensemble du site, ou encore des couleurs de texte et de fond pour des zones spécifiques.

Image d'en-tête[edit]

Cette option permet de choisir une image de fond, qui sera appliquée à l'en-tête sur les petits écrans, et à la barre latérale sur les grands écrans.

Image d’arrière-plan[edit]

Cette option permet de choisir une image qui sera utilisée comme arrière-plan.

Menus[edit]

Permet d'administrer les Menus. Voir fr:Apparence Écran des Menus.

Widgets[edit]

Permet d'ajouter et de paramétrer les Widgets. Voir fr:Apparence Écran des Widgets.

Page d’accueil statique[edit]

Votre page d'accueil peut afficher soit les derniers articles, soit une Page statique. Ce réglage vous permet de définir une Page d’accueil statique (et une Page des articles). Voir Créer une page d'accueil statique (en anglais).

CSS Additionnel[edit]

Cette section vous permet d'ajouter du CSS personnalisé. Le code CSS vous permet de modifier librement l’apparence et la mise en page de votre site. Votre CSS personnalisé sera sauvegardé séparément pour chacun de vos thèmes.

Cette possibilité est pratique pour effectuer de petites surcharges ou retouches. Si vous avez besoin de faire de nombreuses modifications, il sera plus sage de recourir à un thème enfant.

Enregistrer & publier[edit]

Tous les changements que vous faites sur l'écran Personnaliser sont prévisualisés en temps réel. Mais n'oubliez pas de cliquer le bouton "Enregistrer & publier", afin que vos modifications soient conservées. Après l'avoir cliqué, le bouton indiquera le statut "Enregistré".

Voir également panneaux d'administration et administration réseau (en anglais).
