Description[edit]

This creates a link to the next comments page containing newer comments.

Usage[edit]

 <?php next_comments_link( 'Label', 'Max number of pages (default 0)' ); ?> 

Examples[edit]

Default Usage[edit]

<?php next_comments_link(); ?>

Working example[edit]

<?php next_comments_link( 'Newer Comments »', 0 ); ?>

Parameters[edit]

label 
(string) The link text. Default is 'Newer Comments »'.
max_pages 
(integer) Limit the number of pages on which the link is displayed.

Change Log[edit]

Since: 2.7.0
