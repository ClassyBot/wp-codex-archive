Description[edit]

Displays the content of the current post formatted for a Feed. This tag must be within The_Loop.

Usage[edit]

<?php the_content_feed( $feed_type ) ?>

Parameters[edit]

$feed_type
(string) (optional) The type of feed. rss2, atom, rss or rdf
Default: null

Examples[edit]

<?php the_content_feed(); ?>

Change Log[edit]

Since: 2.9.0

Source File[edit]

the_content_feed() is located in wp-includes/feed.php.
