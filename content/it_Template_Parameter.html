Descrizione[edit]

Questo template serve per standardizzare il modo in cui i parametri appaiono in Riferimento funzioni e in Tag dei template.

Uso[edit]

{{it:Parameter|nome|tipo|descrizione|importanza|default}}

Parametri[edit]

nome
Il nome del parametro.
tipo
Il tipo che il valore di questo parametro deve assumere.
  • string
  • integer
  • boolean
  • mixed
descrizione
Una breve descrizione del parametro.
importanza
Impostare questo parametro a opzionale se il parametro è opzionale. Altrimenti non dichiararlo - avrà come valore predefinito richiesto.
  • richiesto
  • opzionale
default
Se questo parametro è opzionale, default è il valore che sarà usato se il parametro non è dichiarato.

Esempio[edit]

{{it:Parameter|$nome|string|Questa è una stringa.|opzionale|bah}}

Output[edit]

$nome
(string) (opzionale) Questa è una stringa.
Default: bah
