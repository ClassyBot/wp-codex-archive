Descrição[edit]

Verifica se qualquer filtro foi registrado para um gancho (hook).

Utilização[edit]

 <?php has_filter( $tag, $funcao_a_verificar ); ?> 

Parâmetros[edit]

$tag
(string) (obrigatório) O nome de gancho(hook) do filtro.
Default: None
$funcao_a_verificar
(callback) (opcional) Se especificado, retorna a prioridade desta função neste gancho (hook). Se a função especificada não está anexada a este ganho (hook), retorna false.
Default: False

Retorno[edit]

inteiro | boleano 
Se nenhuma função for especificada: retorna true se qualquer função estiver registrada para este gancho (hook), ou false se não houver. Se uma função for especificada (como segundo parâmetro): retorna um inteiro para a prioridade do gancho (hook) se a função estiver registrada, ou false se não estiver.

Exemplos[edit]

<?php
if ( ! has_filter( 'the_content', 'example_alter_the_content' ) )
    add_filter( 'the_content', 'prefix_alter_the_content' );
?>

Notas[edit]

  • Utiliza a array global $wp_filter que guarda todos os filtros.

Log de mudanças[edit]

  • Desde: 2.5.0

Arquivo Fonte[edit]

A função has_filter() está localizada em wp-includes/plugin.php.
