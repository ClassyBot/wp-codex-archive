Criando Novo Artigo[edit]

Se você já se registrou e até possui uma Página de Usuário pode começar a criar seus próprios artigos sobre o WordPress


1. Digite na barra de endereço do seu navegador 

http://codex.wordpress.org/pt-br:Meu Artigo

Aparecerá uma página dizendo:

Você seguiu um link para uma página que não existe.

Para criá-la, comece escrevendo na caixa abaixo (veja a página de ajuda para mais informações).
Se você chegou aqui por engano, apenas clique no botão voltar do seu navegador.

2. Comece adicionando {{Codex-pt}} na caixa de texto. 3. Crie as seções usando ==Seção== e subseções ===Seção===. 4. Lembre-se de criar seções fáceis de entender e usar CSS e Modelos para delimitar seções especiais.

Criando um Artigo do Original em Inglês[edit]

  1. Abra um artigo original inglês.
  2. Clique em Editar do lado direito da tela e adicione o nome do artigo em português para o novo artigo traduzido. Ex.: 
 {{Languages|
 {{en|Administration Panels}}
 {{ja|管理パネル}}
 {{zh-hans|管理面板}}
 }}

Fica assim: 

 {{Languages|
 {{en|Administration Panels}}
 {{ja|管理パネル}}
 {{pt-br|Painéis de Administração}}
 {{zh-hans|管理面板}}
 }}

Lembre-se:adicione a referência para o novo artigo na ordem alfabética, geralmente logo após a versão em japonês e use sempre letras maiúsculas nas palavras como Painéis de Administração e não Painéis de administração ou painéis de administração.

  1. Selecione tudo teclando Ctrl+a e copie teclando Ctrl+c. Depois clique no botão Salvar página abaixo da área de edição de texto. Você será redirecionado para o artigo novamente.
  2. Clique no link vermelho que agora aparece na caixa de idiomas para criar o artigo em português.
  3. Tecle Ctrl+v para colar o artigo original.
  4. Adicione {{Codex-pt}} no final do artigo.
  5. Clique no botão Salvar página para salvar.

Dicas[edit]

  • O Tradutor do Google está bem aprimorado, você pode usá-lo para traduzir o Codex Brasil. Selecione um bloco de texto e traduza enquanto lê o original. Você deve observar os códigos wiki, porque o Google pode alterá-lo às vezes.

O que é permitido[edit]

  • Adicionar links de seu site - desde que seja relacionado ao artigo e somente no final do artigo. Não é aconselhável misturar links privados com instruções do sistema WordPress, por exemplo:
No WordPress gerencie tags em ...mas você pode usar o plugin do Renato Brandão para exportar etc

O mais aceito é inserir uma lista de links no final do artigo com ou sem instruções, descrições adicionais. Essa medida previne guerras de edição.

  • Adicionar imagens de sites privados - desde que obedecendo o contexto, sem intuito de gerar publicidade.
