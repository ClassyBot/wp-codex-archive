ขั้นตอนที่ 1 - อ่าน[edit]

ก่อนที่คุณจะใช้เวลาอันมีค่าของคุณไปในการติดตั้งเวิร์ดเพรส มีเอกสารบางอย่างที่คุณต้องอ่านก่อน เวิร์ดเพรสเป็นโปรแกรมชั้นเยี่ยม ใช้งานได้ง่าย และมีความสามารถมากมาย แต่มันอาจจะไม่ใช่โปรแกรมที่ดีพร้อมสำหรับทุกคน เหมือนกับการสร้างบ้าน คุณต้องใช้เครื่องมือให้ถูกต้องสำหรับแต่ละงาน กรุณาอ่านเอกสารต่อไปนี้ก่อน

  • About Weblogs - What is Blogging all about?
  • What is WordPress?
  • WordPress Features
  • ก่อนคุณทำการติดตั้ง

ขั้นตอนที่ 2 - วางแผนงาน[edit]

เมื่อคุณรู้ข้อมูลต่าง ๆ ของเวิร์ดเพรสรวมทั้งคู่มือการติดตั้งเวิร์ดเพรสแล้ว คุณควรจะมีรายชื่อสิ่งที่คุณต้องทำและสิ่งที่คุณต้องมีก่อนการติดตั้งเวิร์ดเพรส ถ้าคุณยังไม่มี คุณควรเริ่มทำรายชื่อพวกนั้น คุณจะได้มั่นใจได้ว่าทุกอย่างที่คุณควรจะมีได้ถูกรวมอยู่ในรายชื่อสิ่งที่คุณต้องทำแล้ว

  • Website Host Requirements Checked and Verified
  • Versions of PHP and MySQL Checked and Verified
  • ชื่อผู้ใช้และรหัสผ่านเวบไซต์ของคุณ
  • Text Editor Software
  • An FTP Client Software
  • เบราเซอร์ของคุณ (อะไรก็ได้)

คู่มือต่อไปนี้จะช่วยให้คุณเข้าใจเกี่ยวกับการทำงานของเวิร์ดเพรสและช่วยให้คุณวางแผนเกี่ยวกับเวบไซต์เวิร์ดเพรสของคุณ

  • WordPress Features
  • First Steps With WordPress
  • WordPress Lessons

เป็นการสำคัญมาที่คุณควรจะวางแผนก่อนการใช้งานเวิร์ดเพรสบนเวบไซต์ของคุณ นี่คือคำถามบางอย่างที่คุณควรถามตัวเอง ทำรายชื่อคำตอบเพื่อให้คุณสามารถเพิ่มสิ่งเหล่านี้ลงไปในแผนงานของคุณได้

  • คุณจะติดตั้งเวิร์ดเพรสลงในไดเรคทอรีหลัก (root directory), ไดเรคทอรีย่อย หรือคุณแค่ต้องการสร้างเวบไซต์ทดลองเพื่อต้องการดูว่าเวิร์ดเพรสใช่สิ่งที่คุณต้องการหรือไม่
  • คุณได้ทำรายชื่อ Categories ทำความเข้าใจว่าเวิร์ดเพรสสามารถเรียงลำดับหมวดหมู่ตามตัวอักษรหรือตามเลขที่หมวดหมู่เท่านั้น (การเรียบลำดับผ่านทางหน้าจอ Manage > Categories ) ดังนั้นถ้าการแสดงผลการเรียงลำดับของหมวดหมู่ของคุณนั้นจำเป็นสำหรับคุณ โปรดเตรียมรายชื่อสำหรับหมวดหมู่บนเวบไซต์ของคุณ
  • คุณได้ทำรายชื่อของ Pages (หน้า) ซึ่งคุณอาจจะต้องการเพิ่มสิ่งนี้ลงในเวบไซต์ของคุณ เช่นหน้า เกี่ยวกับเวบไซต์นี้, ติดต่อเรา หรือ เหตุการณ์ต่าง ๆ

ขั้นตอนที่ 4 - ตั้งค่าเวิร์ดเพรส[edit]

เมื่อคุณทำการติดตั้งเรียบร้อยแล้ว ขั้นตอนต่อไปคือการตั้งค่าเวิร์ดเพรสเพื่อให้เวิร์ดเพรสทำงานอย่างที่คุณต้องการ ทันทีที่เปลี่ยนแปลงการตั้งค่าบางอย่าง เราแนะนำให้คุณสังเกตถึงการเปลี่ยนแปลงของเวบไซต์คุณ โดยกดดูที่ลิงก์ เยี่ยมชมเวบไซต์ ด้านบนของหน้าจอ Administration แม้ว่าคุณอาจจะเลือกที่จะทำขั้นตอนต่อไปนี้ตามคู่มือ แต่เวบไซต์ของคุณอาจะมีปัญหาบ้างเล็กน้อยถ้าคุณทำการเปลี่ยนแปลงการตั้งค่าในสิ่งต่อไปนี้:

  • Administering Your Blog
  • Users > Your User Profile - set the user information you want published on your site
  • Your User Profile > Other Users - add authors and users that will be using your site, if applicable
  • Options > General - set your site name and other site information
  • Options > Writing - set the settings of your Write Post screen
  • Options > Reading - set how many posts to show on the front page and in categories and your feed requirements
  • Options > Discussion - Turn on or off comments and set how to handle them
  • Manage > Categories - add a few categories to get started from your category list
  • Manage > Posts - After you have written a few posts, this is where you will manage them by editing or deleting
  • Presentation > Themes - maybe change the look of your site?
  • Manage > Pages - add a Page or two like "About Us" or "Contact Me"
  • Write > Write Post - start adding content to your site
  • Writing Posts - step-by-step instructions on writing posts

การเพิ่มปลั๊กอิน[edit]

มีเครื่องมือและโปรแกรมมากมายสำหรับเวิร์ดเพรสที่เรียกว่า ปลั๊กอิน ที่จะเพิ่มความสามารถ ทางเลือกและการทำงานต่าง ๆ ให้กับเวบเวิร์ดเพรสของคุณ ปลั๊กอินของเวิร์ดเพรสสามารถทำสิ่งต่าง ๆ ได้มากมาย รวมทั้ง การแก้ไขการแสดงข้อมูลของเวบไซต์คุณ เพิ่มการแสดงผลดินฟ้าอากาศ การเช็คคำผิดและการแสดงผลของบทความและสิ่งต่าง ๆ ในหลาย ๆ วิธี โปรดอ่านบทความต่อไปนี้เพื่อเรียนรู้ว่าปลั๊กอินของเวิร์ดเพรสทำงานอย่างไรและเราสามารถหาปลั๊กอินเพิ่มเติมได้ที่ไหนบ้าง:

  • จัดการปลั๊กอิน

ต้องการความช่วยเหลือ[edit]

การใช้งานเวิร์ดเพรสนั้นเรียบง่ายมาก ถ้ามีปัญหาเกิดขึ้น ถ้าคุณสงสัยในบางอย่าง ถ้าระบบบางอย่างไม่ทำงาน อย่าวิตกไปเพราะคุณสามารถขอความช่วยเหลือได้ แม้ว่าเวิร์ดเพรสจะเป็นโปรแกรมโอเพ่นซอร์ส (โปรแกรมฟรี) แต่ก็มีอาสาสมัครจำนวนมากที่พร้อมจะช่วยเหลือคุณ นี่คือแหล่งที่คุณสามารถหาความช่วยเหลือได้

  • เวบบอร์ด เวิร์ดเพรสไทย
  • Wordpress.org Forum

สุดท้ายนี้[edit]

ตอนนี้คุณก็กลายเป็นผู้ใช้เวิร์ดเพรสเต็มตัว โปรดพิจารณาถึงการอุทิศตัวให้กับคู่มือเวิร์ดเพรส การตอบปัญหาในเวบบอร์ด การพัฒนาต่าง ๆ หรือการพยายามอื่น ๆ ที่จะช่วยให้เวิร์ดเพรสสามารถพัฒนาต่อไปเรื่อย ๆ ได้ เวิร์ดเพรสนั้นเป็นโปรแกรมฟรีที่ได้รับการสนับสนุนจากผู้สมัครใจทั้งหลาย ความช่วยเหลือของคุณไม่ว่าจะเป็นอะไรก็เป็นที่ต้องการของเรา
