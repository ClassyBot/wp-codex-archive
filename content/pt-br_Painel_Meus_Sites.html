Painel → Meus Sites[edit]

Ao clicar no item Meus Sites, você é levado para uma tela que lista os sites em que é administrador. Se estiver em um nível de usuário diferente, nada será listado.

Uma lista suspensa no topo da página referência o Site Principal. Isto é usado como um identificados ao fazer coisas como sair do sistema. O usuário é frequentemente direcionado ao backend do site primário da rede.

Cada site é listado na ordem em que foi criado e abaixo de cada nome, há links para o backend e área de adminstração de cada site.
