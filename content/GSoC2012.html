Introduction[edit]

Howdy Students! Google Summer of Code 2012 is a program in which Google sponsors exceptional college students to develop open source code under the guidance of mentoring open source projects.

Unfortunately, WordPress was not selected as a mentoring this year. Students participating in GSoC will need to choose from one of the other participating organizations. If you are interested in contributing to WordPress anyway, we'd love to have you! See Contributing to WordPress and this post for more information.
