Nota

A partir da versão 3.5, a o gerenciador de links (blogroll) ficará escondido para novas instalações e quaisquer instalações existentes do WordPress que não criaram nenhum link no gerenciados.

Se você está atualizando a partir de uma versão anterior do WordPress e possui link guardados no gerenciador, ele vai continuar a aparecendo no menu da esquerda, normalmente.

Se você gostaria de ativar o gerenciador em sua instalação você pode baixar e instalar o plugin Link Manager. Esta página ainda é relevante para qualquer usuário com um Gestor de ligações existentes ou com o plugin instalado.



Links → Adicionar Novo[edit]

Você pode adicionar ou editar links nesta tela ao digitar informações em cada caixa. Somente o endereço do link e o nome (o texto que você quer que seja exibido em seu site como o link) são campos obrigatórios.

Opções de Tela[edit]

Na aba Opções de Tela você pode escolher quais itens são mostrados ou não na tela atual. Clicando na aba Opções de Tela aparece uma lista de opções com uma caixa de seleção ao lado de cada uma. Marque a caixa para cada item que deseja exibir na tela, ou desmarque a caixa para não exibir esse item. Clique na aba Opções de Tela novamente para fechar este quadro.

Ajuda[edit]

Clique na aba Ajuda na parte superior à direita, para uma explicação sobre os recursos disponíveis na tela atual.

Adicionar novo link[edit]

Os três primeiros campos na tela são os campos mais importantes ao se adicionar links.

Nome
O nome ou título do link.
Endereço (URL)
Este é o lugar onde o link aponta. Às vezes chamado de URL, este é o destino, ou o site, que quer ir quando clicar no link. Por padrão, o valor http:// é colocada no campo para começar por você.
Descrição
Você pode opcionalmente, digitar uma descrição para seu link. Como a descrição será exibida em seu site depende do tema que você usa, alguns temas mostram a descrição e outros não.

Categorias[edit]

As categorias são exibidos em uma caixa de módulo que permite a seleção de uma ou mais Categorias de Links. Novas categorias também podem ser adicionadas simplesmente digitando um novo nome de categoria na caixa e clicando no link Adicionar nova categoria.

Destino[edit]

Este módulo com botões de opção permitem decidir como o link será visualizado.

  • _blank - abre o link em uma nova janela.
  • _top - Se o seu site usa quadros (frames), esta opção pode ser útil. O link será aberto na janela atual do navegador, em vez do quadro atual da página. Não se preocupe com esta opção se você não usar quadros, ou se você não sabe o que são quadros.
  • _none - A opção padrão. Isto irá abrir o destino na janela atual (ou o quadro atual se o site utiliza quadros).

Avançado[edit]

Os seguintes controles permitem a manipulação mais avançada de seus links.

Endereço da Imagem
Você pode especificar uma URL de uma imagem para associar com o link, a exibição da imagem depende de seu tema
Endereço do Feed RSS
Se você estiver criando um link para um site que contém feed RSS, você pode digitar a URL do feed aqui. Novamente, a exibição desse dados depende de seu tema.
Notas
anotações para si mesmo sobre o link.
Classificação
Você pode optar por avaliar um link de 0-9, onde 0 significa que o link está sem classificação. O que o sistema de classificação na verdade, depende do que você decidir.

Se o link é para uma pessoa, você pode especificar a sua relação com ela usando o formulário acima. Se você quiser saber mais sobre esta idéia consulte [XFN].
