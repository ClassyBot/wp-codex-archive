Unix/Linux işletim sistemine sahip çoğu hosting sağlayıcısında WordPress'i sorunsuz kullanabilirsiniz. WordPress çok özel bir konfigürasyon gerektirmez.

Sunucu Gereksinimleri[edit]

  • WordPress server requirements for Version 5.2+:
    • PHP version 5.6.20 or greater, PHP 7 is highly recommended
    • MySQL version 5.0.15 or greater or any version of MariaDB
    • (Optional) (Required for Multisite) Apache mod_rewrite module (for clean URIs known as Permalinks)

» Requirements for Older Versions

Tavsiye edilen kurulum[edit]

Linux altyapısına sahip güçlü bir platform, Apache veya NGINX web sunucusu kullanmanız tavsiye edilir. Bu özelliklere sahip neredeyse tüm sunucularda PHP ve MySQL çalışacaktır. Eğer hosting firmanız bahsedilen platformları ve mod_rewrite özelliğini desteklemiyorsa farklı bir hosting firması tercih etmenizi tavsiye ediyoruz.

Hosting firmanızın uzaktan bağlantılara izin vermesi çoğu WordPress özelliğinin çalışması için elzemdir. Eğer hosting firmanız dış HTTP bağlantılarını blokluyorsa WordPress'in çoğu parçası çalışmayacaktır.

Hosting Firmaları[edit]

Eğer hosting firması önerisine ihtiyacınız varsa aşağıdakileri deneyebilirsiniz:

  • WordPress Türkiye tarafından tavsiye edilen Alastyr, Turkticaret.Net ve Turhost'u deneyebilirsiniz. Belirtilen firmalar WordPress'i sorunsuz desteklemektedir.
  • WordPress destek forumlarını yada domain sorgulama web sitesinden araştırabilirsiniz.
