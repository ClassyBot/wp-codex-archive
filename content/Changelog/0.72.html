WordPress 0.72[edit]

Version .72 was fairly significant because it represented the most stable and capable version of WordPress since the project begun. With this version WordPress started expanding its userbase beyond the handful that used it in the early releases.

Features[edit]

  • Password protected posts
  • Cursor aware quicktags
  • Support for Blogger, Metaweblog, and MT APIs
  • New template
  • Easy options system
  • Blogger import
  • RSS feeds for comments
  • Conditional GET support
  • Geographical data support

Fixes[edit]

  • Improvements to Texturize engine
  • Image upload re-added
  • Better calendar
  • New smilies
