WordPress 0.71[edit]

  • Released June 9, 2003
  • Dev Blog Announcement

The second release of WordPress focused a lot of core code improvements, notably abstracting all of the database functions into a class.

Features[edit]

  • Major speedup, 300% faster than previous versions and b2
  • Drafts
  • Private posts
  • Open or close comments
  • Combined comments, trackbacks, and pingbacks
  • OPML import for links
  • Database abstraction class
  • Improved administration
  • New upgrade system
  • Ability to check weblogs.com for updated blogs

Fixes[edit]

  • SQL injection vulnerability
