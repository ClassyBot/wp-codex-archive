WordPress 1.2[edit]

  • "Mingus" released on May 22, 2004.
  • Dev blog announcement

WordPress 1.2 was significant because it represented a fully mature product with features above and beyond other tools in the market. It was released right after Movable Type had some issues with their licensing and many users were looking for a new system to move to. Version 1.2 was used by an order of magnitude more people than any other release.

Features[edit]

  • Plugin architecture
  • Custom fields
  • Sub-categories
  • Thumbnail creation
  • OPML import and export
  • Encrypted passwords
  • Localization and Unicode support
  • Comment management tools
  • Post preview
  • RSS import
  • Livejournal import
  • Ping more than one update service at once
