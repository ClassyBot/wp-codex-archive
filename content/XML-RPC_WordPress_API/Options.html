wp.getOptions[edit]

Retrieve blog options.

Parameters[edit]

  • int blog_id
  • string username
  • string password
  • array options: List of option names to retrieve. If omitted, all options will be retrieved.

Return Values[edit]

  • array
    • struct
      • string desc
      • string value
      • bool readonly

Errors[edit]

  • This method will only return white-listed options. If a non-white-listed option is included in options, it will be omitted from the response.

wp.setOptions[edit]

Edit blog options.

Parameters[edit]

  • int blog_id
  • string username
  • string password
  • array options: keys are option names, values are the new option values.

Return Parameters[edit]

  • array: the options with updated values.
    • struct: See #wp.getOptions

Errors[edit]

  • 403
    • If the user does not have the manage_options cap.
