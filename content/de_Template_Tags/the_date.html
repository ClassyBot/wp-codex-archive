Beschreibung[edit]

Zeigt oder gibt das Datum eines Beitrags oder einer Reihe an Beiträgen aus, wenn sie am gleichen Tag veröffentlicht wurden.

GESONDERTE NOTIZ: Wenn es mehrere Beiträge einer unter dem GLEICHEN TAG veröffentlichten Seite gibt, gibt the_date() nur das Datum des ersten Beitrags aus (d.h. die erste Instanz von the_date()). Um das Datum für Beiträge zu wiederholen, die am gleichen Tag veröffentlicht wurden, sollte das Template Tag the_time() oder get_the_date() (seit 3.0) mit einem einem Datums-spezifischen Formatierungsstring verwenden werden.
Verwende <?php the_time( get_option( 'date_format' ) ); ?>, um das in der Admin-Oberfläche eingestellte Datum hinzuzufügen.

Dieser Tag muss innerhalb des The Loop verwendet werden.

Benutzung[edit]

 <?php the_date( $format, $before, $after, $echo ); ?> 

Parameter[edit]

$format
(string) (optional) Das Format für das Datum. Standardmäßig bezieht es sich auf das in den WordPress-Optionen konfigurierte Datumsformat. Siehe de:Formatierung von Datum und Uhrzeit.
Default: F j, Y
$before
(string) (optional) Text, der vor dem Datum platziert wird.
Default: None
$after
(string) (optional) Text, der nach dem Datum platziert wird.
Default: None
$echo
(boolean) (optional) Zeigt das Datum an (TRUE) oder gibt das Datum zurück, das in PHP (FALSE) verwendet werden wird.
Default: TRUE

Ausgabe[edit]

(string|null) Null zum Anzeigen, Zeichenfolge zum Abrufen.

Beispiele[edit]

Standardmäßige Benutzung[edit]

Zeigt das Datum mit den Standardeinstellungen an. 

<?php the_date(); ?>

Datum als Jahr, Monat, Tag in der Überschrift[edit]

Zeigt das Datum in dem Format '2007-07-23' (Beispiel: 2004-11-30), innerhalb eines <h2> Tags. 

<?php the_date('Y-m-d', '<h2>', '</h2>'); ?>

Datum mit der $my_date Variable in der Überschrift[edit]

Gibt das Datum im Standardformat innerhalb eines <h2> Tags aus und weist es der Variablen $my_date Variable zu. Der Wert der Variablen wird dann mit dem Befehl PHP echo ausgegeben. 

<?php $my_date = the_date('', '<h2>', '</h2>', FALSE); echo $my_date; ?>

Notizen[edit]

  • Wirkt sich auf den Rückgabewert der Funktion is_new_day() aus.
