Lo shortcode Video permette di includere file video e riprodurli. È stato aggiunto in WordPress 3.6 e si usa così:

[video]

È anche possibile includere un file video, senza l'uso di questo shortcode, inserendo semplicemente l'indirizzo del file su una propria riga, isolato dal resto del contenuto: 

  Il mio magnifico contenuto

  http://my.movies.com/cool/movie/coolest.mov

  Altro magnifico contenuto

Uso pratico[edit]

C'è un vecchio post che ha un file video allegato nella Media Library, e si vuole usare il nuovo shortcode:

[video]

Si possiede l'URL di un MP4, presente nella Media Library o esterno, che si vuole riprodurre:

[video src="video-source.mp4"]

Si possiede un URL principale e altri per i vari tipi di file supportati da HTML5:

[video mp4="source.mp4" ogv="source.ogv" mov="source.mov"]

Opzioni[edit]

Sono supportate le seguenti opzioni di base:

src
(string) (opzionale) La sorgente del file video. Se non è indicata, verrà usato il primo file video allegato al post. Si possono usare le seguenti opzioni per definire tipi di file specifici, che verranno usati a seconda dei casi:
  • 'mp4', 'm4v', 'webm', 'ogv', 'wmv', 'flv'
Default: Il primo file video allegato al post
poster
(string) (opzionale) Definisce un'immagine da mostrare come segnaposto prima che parta la riproduzione del video.
Default: Nessuno
loop
(string) (opzionale) Permette di riprodurre il file in modo continuativo. Il valore predefinito è "off".
  • "off" - ("default") il file non viene riprodotto in modo continuativo
  • "on" - il file viene riprodotto in modo continuativo
Default: "off"
autoplay
(string) (opzionale) Indica di riprodurre automaticamente il file appena esso è pronto. Il valore predefinito è "off".
  • "off" - ("default") il file non viene riprodotto automaticamente appena è pronto
  • "on" - il file viene riprodotto automaticamente appena è pronto
Default: "off"
preload
(string) (opzionale) Indica se e come caricare il file quando la pagina viene caricata. Il valore predefinito è "metadata".
  • "metadata" - ("default") vengono caricati solo i metadati quando la pagina viene caricata
  • "none" - il file non viene caricato quando la pagina viene caricata
  • "auto" - il file viene caricato interamente quando la pagina viene caricata
Default: "metadata"
height
(integer) (richiesto) Definisce l'altezza del video. Il valore è rilevato automaticamente al caricamento del file.
Default: [Altezza del file video]
width
(integer) (richiesto) Definisce la larghezza del video. Il valore è rilevato automaticamente al caricamento del file.
Default: [Larghezza del file video]

File sorgente[edit]

Lo shortcode Video si trova in wp-includes/media.php.
