Pour utiliser la fonction WordPress multisite, vous devez au préalable créer un sous-domaine générique.

  1. Connectez-vous à Plesk.
  2. Cliquez sur Ajouter un sous-domaine.
  3. Tapez le caractère * dans le champ "Nom du sous-domaine", puis cliquez sur OK.

C'est fait! Désormais, lorsqu'un utilisateur accède à un sous-domaine que vous n'avez pas créé explicitement dans Plesk, il est renvoyé vers le domaine parent. Procédez comme suit pour parachever la configuration de votre réseau WordPress.
