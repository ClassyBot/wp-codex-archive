Graças ao suporte WordPress ao XML-RPC, você agora publica postagens para seu blog WordPress usando qualquer um dos Clientes de Blogs atuais.

O WordPress usa uma interface XML-RPC! Nós atualmente suportamos a API do Blogger, API de metaWeblog e a API do Movable Type.

Nós também implementamos várias extensões a API do Movable Type, em uma API que chamamos de WordPress API(em inglês). Se possível, seu cliente deve usar as variantes iniciadas com o prefizo wp.

As APIs metaWeblog e MovableType são suportadas com a excessões seguintes:

  • O rascunho mt.supportedTextFilters é uma função em teste que retorna uma string vazia.

Entradas extendidas na API do MovableType API são automaticamente convertidas de/para a tag <!--more-->.

Crie seu Próprio Cliente de Blog[edit]

You can always develop your own. Many scripts in Perl, Python, PHP, Applescript and Java have already been written and are ripe for the picking (and tweaking). If you are planning on developing a Mac OS X Dashboard Widget, start with Apple's very detailed documentation.

Your XML-RPC server/path are as described here:

  • If your WordPress root is http://example.com/wordpress/, then you have:
    • Server: http://example.com/ (some tools will just want the 'example.com' hostname part)
    • Path: /wordpress/xmlrpc.php
    • complete URL (just in case): http://example.com/wordpress/xmlrpc.php
