Action[edit]

Dans le code de WordPress, une action ou une fonction d'action est une fonction qui peut être associée (« accrochée ») avec un événement (appelé Crochet) à l'aide d'un identifiant, puis exécutée en invoquant cet événement à un point particulier dans l'exécution de ce code. Par exemple, une extension ou un thème peut accrocher une fonction au crochet d'action wp_footer afin d'ajouter des informations au bas de chaque page web qui génère WordPress.

Dans le Codex WordPress et le code source, le terme est parfois utilisé de façon ambiguë, voir la note Confusion terminologique.

  • Voir aussi: Filtre, Crochet
  • Articles connexes : Actions, L'action de référence, add_action()

AJAX[edit]

AJAX est une technique que les pages web utilisent pour que le serveur effectue certains traitements sans rechargement de la page Web. Par exemple, lorsque vous approuvez un commentaire dans un blog WordPress, WordPress utilise AJAX pour modifier le statut du commentaire, et vous voyez le changement sans avoir à recharger l'écran des commentaires.

Apache[edit]

Apache est l'abréviation de Apache HTTP Server Project, un serveur Web robuste, de qualité commerciale, avec de nombreuses fonctionnalités, et librement disponible en open source produit par la Apache Software Foundation. C'est le serveur web le plus utilisé sur Internet, et il est disponible sur de nombreuses plate-formes, dont Windows Unix/Linux, et Mac OS X. Apache est une excellente base pour la publication de sites WordPress.

ASCII[edit]

ASCII (prononcé "askii") est un standard, mais limitée de jeu de caractère contenant uniquement les lettres anglaises, les nombres, quelques symboles communs, et les caractères de ponctuation communs anglais. Le contenu WordPress n'est pas limité au jeu ASCII, mais inclue les caractères Unicode

ASCII est l'abréviation pour American Standard Code for Information Interchange.

Barre d'Administration[edit]

La barre d'administration est une zone de l'écran, juste au-dessus de votre site qui répertorie les liens utiles de l'écran d’Administration comme ajouter un nouveau message ou modifier votre profil. Le concept de la barre d'administration a été ajouté à WordPress depuis la Version 3.1 et a été remplacé par la Barre d'outils dans WordPress Version 3.3. Chaque utilisateur peut utiliser Administration> Utilisateurs> Votre profil pour activer (ou désactiver) la barre d'administrateur lors de l'affichage du site ou du tableau de bord.

Blog[edit]

Un Blog ou weblog, est un journal en ligne, agenda, ou série d'articles publiés par une personne ou un groupe de personnes.

Les blogs sont généralement utilisés par des individus ou des groupes, mais sont aussi parfois utilisés par des entreprises ou des organisations. Dans l'univers des entreprises, les seuls adeptes du format blog jusqu'ici ont tendance à être des entreprises de conception, les sociétés de médias web et autres entreprises "Bleeding Edge" de haute technologie.

Les blogs contiennent souvent du contenu public ainsi que privé. Selon les fonctionnalités du logiciel de CMS qui est utilisé, certains auteurs peuvent restreindre l'accès — grâce à l'utilisation des comptes et mots de passe — pour que le contenu trop personnel pour être publié publiquement.

Commentaires (Comments)[edit]

Les Commentaires sont une fonctionnalité des blogs qui permettent aux lecteurs de réagir à desArticles. Typiquement, les lecteurs peuvent simplement exprimer leurs propres pensées concernant le contenu de l'article, mais peuvent également fournir des liens vers d'autres ressources, générer des discussion, ou simplement féliciter l'auteur pour un article bien écrit.

Vous pouvez contrôler et réguler les commentaires avec des filtres sur la langue et le contenu. Les commentaires peuvent être mis en attente pour approbation avant qu'ils soient visibles sur le site. Cette fonction est utile dans le traitement des spams de commentaires.

Entité de caractères (Character Entity)[edit]

L'entité de caractères est une méthode utilisée pour afficher les caractères spéciaux normalement réservés poue une utilisation dans le code HTML. Par exemple, le signe inférieur à (<) et supérieur à (>) sont utilisés poue construire les balises HTML tag structure, ces deux symboles sont donc réservés pour cette usage. Mais, si vous avez besoin d'afficher ces symboles sur votre site, vous pouvez utiliser les entités de caractère, comme dans l'exemple suivant :

use &lt;  pour le symbole inférieur à (<)
use &gt;  pour le symbole supérieur à (>)
