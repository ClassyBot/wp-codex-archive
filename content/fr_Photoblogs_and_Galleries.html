Une image vaut parfois plus que plusieurs phrases.

Les images peuvent être affichées de plusieurs façons dans WordPress, du simple affichage jusqu'à l'utilisation de galeries complexes, voire de blog photos sans texte. Voyons de plus près quelques-unes des différentes options pour les scripts de blogs photos et de galeries.

Blogs photos[edit]

Les blogs photos sont différents des blogs normaux. Les blogs normaux n'utilisent pratiquement que des phrases avec de temps en temps une image pour les illustrer, alors que les blogs photos misent tout sur les images et non sur les mots.

Le moyen le plus simple pour se lancer dans la création d'un blog photo avec WordPress est d'installer le plugin YAPB (yet another photo blog - encore un autre blog photo) de Johannes Jarolim ou bien le plugin PhotoQ Photoblog.

YAPB ajoute toutes les fonctionnalités standards d'un blog photo directement dans WordPress avec un minimum de configuration. Cela inclus le redimensionnement automatique des images, ainsi que d'autres outils. Le site de Johannes a un lien vers les modèles incorporant déjà le plugin YAPB (anglais), sinon vous pouvez créer le vôtre (anglais).

PhotoQ adopte une approche un peu différente. Le plugin met à disposition une file d'attente que vous pouvez remplir avec vos photos. PhotoQ est optimisé pour le traitement de photos multiples et offre entre autres le téléchargement par lots, le redimensionnement automatique des photos, du support exif, la création de filigranes et la publication automatique via des cronjobs.
