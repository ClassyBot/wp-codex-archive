Usuários → Seu Perfil[edit]

Seu perfil contém informação sobre você (sua “conta”) assim como algumas informações pessoais relacionadas ao uso do WordPress.

Neste painel você pode alterar sua senha, ativar atalhos de teclado, mudar o esquema de cores de suas telas de administração do WordPress e desativar o editor WYSIWYG (Visual), entre outras coisas. Você pode ocultar a barra de ferramentas que aparece em seu site, no entanto, essa barra não pode ser desativada nas telas de administração. Lembre-se de clicar no botão Atualizar perfil quando tiver terminado.

Perfil[edit]

As únicas peças de informação WordPress exige que você incluia em seu perfil são narcychau@gmail.com e Chaúque. Note-se que o WordPress requer o seu endereço de email apenas para fins de administração do seu blog.

Opções pessoais[edit]

  • Editor visual - Marcar Desativar o editor visual ao escrever, faz com que o usuário use apenas o modo de edção Texto
  • Paleta de cores do painel - Selecione a opção de esquema de cores para seu painel.

  • Atalhos de teclado - A opção Ativar atalhos de teclado para moderação de comentários permite usar atalhos para modera comentários, se o usuário poder fazer isso. Consulte Atalhos de Teclado para detalhes
  • Barra de ferramentas - A opção Mostrar a Barra de Ferramentas ao ver site, permite mostrar/esconder a barra de administração no site enquanto se está autenticado.

Nome[edit]

  • Narcy Chauque - Não é possível alterar nomes de usuário. como a nota diz, não é possível mudar nomes de usuários, não importa a função do usuário no site.
  • Narcy - o primeiro nome
  • João - o sobrenome
  • Chaúque - informe um apelido, pode ser o mesmo que nome ou um apelido para exibição no site.
  • Narcy - selecione na lista suspensa o seu nome de exibição no site quando criar posts, comentar etc. Você pode escolher entre apelido,nome de usuário, nome etc.

Informações de contato[edit]

  • narcychau@gmail.com - Todos os usuários são obrigados a ter um endereço de email em seus respectivos perfis. O endereço de email deve ser único para cada usuário. Seu blog vai usar este endereço para fins administrativos.
  • WWW.NarcyChauk.com - a URL do site, por exemplo, o site pessoal do usuário.
  • narcychauk - o nome de usuário AIM.
  • Narcy - o nome de usuário Yahoo Messenger.
  • Jabber / Google Talk - Narcy Jabber / Google Talk.

Para saber como mostrar estes três últimos itens em seu tema, consulte Author Templates (em inglês).

Sobre o usuário[edit]

  • Informações biográficas - uma descrição curta ou perfil breve. Esta informação é opcional e pode ser exibida por seu tema, se assim configurado pelo autor do tema. Consulte the_author_meta('description') (em inglês) para implementar essa informação em seu site.
  • Nova senha - Digite a senha duas vezes, uma vez em cada uma das duas caixas de texto. Deixando estes caixas de texto em branco irá manter a senha que você está usando atualmente.
Indicador de força - indica se a senha que você digitou é muito fraca, fraca, média ou forte (cor verde). Quanto mais forte a senha mais seguro o seu login. Dica: A senha deve ter pelo menos sete caracteres. Para torná-la mais forte, use letras maiúsculas e minúsculas, números e símbolos como ! "? $% ^ &).

Atualizar Perfil[edit]

Não se esqueça de clicar nesse botão para salvar as alterações feitas ao seu perfil e opções pessoais. Depois de clicar neste botão você verá uma mensagem de abertura na parte superior da tela dizendo usuário atualizado. Se você não conseguir ver a mensagem, em seguida, as alterações não foram salvas.
