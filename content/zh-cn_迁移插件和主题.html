介绍[edit]

当 WordPress 发布新的主版本时，如 WordPress 2.1 或 2.2，你可能会发现外观主题和插件不再可用。它们可能已经完全损坏，因为 WordPress 中支持它们运作的机制已经改变。但是你还是可以通过更新一些设定或做些小修改就可以让它们恢复运作。

此文仅为关于插件和外观主题更新升级到下一个 Wordpress 版本的 Codex 文章的一个列表。

  • 从 1.2 升级到 1.5 - 转化很老旧的外观主题，使其适用于当前的外观主题系统。
  • 迁移插件和主题到 2.1 - 涉及 WordPress 2.0 和 WordPress 2.1 之间的改变
  • 迁移插件和主题到 2.2 - 涉及 WordPress 2.1 和 WordPress 2.2 之间的改变
  • 迁移插件和主题到 2.3 - 涉及 WordPress 2.2 和 WordPress 2.3 之间的改变
  • 迁移插件和主题到 2.5 - 涉及 WordPress 2.3 和 WordPress 2.5 之间的改变
  • 迁移插件和主题到 2.6 - 涉及 WordPress 2.5 和 WordPress 2.6 之间的改变
  • 迁移插件和主题到 2.7 - 涉及 WordPress 2.6 和 WordPress 2.7 之间的改变
  • 迁移多个博客到 WordPress 3.0 多站点 - 涉及 WordPress 博客和 WordPress 3.0 多站点之间的改变
  • 迁移插件和主题到 3.1 - 涉及 WordPress 3.0 和 WordPress 3.1 之间的改变
