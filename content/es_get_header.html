Descripción[edit]

Incluye el archivo de la plantilla header.php del directorio de tu tema actual. Si se especifica un nombre entonces se incluirá una cabecera especializada header-{nombre}.php.

Si el tema no contiene ningún archivo header.php entonces la cabecera por defecto será wp-includes/theme-compat/header.php.

Uso[edit]

<?php get_header( $nombre ); ?>

Parámetros[edit]

$nombre
(string) (optional) Llama a header-nombre.php.
Default: None

Ejemplos[edit]

Página 404 simple[edit]

El siguiente código es un simple ejemplo "HTTP 404: No Encontrada" (la cuál deberías incluir en tu Tema como 404.php). 

<?php get_header(); ?>
<h2>Error 404 - No Encontrada</h2>
<?php get_sidebar(); ?>
<?php get_footer(); ?>

Cabeceras múltiples[edit]

Diferente cabecera para diferentes páginas. 

<?php
if ( is_home() ) :
	get_header( 'home' );
elseif ( is_404() ) :
	get_header( '404' );
else :
	get_header();
endif;
?>

Los nombres de fichero las cabeceras de 'home' y '404' deberían ser header-home.php y header-404.php respectivamente.

Notas[edit]

  • Usa: locate_template()
  • Usa: do_action() Llama a la acción 'get_header'.

Log de cambios[edit]

Fichero fuente[edit]

get_header() se ubica en wp-includes/general-template.php.
