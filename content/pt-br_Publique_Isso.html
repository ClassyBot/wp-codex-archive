A função "Publique Isso" permite publicar rapidamente através de um bookmarklet. Você pode criar uma postagem citando textos, imagens e vídeos de qualquer página web.

O Publique Isso fica no Subpainel Ferramentas Disponíveis


Uso[edit]

  1. Crie este bookmarklet em Ferramentas > Ferramentas
  2. Arraste o link para a barra de Favoritos do seu navegador
  3. Durante a navegação, selecione algum texto (opcional) e clique no bookmarklet nos Favoritos do seu navegador.
  4. Edite o conteúdo da janela pop-up.
  5. Salve ou publique a postagem.

Nota Técnica[edit]

Usando este bookmarklet para publicar postagens não passa a variável $_POST['post_type'] com o valor post.

O bookmark chama http://exemplo.com/wp-admin/press-this.php?u=&t=&s=&i=

  • u = url da página
  • t = título
  • s = texto selecionado da página
  • i = url de uma arquivo de imagem

Edite os valores do bookmark na barra de endereço do browser se achar necessários.
