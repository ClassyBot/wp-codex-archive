Every localization team can have a WordPress forum in their language, hosted on WordPress.org.

How to get local forums?[edit]

  1. Choose forums URL slug. Usually your locale does a good job. For example, the URL of the Bulgarian forums is http://bg.forums.wordpress.org/
  2. Choose forums title, something like Bulgarian Forums, but in your language.
  3. Choose the name and description for your first forum.
  4. Translate the POT file of the bbPress theme used in the forums and commit it to http://i18n.svn.wordpress.org/your-locale/rosetta/wporg-bbpress-theme.po
  5. Translate the Support Forum bbPress plugin's POT file and commit it to http://i18n.svn.wordpress.org/your-locale/rosetta/support-forum.po
  6. Translate trunk of bbPress. Of course, there is a POT file for it too. If you still don't have commit access for the bbPress i18n repository, post a request on the Polyglots' blog.
  7. Send the collected information to the Polyglots' blog.
