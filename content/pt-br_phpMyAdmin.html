O que é o phpMyAdmin?[edit]

Uma ferramenta dos administradores de sortes, phpMyAdmin é um script PHP feito para dar aos usuários a capacidade de interagir com suas bases de dados MySQL. WordPress armazena todas as suas informações no banco de dados MySQL e interage com o banco de dados para gerar informações de seu site WordPress. Uma visão "crua" dos dados, tabelas e campos armazenados no banco de dados MySQL está acessível através do phpMyAdmin.

Por que é bom?[edit]

O programa phpMyAdmin é útil para executar operações de manutenção em tabelas, fazer backup de informações e edição diretamente nos dados que o WordPress cria. Ocasionalmente, no Fórum de Suporte, alguém vai perguntar sobre uma consulta SQL de algum benefício ou outros que podem ser executados usando o phpMyAdmin. Embora muitas das mesmas tarefas podem ser executadas na linha de comando do MySQL, isso não é uma opção para muitas pessoas.

Onde eu posso conseguir isso?[edit]

Muitas vezes, os painéis de controle da hospedagem, como o cPanel e Plesk, têm phpMyAdmin pré-instalado, então não há nada especial que você tem que fazer para usá-lo. É geralmente associada a partir da página do banco de dados. Pergunte ao seu serviço de hospedagem se este estiver disponível.

Você pode baixar o phpMyAdmin sozinho e e instalá-lo a partir da Página de Pojeto do phpMyAdmin principal. O programa não é o mais fácil de instalar do mundo, mas não é incrivelmente difícil.

Alternativamente, você também pode fazer download do pluginWP-phpMyAdmin . Este plugin instala o phpMyAdmin e o torna disponível a partir do seu console de administração do WordPress.

Atenção[edit]

Muito poder vem com muita responsabilidade! O phpMyAdmin permite que você interaja com o banco de dados diretamente: ele também permite que você modifique o banco de dados diretamente. Não há "desfazer" ou "restaurar da lixeira" em seu banco de dados! Sempre tenha cuidado ao trabalhar diretamente no banco de dados.
