Em 04 de Setembro de 2014, foi lançado o WordPress Version 4.0, que recebeu o nome do músico de Jazz Benny Goodman. Para mais informações sobre esta versão de aprimoramento e de correção de bugs, leia o WordPress Blog, e o log de lançamento para 4.0.

Para a Versão 4.0, a versão do banco de dados (db_version in wp_options) foi alterado para 29630, e o número da revisão do Trac é 29710.

Destaques[edit]

O que há de novo[edit]

Geral[edit]

Posts[edit]

  • Adicionado botão de preview para áudio/vídeo no editor visual.
  • Rolagem de página acontece somente no campo de conteúdo da mesma.
  • Menu Editar Post/Página fica fixo quando há rolagem no campo de conteúdo.

Mídia[edit]

Comentários[edit]

Aparência[edit]

Usuários[edit]

Acessibilidade[edit]

  • Melhoria da acessibilidade de teclado no painel Adicionar mídia
  • Melhor suporte de leitura de tela para seções Customizer
  • Cria links acessíveis via teclado na guia de ajuda
  • Melhorias para leitores de tela quando o gerenciar widgets no Customizer

Processo de Instalação[edit]

  • Adicionado lista de idiomas na tela de instalação.

Multisite[edit]

Por baixo do capô[edit]

Geral[edit]

  • Melhoria de performance e eficácia do wp_texturize()
  • Permitido o uso de classes personalizadas em custom post type aninhados em itens de menu de nível superior
  • Adicionado parâmetro src para o shortcode de incorporação
  • Correções para lidar com códigos de acesso aninhadas
  • Make default 'template' argument of the_taxonomies() and get_the_taxonomies() translatable
  • Remover uma condição redundante para comentar feeds de WP_Query::get_posts()
  • The Customizer now properly honors theme support defaults for background images
  • Installed Themes search now shows a proper "no results" message
  • Improved, more consistent styling for HTML5 input elements

JavaScript[edit]

  • Labels experimentais do Heartbeat API foram removidos

Bugs Consertados[edit]

Multisite[edit]

Classes[edit]

  • Adicionados métodos mágicos em diversas classes para tornar varias própriedades acessíveis e prover compatibilidade com versões anteriores

O que há de novo[edit]

  • WP_Customize_Panel
  • WP_Session_Tokens
  • WP_User_Meta_Session_Tokens

Funções[edit]

Novidades[edit]

  • WP_Query - Melhor flexibilidade quando contém vários `orderby`

Descontinuado[edit]

  • get_all_category_ids() - foi substituído por get_terms()
  • like_escape() - foi substituído por wpdb::esc_like()
  • url_is_accessable_via_ssl()

Actions & Filtros[edit]

  • O formato de data AAAA-MM-DD padrão ISO foi adicionado ao filtro date_formats.
  • O parâmetro $post foi adicionado ao preview_post_link.
  • wp_handle_upload_prefilter foi renomeado para {$action}_prefilter.

Novas Actions[edit]

  • Adicionado customize_render_panel
  • Adicionado customize_render_panel_{$this->id}
  • Adicionado delete_user_form
  • Adicionado pre_get_users
  • Adicionado rss_tag_pre

Novos Filtros[edit]

  • Adicionado oembed_ttl

  • Adicionado wp_editor_settings

  • Adicionado wp_list_comments_args
  • Adicionado wp_spaces_regexp
  • Adicionado wp_title_parts
  • Adicionado wxr_export_skip_commentmeta
