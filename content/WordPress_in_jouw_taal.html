Talen: English • Nederlands • 日本語 • 한국어 • ไทย • (Voeg jouw taal toe)

Hoewel de meeste teksten over WordPress in het Amerikaans-Engels verschijnen, heeft de software een ingebouwde mogelijkheid om ook andere talen te ondersteunen. Inmiddels zijn niet alleen deze teksten, maar ook WordPress onderdelen zoals thema's al in diverse talen vertaald. Met de hulp van sommige plugins kun je zelfs je blog in verschillende talen om laten zetten.

Introductie[edit]

Wanneer je WordPress in jouw taal (of in een andere taal) wilt gebruiken, kun je het volgende doen:

  • Zoek op deze pagina naar teksten die al in jouw taal zijn geschreven.
  • Lees hoe je WordPress in jouw taal kunt installeren, zodat WordPress in jouw taal zal verschijnen.
  • Wanneer jouw taal niet in het lijstje voorkomt, dan kun je de databank voor WordPress taaldocumenten raadplegen: WordPress Language File Repository (zie dit artikel voor meer informatie).
  • Er is een lijst van vertaalteams die je kunt raadplegen om te zien welke vertalingen er worden gemaakt. Kijk op vertaalteams en nieuwe vertaalteams.
  • Wanneer jouw taal niet op de lijst staat en niet in de databank te vinden is, of wanneer de vertaling aan verbetering toe is, kun je hulp bieden! Zie Vertaling WordPress en I18n voor WordPress Ontwikkelaars voor meer informatie.
  • Er bestaat een scala aan WordPress plugins WordPress Plugins waaruit je kunt kiezen om je blogberichten te laten vertalen. Kijk voor meer informatie op de Plugin inhoudsopgave van de officiële WordPress Plugin Directory op op de WordPress Plugin Database: Official WordPress Plugin Directory of WordPress Plugin Database.
  • Wanneer je via jouw bedieningspaneel de WordPress vertalingsdocumenten wil gebruiken, dan zijn deze te vinden op http://i18n.svn.wordpress.org/. Hier kun je kiezen voor een wp-native-dashboard: wp-native-dashboard.

Lijst van talen[edit]

De lijst van talen is tijdelijk weggehaald, omdat hij niet meer klopte. Zodra hij is bijgewerkt en vertaald komt hij weer terug.
