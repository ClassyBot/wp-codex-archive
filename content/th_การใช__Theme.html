Before the advent of Themes, WordPress generated content using a single file, index.php, and files to support comment display and submission. A single style sheet controlled the presentation. All other pages, including the category and archive pages, were generated by passing parameters to the index.php page.

The new Theme system provides two convenient features.

Physically Separate Components 
The new WordPress modular template files system provides a method to define separate physical PHP files for the different components of your WordPress site. This allows creation of unique designs and functionality for many special pages, such as category archives, monthly archives, and the individual entry pages.
Quickly Change Layout and Design 
It allows users with appropriate permissions to quickly change the layout of the entire site by uploading a new theme and essentially flipping a switch in the admin panel.

The old method of generating pages will still work. If you are upgrading from v1.2 or v1.2.1 or v1.2.2, you can continue using your existing design. Keeping your older design will not prevent you from adding additional themes and easily switching between designs.

Theme คืออะไร[edit]

โดยปกติแล้ว ระบบเวิร์ดเพรสธีมคือวิธีเปลี่ยน "หน้าตา" บล็อกของคุณ แต่มันมากกว่าการเปลี่ยน "หน้าตา" การเปลี่ยนหน้าตาเวบไซต์ของคุณหมายถึงหน้าเวบนั้นมีดีไซน์ที่เปลี่ยนไป แต่เวิร์ดเพรสธีมสามารถควบคุมได้มากกว่าหน้าตา "และการนำเสนอ" ของสิ่งต่าง ๆ ที่อยู่บนเวบไซต์ของคุณ

เวิร์ดเพรสธีมนั้นเป็นการประกอบกันของไฟล์หลาย ๆ ไฟล์ที่จะทำงานร่วมกันเพื่อสร้างหน้าตาเวบ ไฟล์เหล่านี้เราเรียกว่า ไฟล์ template ธีมนั้นถูกแก้ไขเพื่อให้การแสดงผลหน้าตาเวบเปลี่ยนไป โดยไม่ได้แก้ไขโค๊ดหลักของระบบ ธีมอาจจะรวมไฟล์ template ที่แก้ไขแล้วหลาย ๆ ไฟล์, ไฟล์รูปภาพ(*.jpg, *.gif), ไฟล์สไตล์ชีท (*.css), Pages ที่ดัดแปลง, และไฟล์ (*.php) ที่จำเป็นอื่น ๆ อีก สำหรับความรู้เบื้องต้นเรื่อง ไฟล์ template โปรดดู Stepping Into Templates

ธีมเป็นเหมือนเครื่องมือที่ควบคุมหน้าตาเวบไซต์คุณทั้งหมด สมมุติว่าคุณเขียนเรื่องเกี่ยวกับการ์ตูนและเพลง โดยผ่านทางการใช้งาน WordPress Loop และ template files คุณสามารถดัดแปลงหน้าตาเรื่องในหมวดหมู่การ์ตูนของคุณให้ดูแตกต่างจากเรื่องในหมวดหมู่เพลงได้ ด้วยการควบคุมที่มีประสิทธิภาพ คุณสามารถสร้างหน้าและหมวดหมู่ที่แตกต่างกันได้บนเวบไซต์ของคุณ สิ่งที่เป็นข้อจำกัดอย่างเดียวก็คือ "ความคิดของคุณ" สำหรับรายละเอียดเพิ่มเิติมเพื่อใช้ธีมที่แตกต่างกันสำหรับหมวดหมู่หรือเรื่องที่ต่างกัน โปรดดู The Loop in Action และ Category Templates

ดาวน์โหลด Themes ใหม่ ๆ[edit]

หน้าดาวน์โหลดเวิร์ดเพรสธีม เป็นหน้าของเวิร์ดเพรสอย่างเป็นทางการสำหรับการดาวน์โหลดเวิร์ดเพรสธีม ธีมทุกธีมที่อยู่ในเวบไซต์นี้ถูกตรวจสอบเป็นอย่างดีแล้ว และสามารถใช้งานได้ฟรี คุณสามารถใช้คำสั่งในหน้านั้นเพื่อค้นหาธีมที่คุณต้องการ โดยเลือกประเภทและรูปแบบได้ และยังแสดงการแสดงผลของแต่ละธีมได้ด้วย

การใช้ Themes[edit]

เวิร์ดเพรสนั้นจะมีธีมสองธีมให้คุณเลือกใช้ติดมากับการติดตั้ง คุณสามารถเปลี่ยนหน้าตาไปมาระหว่างสองธีมได้โดยใช้หน้าควบคุมของ admin และธีมใหม่ที่คุณอัพโหลดขึ้นไปจะแสดงใน Administration Panels > Design > Themes เป็นธีมเพิ่มเติมให้เลือก

เพิ่ม Themes ใหม่[edit]

มีหลายธีมที่คุณสามารถดาวน์โหลด เพื่อที่จะทำงานพร้อมกับการติดตั้ง WordPress ของคุณ

หากธีมที่คุณกำลังติดตั้งมีคำแนะนำในการติดตั้ง กรุณาอ่านและปฏิบัติตามคำแนะนำดังกล่าวเพื่อความสำเร็จในการติดตั้งธีม โดยปกติแล้ว เราจะแนะนำให้ผู้พัฒนาธีม ใส่คำแนะนำในการติดตั้งธีมของพวกเขาลงไปด้วย เนื่องจากธีมบางอัน มีตัวเลือกการทำงานพิเศษที่อาจต้องติดตั้งเพิ่มเติมกว่าการติดตั้งขั้นพื้นฐาน ถ้าหากธีมของคุณไม่ทำงาน หลังจากที่ปฏิบัติตามคำแนะนำแล้ว โปรดติดต่อผู้พัฒนาธีมเพื่อขอความช่วยเหลือ


ในการเพิ่มธีมใหม่ของคุณลงในการติดตั้ง WordPress กรุณาทำตามขั้นตอนพื้นฐานเหล่านี้:

  1. ดาวน์โหลดธีมที่อยู่ในรูปแบบการบีบอัด หลังจากนั้นจึงคลายออก. คุณอาจต้องคงรูปแบบของโครงสร้างไดเรกทอรีที่ให้มาก่อน หลังจากนั้นจึงทำตามคำแนะนำของผู้พัฒนาธีม
  2. ใช้ FTP ของคุณเพื่อเข้าถึงโฮสต์เว็บเซิร์ฟเวอร์ หลังจากนั้นสร้างไดเรกทอรีที่มีธีมของคุณในไดเรคทอรี wp-content/themes ตัวอย่างเช่นธีมชื่อ test ควรอยู่ใน wp-content/themes/test. ธีมของคุณ อาจแสดงไดเรกทอรีนี้เป็นส่วนหนึ่งของข้อมูลที่อยู่ในรูปแบบการบีบอัด
  3. อัปโหลดไฟล์ธีมไปยังที่ไดเรกทอรีของเซิร์ฟเวอร์โฮสต์
  4. ทำตามคำแนะนำต่อไปนี้ สำหรับการเลือกใช้งานธีมใหม่ของคุณ

การเพิ่มธีมใหม่ใน cPanel[edit]

หากคุณมีโฮสต์ทีเป็น cPanel และชุดรูปแบบไฟล์ที่อยู่ในรูปแบบ .zip หรือ .gz กรุณาปฏิบัติตามคำแนะนำเหล่านี้. หมายเหตุ: การทำงานนี้ถือว่าชุดรูปแบบที่คุณดาวน์โหลดในรูปแบบการบีบอัด (zip) ไฟล์และแฟ้มต่างๆในไฟล์ zip อยู่ใน 'ชื่อ' โฟลเดอร์.

  1. ดาวน์โหลดธีมที่เป็นไฟล์ zip ลงที่เครื่องของคุณ
  2. ในตัวจัดการไฟล์ของ cPanel, นำทางคุณไปยังโฟลเดอร์ที่เก็บธีม. หากคุณมีการติดตั้ง WordPress ที่โฟลเดอร์ชื่อ "wordpress" แล้วคุณจะต้องไปที่ "public_html/wordpress/wp-content/themes" WordPress และหากมีการติดตั้งของคุณในเว็บโฟลเดอร์หลัก คุณจะต้องไปที่ "public_html/ wp-content/themes"
  3. เมื่อคุณได้ไปที่ธีมโฟลเดอร์แล้ว คลิกที่อัปโหลดไฟล์ในตัวจัดการไฟล์ cPanel และเลือกอัปโหลดไฟล์ซิปที่คุณบันทึกไว้ในขั้นตอนที่ 1
  4. เมื่อมีการอัปโหลดไฟล์ซิปแล้ว ให้คลิกที่ชื่อไฟล์ซิปใน cPanel แล้วในแผงควบคุมด้านขวาให้คลิกที่"ขยายไฟล์" (Extract) หลังจากนั้นไฟล์ซิปที่เลือกก็จะถูกคลายออก
  5. ทำตามคำแนะนำต่อไปนี้ สำหรับการเลือกใช้งานธีมใหม่ของคุณ

เลือกธีมเพื่อใช้งาน[edit]

  1. เข้าสู่เวิร์ดเพรสหน้าจัดการหลัก (WordPress Administration Panels)
  2. เลือกที่รูปแบบ Design แล้วธีม Theme
  3. คลิกที่ชื่อธีม (หรือภาพธีม) ที่คุณต้องการใช้งาน
  4. ตัวอย่างของธีมจะปรากฏขึ้น คลิกที่ลิงก์ด้านบนขวา "ใช้งาน(Activate)" เพื่อใช้งาน

การสร้างธีม[edit]

หากคุณสนใจในการสร้างธีมของคุณเองเพื่อแจกจ่าย หรือต้องการเรียนรู้เพิ่มเติมเกี่ยวกับสถาปัตยกรรมของธีม โปรดอ่านเอกสารประกอบเรื่องรูปแบบการพัฒนาธีม และการออกแบบธีมสำหรับใช้งานทั่วไป

Theme Files[edit]

The following are the files typically included within a Theme.

  • 404 Template = 404.php
  • Archive Template = archive.php
  • Archive Index Page = archives.php
  • Comments Template = comments.php
  • Footer Template = footer.php
  • Header Template = header.php
  • Links = links.php
  • Main Template = index.php
  • Page Template = page.php
  • Popup Comments Template = comments-popup.php
  • Post Template = single.php
  • Search Form = searchform.php
  • Search Template = search.php
  • Sidebar Template = sidebar.php
  • Stylesheet = style.css
