许可和平台[edit]

  • 许可 : WordPress 遵循 GPL 协议的。
  • 平台 : 必须安装 PHP (v4.3 或更高版本) 和 MySQL(v4 或更高版本)。

管理博客[edit]

本地安装 
WordPress 被设计成既可以按照在你自己的Web服务器上，又或者建立分享主机帐号，帐号可以让你完全掌控你自己的网络博客。和第三方的主机服务不同，你完全可以访问并修改与你的博客相关的任何东西，只要你需要。这里也允许你在自己的台式机、家用电脑甚至是局域网上安装 WordPress 。
便携核心 
你可以选择wordpress 相关文件的树形结构，该结构将构造你公开显示的网络博客的后台，可以把文件放在与网络日志相同的目录中，当然也可以不同。举例来说，如果你想把你的博客放在 http://example.com (public_html - 你的服务器的或者是帐号的根目录) 同时你又要把相关的文件放在 http://example.com/wordpress (public_html/wordpress) 目录下，这样是可行的！
UTC 友好
WordPress 允许你用协调通用时间定义(UTC)的偏移值你的时间，所有时间有关的元素都会存入数据库，存入形式为GMT值，GMT是一个通用标准。此外，它将为你的博客显示正确的时间，即使你的主机服务器在一个不同的时区。
用户管理 
WordPress 使用用户等级来控制用户的访问权限，因此你可以通过改变用户级别来限制个别用户的权力，比如创建或者修改你的博客的内容的权力。
用户资料 
每个在你博客上的用户能定义一个资料，其中包括他们的电子邮件地址，即时通信地址等。用户也可以控制他们的详细资料在博客上的显示方式。
简单的安装和升级 
WordPress 著名的五分钟安装已经是无人能敌的，它简单易用。升级你的网络日志到最新版本也同样简单，而且需要的时间更少！
动态网页生成 
在每次你升级你的博客时不会重新构建你的所有页面，或者部分网页。所有的网页都是通过数据库和模板生成的，而且只有当观看者浏览时才会产生相应的页面。这样就意味着升级操作或者其设计都是尽量快速，同时需要的服务器存储空间占用最小。
国际化和本地化 
你能创建一个你选择的本地化的博客，以你所选择的语言进行发布内容。gettext 方法的应用促进全面翻译和本地化 WordPress 的工作。

发布你的作品[edit]

供稿（Feeds） 
RSS 1.0 (aka RDF)， RSS 2.0 和ATOM 的规格在WordPress中被完全支持的，而且在你网志上的任何一个页面都有相联系的一个供稿供你的读者们进行订阅——有最新日志的供稿，页面分类以及评论的相关供稿，就像我们之前所说的，你可以得到任何供稿。你的读者需要跟进你的博客的不同模块的选择越多，你向世界传播你的文章就越容易。WordPress 也完全支持带附件的RSS 2.0，所以添加MP3（就像播客那样）到你的RSS供稿是极其简单的。
非代码形式固定链接 
你的博客上所有页面的URL可以做成符合一定的标准，非代码系统，同时所有链接都井井有条，合理的，并且对于人类和机器甚至搜索引擎来说都是易于理解的。清楚的URL对于搜索引擎优化来说是重要的而且能改善用户体验。
外部博客联系 
在这个相互联系日益紧密的世界里，WordPress 早就准备了 PingBack 和 TrackBack，两个很有用的方式来连接别人的博客，同时他们也可以连接到你的博客。

定制设计[edit]

模板驱动设计 
WordPress 使用模板来动态生成页面。你可以用模板编辑器和模板标签编辑模板从而控制内容的呈现方式。
模板和文件编辑器 
每次 WordPress 的安装都会带来一个文件编辑器，你可以使用它来编辑你的模板和其它 WordPress 相关文件，这些操作都在你的浏览器中进行，不必操心下载上传文件来编辑它们。
模板标签 
模板标签使得内容信息展示的设计更加容易。你无需编程一个PHP高手就能设计你的网志。
主题 
你可以为你的博客换上现成的主题或者风格。你也可以自己创造并分享你自己的主题。
插件 
插件扩充了你的博客的核心功能。有大量用户自己开发的插件供人们使用，事实上，它们可以被用来在你的博客上为你做任何事。

创作[edit]

密码保护 
如果你只想分享你的内容给一些人，而非所有人，怎么办？ 很简单，可以在这篇文章上加设密码。
文章别名 
如果你在网站上使用清晰的固定链接，你可以通过文章别名定义个链接到个别的文章上。
定时发布 
你可以在今天写完一篇文章后，让它定时在未来某天自动张贴出来。
分页文章 
如果你的文章太长，可以把它分成多个页面，这样你的读者就不必使用卷轴到世界的尽头。
文件/图片上传 
你能够上传图片或者文件，在文章中可以插入它们的链接或者显示。在你上传时，还可以创建图片的缩略图。
分类 
将你的文章分门别类...
表情图片 
WordPress 很聪明能够将字符笑脸，如“:)”转换成对应的图片。
保存草稿 
保存你未完成的文章，稍后做完善，完成后再发布。
预览文章 
在你按下“发布”按钮前，你可以观看文章的预览，从而确定是否万事俱备。其实，你可以随时预览，因为它是“实时的”。
桌面工具 
你不一定要使用浏览器来更新你的博客，你可以使用任意一款支持MetaWeblog 或 Blogger API 的桌面工具。
通过邮件写博客 
你能以邮件形式发送你的文章并发布在你的网志上。
快速发布 
WordPress 提供的“快速发布!”书签工具，在你的书签中添加此链接，从此可以有一个快捷键来摘取当前所浏览的网页内容，并以此内容新建文章。
侧栏 
如果你不喜欢书签工具，可以使用我们友好的浏览器侧栏，也是应用类似风格的。
格式化 
把 WordPress 想象成一个使你的文字更加流畅，你的页面更加吸引人的工具。WordPress 装载着文字格式化插件，帮助整理你的内容并且加入优美的版式美化你的文章。

存档和搜索[edit]

存档 
在你写日志一段时间后，如何组织你的日志显得非常重要，为此，WordPress 提供了一些现成的存档展现方式，囊括了以前所有的文章。你可以从年，月，周，日，类别和作者中进行选择，可以很方便地从主界面（或者任一单独页面）链接到存档页面，这也是通过模板标签生成指向这些存档的链接。由于WordPress 动态生成页面，所有这些存档页面不会占用你服务器上的任何空间。
搜索 
WordPress 有一个内置的搜索工具，它允许你博客的访问者搜索他们感兴趣的项目，搜索关键字插件装载在WordPress 上可以高亮显示他们的搜索词，这样就能够更容易地找到他们要搜索的内容。除此之外，这个插件同样可以为那些通过点击搜索引擎的搜索结果进入你的博客的人提供同样的功能，譬如Google。综上，对于 WordPress 来讲，搜索是快乐的。

讨论与评论[edit]

社区建设 
WP不是基督教青年会（YMCA），但是它也帮助建立以博客为中心的社区，通过评论、Trackback 和 Pingback 的应用，帮助你与读者保持联系，并且培养友谊。
HTML标签使用限制 
不是所有人都是魔鬼，不过对他们进行控制，在你的网络日志上限制其HTML标签的使用是合理的。被WordPress 默认允许的标签是经过明智选择后的结果，允许人们在评论中使用它们，这样就不用担心你的数据或者服务器的安全了。
评论审查 
为了满足一些人的控制欲，WordPress 提供了一组审查选项。你可以审查
  • 所有的评论，决定它们是否出现在日志上
  • 有特定文字的评论
  • 特定IP的评论
  • 包含超过一定数量链接的评论。
所有这些选项都能够将垃圾评论和破坏性评论拒之门外。
通知 
WordPress 可以不断地发送给你邮件，每当有一个新的评论或者有一个评论等等评审。

创建和管理博客名单[edit]

博客名单（Blogroll） 
博客名单是个你可以从此链接到你经常阅读的博客的地方——一种推广优秀博客的友好方式。WordPress 内建的 链接管理允许你毫不费力地添加和管理链接。
书签工具 
一个简洁的书签工具能够让你轻松地添加到你浏览器的书签或者收藏夹里。当你再次访问一个有趣博客或网站时，为此添加一个链接与点击加入书签或收藏按钮同样简单！
归类 
博客名单上的链接能够进行归类以及简单地组织。
导入 
如果你已经拥有一个链接列表的OPML（大纲处理标记语言）文件，可以把它导入你的WordPress。那些从其它博客工具而来的列表，可以从Blogrolling.com导入你的博客名单，无需使用第三方服务管理。
导出 
难道还需要说我们能够导出OPML文件吗？
展示 
和其他一样，你有一些简单的模板标签来展示你的链接——譬如按字母排列，按评分，按更新时间——由你来掌控。
