Descripción[edit]

Codifica < > & " ' (menor que, mayor que, ampersand, comillas dobles, comillas simples).

Uso[edit]

<?php esc_attr( $text ) ?>

Parámetros[edit]

$text
(string) (required) El texto que será codificado.
Default: None

Valores que devuelve[edit]

(string) 
La cadena de texto con las entidades HTML.
