描述 Description[edit]

该 条件标签(conditional tag) 检查是否是一个单页面(singular post)被呈现，即当以下函数返回真的时候的页面：is_single(), is_page() or is_attachment() 。 如果 $post_types 参数被指定，该函数还会判断是否 请求(query) 中包含了明确的文章(post)类型。

使用 Usage[edit]

<?php is_singular( $post_types ); ?>

Parameters[edit]

$post_types
(string/array) (optional) Post type or types to check in current query. 检查当页的文章类型。
Default: None

Return Values[edit]

(boolean) 
True on success, false on failure. 成功则返回True，失败则返回false。.

示例 Examples[edit]

在单型页显示不同的边栏 Different Sidebar Ads in Singular Page[edit]

<?php
if ( is_singular() ) {
  // show adv. #1
} else {
  // show adv. #2
}
?>

默认文章类型 Default Post Type[edit]

下式在探测到一个文章(post)类型的时候返回True。 

is_singular( 'post' );

自定义文章类型 Custom Post Types[edit]

当 is_single(), is_page() or is_attachment() 中任一返回True的时候，下式返回True 

is_singular();

当检测到一个 自定义文章类型(Custom Post Type) book 的时候，下式返回True 

is_singular( 'book' );

当检测到 自定义类型(Custom Post Type) newspaper or book 的时候，下式返回True 

is_singular( array( 'newspaper', 'book' ) );

使用 Notes[edit]

  • 版本3.0.0: 加入了 $post_types 参数。
