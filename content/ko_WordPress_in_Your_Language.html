워드프레스는 기본적으로 미국 영어로 표시되지만, 모든 언어에서 사용할 수 있는 기능을 가지고 있는 소프트웨어 입니다. 워드프레스 커뮤니티에서는 이미 여러 나라 언어로 워드프레스가 번역되고 있으며 테마, 번역 파일 및 사용가능한 많은 지원들을 다른 언어에서 사용할 수 있습니다. 그러한 플러그인 덕분에 워드프레스를 이용하여 다국어 블로그를 만들 수도 있습니다.

소개[edit]

다른 언어로 워드프레스를 사용하기 위해서는, 다음과 같이 따라하세요:

  • 이미 당신의 언어로 지원하고 있는지 확인하세요.
  • Read Installing WordPress in Your Language to learn how to install files that will transform WordPress so it displays in your language.
  • If your language is not listed below, you can also try the WordPress Language File Repository (see this article on the Repository for more information).
  • If your language is not available in the Repository or on this page, or if the available translation needs to be improved, you can help! See Translating WordPress and I18n for WordPress Developers for more information.
  • There is also a list of localization teams and localization teams currently forming, which you can check to see if a translation is in progress.
  • There are also a variety of WordPress Plugins to help create bilingual posts, translations, and change languages between posts. For more information see the Official WordPress Plugin Directory and WordPress Plugin Database.
  • If you want the admin interface to use and download the wordpress translation files available at http://svn.automattic.com/wordpress-i18n/. Then wp-native-dashboard is very useful. (I have added it specifically here as it did not come up easily when searching. It provides partially the missing ability to upgrade language files from the svn.)
