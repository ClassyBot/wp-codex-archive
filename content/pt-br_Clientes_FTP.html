Para acessar os arquivos de um servidor através de FTP, você tanto pode usar o gerenciador de arquivos do seu serviço de hospedagem quanto um Cliente de FTP.

O cliente de FTP se conecta ao seu servidor onde estão seus arquivos. É preciso uma senha e nome de usuário para se conectar e acessar estes arquivos, estas informações sempre são providas pelo serviço de hospedagem.

Gerenciador de Arquivos[edit]

Os serviços de hospedagem oferecem paineis com ferramentas que incluem um gerenciador de arquivo, asism você edita, envia, muda permissões etc diretament pelo painel do serviço.

Entre no painel do serviço de hospedagem e procure por "Gerenciador de Arquivos", "Enviar Arquivos" ou entre em contato com o serviço.

Clientes de FTP[edit]

Net2FTP permite conectar a um site por FTP


FireFTP é uma extensão para FIreFox


Em vez de entrar no painel do serviço, você pode realizar as tarefas de arquivos a partir de seu computador, usando um software cliente de FTP.

Há muitos tipos de software, gratuitos e pagos. Entre os softwares gratuitos mais populares estão:

  • FileZilla
  • Fugu
  • LeechFTP
Extensões para navegadores
  • FireFTP - Extensão para FIrefox, disponível em português
Baseado em web
  • Net2FTP - Cliente FTP baseado em web, disponível em português.
