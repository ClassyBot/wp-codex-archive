Описание[edit]

get_post_ancestors() возвращает родительскую запись или страницу для заданной. В качестве аргумента принимает ID записи.

Использование[edit]

<?php get_post_ancestors( $post ) ?>

Параметры[edit]

$post
(mixed) (required) ID записи или ее объект.
Default: None

Возвращаемые значения[edit]

Массив состоящий из ID родительских записей или пустую строку в случае отсутствия таковых.

Исходный файл[edit]

Функция get_post_ancestors() объявляется в /wp-includes/post.php
