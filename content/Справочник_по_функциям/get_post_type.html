Описание[edit]

Возвращает тип записи текущей записи или переданной записи в качестве параметра.

Использование[edit]

<?php get_post_type( $post ) ?>

Параметры[edit]

$post
(mixed) (optional) Объект Post или ID записи.
Default: false

Возвращаемые значения[edit]

(boolean|string) 
тип записи или false при ошибке.

Примеры[edit]

Примечание[edit]

  • Использует: $wpdb
  • Использует: $posts Глобальную переменную post в цикле WordPress

История изменений[edit]

Начиная с: 2.1.0

Исходный файл[edit]

get_post_type() содержится в wp-includes/post.php.
