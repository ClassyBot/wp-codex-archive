Описание[edit]

Создает или добавляет новые элементы в админ-баре.

Это не функция. Это метод глобального объекта $wp_admin_bar (экземпляр класса WP_Admin_Bar), который может не всегда существовать, однако всегда существует в хуке 'admin_bar_menu'.

Примечание: Админ-бар был заменен на панель инструментов, начиная с WordPress версии 3.3. Рекомендуемый способ добавления элементов на панель инструментов - использование функции add_node().

Начальными являются следующие элементы Админ-бара ($menu_id):

  • my-account-with-avatar - Консоль, Персональные настройки
  • new-content - Ярлык для всех типов записей и прочих оригинальных
  • comments - Модерация комментариев
  • appearance - Выбор темы и Виджетов

Использование[edit]

<?php $wp_admin_bar->add_menu( $args ) ?>

Параметры[edit]

$args
(array) (required) Массив аргументов.
Default: None

Аргументы[edit]

id
(string) (required) ID элемента.
Default: false
title
(string) (optional) Текст, отображаемый на панели инструментов. Включая разрешенные html-теги.
Default: false
parent
(string) (optional) ID родительского элемента.
Default: false
href
(string) (optional) Атрибут 'href' для ссылки. Если 'href' не указан, то элемент будет текстовым.
Default: false
group
(boolean) (optional) Это сделает из элемента группу (элементов), если указан как 'true'. Сгруппированные элементы не видны на панели инструментов, но добавленные внутрь элементы видны. Смотри add_group().
Default: false
meta
(array) (optional) Массив meta-данных для элемента.
Default: array()
  • 'html' - html, используемый для элемента.
  • 'class' - Атрибут класса для элемента списка, содержащего ссылку или текстовую запись.
  • 'rel' - Атрибут rel.
  • 'onclick' - Атрибут onclick для ссылки. Может задаваться только, если указан аргумент 'href'.
  • 'target' - Атрибут target для ссылки. Может задаваться только, если указан аргумент 'href'.
  • 'title' - Атрибут title. Будет применен к ссылке или к div'у, содержащему текстовый элемент.
  • 'tabindex' - Атрибут tabindex. Будет применен к ссылке или к div'у, содержащему текстовый элемент.
