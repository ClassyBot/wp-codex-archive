Описание[edit]

Этот условный тег проверяет, является ли текущая запись избранной записью, что означает галка "Закрепить эту запись на главной странице" была задана для этой записи. Это логическая функция, она возвращает значение TRUE или FALSE.

Использование[edit]

<?php is_sticky($post_ID); ?>

Параметры[edit]

$post_ID
(string) (необязательный) ID записи
Default: None

Возвращаемое значение[edit]

(boolean) 
True при успехе, false при неудаче.

Примеры[edit]

is_sticky();
// Когда отображается страница любой избранной записи.

is_sticky('17');
// Когда отображается избранная запись 17 (ID).
