Описание[edit]

Безопасный способ получения значений для именованых опций из таблицы options из базы данных. Если желаемая опция не существует или нет значения ассоциированного с ней, FALSE будет возвращен.

Использование[edit]

 <?php echo get_option( $show ); ?> 

Параметры[edit]

$show
(string) (required) Название опции для получения. Список опций по умолчанию может быть найден в Справочнике опций.
Default: None
$default
(mixed) (optional) Значение по умолчанию для возврата, если никакого значения не вернулось (в т.ч. если опция не найдена в базе данных).
Default: false

Значения[edit]

Подробнее: http://codex.wordpress.org/Option_Reference

Примеры[edit]

Показать название блога[edit]

Покажет название вашего блога в теге <h1>.

 <h1><?php echo get_option('blogname'); ?></h1> 

Показать кодировку[edit]

Покажет кодировку, которую использует ваш блог (например: UTF-8)

 <p>Кодировка: <?php echo get_option('blog_charset'); ?> </p> 

Получить E-mail администратора[edit]

Получить e-mail администратора блога и поместить в переменную.

 <?php $admin_email = get_option('admin_email'); ?>
