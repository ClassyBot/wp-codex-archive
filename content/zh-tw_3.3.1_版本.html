2012 年 1 月 3 日，WordPress 3.3.1 公開發佈。這是一個維護與安全性更新。

3.3.1 版本的 資料庫版本號（wp_options 中的 db_version）依然是 19470。

安裝／升級資訊[edit]

如果希望更新為 WordPress 3.3.1，可以從網站後台的 "控制台" > "更新" 選單進行自動下載與更新。或是前往 http://wordpress.org/download/release-archive/ 手動進行下載與更新。

安裝與更新 WordPress 的步驟說明：

  • 升級 WordPress

如果你是 WordPress 新手，我們建議你可以先閱讀：

  • WordPress 新手 - 如何開始
  • 使用 WorssPress 的第一步 或 升級 WordPress 的延伸說明
  • WordPress 教學

摘要[edit]

來自 發佈網誌的文章：「這個維護版本修正了 WordPress 3.3 版本的 15 個問題，以及影響 3.3 版本的一個跨網站腳本漏洞。感謝 Joshua H.、Hoang T.、Stefan Zimmerman、Chris K.以及 Go Daddy 安全團隊，發現了這個錯誤並回報給我們的安全團隊。」

3.3.1 版本的完整修改紀錄可以在 http://core.trac.wordpress.org/changeset?new=19669%40branches%2F3.3&old=19590%40trunk 找到
