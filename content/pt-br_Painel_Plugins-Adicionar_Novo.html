Plugins → Adicionar Novo[edit]

Plugins acessam o WordPress para estender sua funcionalidade com recursos personalizados. Os Plugins são desenvolvidos de forma independente do núcleo WordPress, por milhares de desenvolvedores em todo o mundo. Todos os plugins no Diretório de Plugin WordPress.org são compatíveis com a licença usada pelo WordPress. Você pode encontrar novos plugins para instalar pesquisando ou navegando na Diretório, bem aqui, na sua própria seção de Plugins.

Opções de Tela[edit]

Clique para selecionar o tipo de informação que quer deixar visível sobre os plugins como Versão, Classificação e Descrição.

Ajuda[edit]

Contém uma breve descrição sobre este painel e links para esta documentação.

Instalar Plugins[edit]

O topo desta tela contém seis links: Pesquisar, Enviar, Destaques, Populares, Mais recentes e Favoritos. Veja o que cada link significa:

Pesquisa[edit]

Oferece um campo de pesquisa para entrar uma palavra-chave e pesquisar por plugins no repositório do WordPress.org.

Nota': para obter mais resultados é necessário pesquisa por termos em inglês.

Resultados da pesquisa[edit]

Resultados de plugins


Ao enviar um termo para pesquisa você será levado para a página de resultados contendo uma lista de plugins com o termo que pesquisou.

  1. Observe que nesta página também passa a ser possível pesquisar plugins por autor e tag, além de palavra-chave através da lista suspensa no topo da lista, à esquerda.
  1. Em caso de existirem muitos resultados, há botões de navegação no topo da tabela, à direita.

Cada plugin é mostrado em uma tabela com os seguintes itens:

  • Nome

O nome do plugin

  • Versão

A versão do plugin. Quanto menor o número mais novo o plugin e quanto maior, mais antigo e atualizado é o plugin.

  • Classificação

A avaliação é feita pelos usuários no site de repositório do WordPress.org.

  • Descrição

Uma descrição sobre o plugin, geralmente em inglês.

Abaixo do nome de cada plugin da lista, há dois links:

  • Detalhes

Mostra todos os detalhes do plugin diretamente do WordPress.org em uma janela modal

  • Instalar agora

Inicia o donwload do arquivo do plugin para seu servidor e então começa a instalação do plugin

Instalando um plugin da lista[edit]

Ao clicar no link Instalar agora você será perguntado: Tem certeza que quer instalar esse plugin?, clique Sim para começar o download do arquivo do plugin para instalá-lo.

Enviar[edit]

Nesta tela é possível enviar um arquivo de plugin para instalar. O formato do arquivo deve ser .zip.

If you have a plugin in a .zip format on your local computer, you may install it by uploading the file from your local computer here.
  • text box - Enter the name of a Plugin zip file from your local computer.
  • Browse - Click to browse your computer's local hard drive and the select the Plugin zip file name is placed in the text box.
  • Install Now - Click this button to install the Plugin from the zip file detailed in the text box; display messages similar to the following:
Installing Plugin from file: plugin_file.zip
Unpacking the plugin package
Installing the plugin
Successfully installed the plugin.
Actions: Activate Plugin | Return to Plugins page
* As part of a successful Plugin installation, two links are displayed.
  • Activate Plugin - Click this link to activate the Plugin just installed.
  • Return to Plugins page - Click this to be directed to the Plugins Installed SubPanel.
