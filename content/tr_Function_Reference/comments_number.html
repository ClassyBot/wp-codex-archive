Açıklama[edit]

Yorum sayısını belirlemek için kullanılır.Döngü içinde kullanılmalıdır.

Değerini almak için get_comments_number () kullanın.

Kullanımı[edit]

<?php comments_number( $0yorum,$1yorum,xyorum);?>

Parametreler[edit]

$0yorum:Yorumyapılmadığı zaman gösterilecek string

$1yorum:1 tane yorum yapıldığında gösterilecek string

$xyorum:1 den fazla yorum yapıldığında gösterilecek strign (% Yorum yapıldı)
