Depuis la version WordPress 3.5 (décembre 2012), le gestionnaire des liens est caché par défaut. Il reste néanmoins visible pour les sites qui ont créés un Blogroll par le passé. La fonctionnalité peut être activée au moyen de l'extension Links Manager.

Créer et gérer un Blogroll[edit]

Blogroll 
Le blogroll est l'espace où vous placez les liens vers les blogs que vous lisez régulièrement - une manière sympathique d'afficher les blogs que vous jugez bons. Le Links Manager de WordPress vous permet de créer et de gérer ces liens sans effort.
Bookmarklet 
La simplicité commence par un élégant bookmarklet (script lancé depuis les favoris ou la barre de liens) que vous pouvez ajouter à vos favoris. Ajouter un lien vers un blog ou un site intéressant est aussi simple qu'un clic sur le favori à la visite de ce site.
Organiser 
Les liens peuvent être classés et proprement organisés.
Import 
Si vous avez déjà une liste de lien dans un fichier OPML, vous pouvez l'importer dans votre blog WordPress. Pour les utilisateurs d'autres moteurs de blog, cela signifie que vous pouvez importer votre blogroll de Blogrolling.com et ne plus avoir à utiliser un service tiers pour gérer votre blogroll.
Export 
Avons-nous mentionné que vous pouvez exporter un fichier OPML avec la liste de vos liens?
Affichage 
Comme pour le reste, vous avez à votre disposition des balises de modèles pour afficher votre blogroll selon vos goûts - par ordre alphabétique, par évaluation, par mise à jour.
