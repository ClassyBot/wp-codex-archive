简介 Introduction[edit]

阅读这一节请注意区别 header 的单复数。

File Header API 由WordPress中关于File Header的所有函数和钩子(hook)构成。在WordPress中，Themes 和 Plugins 由一个或多个文件构成，其中至少有一个包含了一些元信息(meta-information)， 其内容如 Name, Version, Author 等等，阐明了主题或插件的具体信息，这些元信息就是所谓的 File Headers ( 特别地，header的内容可以是Drop-Ins,Must-Use-Plugins )

File Headers 置于文件的开头(可以不必一定是文件的第一行)，每行一个header.每个header由一个 Name 和一个 Value 组成
