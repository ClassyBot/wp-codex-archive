Description[edit]

private_title_format is a filter applied to the post title when the post visibility is set to private.

Parameters[edit]

$format
(string) (required) The string to output
Default: Private: %s
$post
(WP_Post) (required) The related post
Default: None

Examples[edit]

/* Removing protected prefix from post titles */
function spi_remove_private_protected_from_titles( $format, $post ) {
    return '%s';
}
add_filter( 'private_title_format', 'spi_remove_private_protected_from_titles' );

Source File[edit]

get_the_title() is located in wp-includes/post-template.php.
