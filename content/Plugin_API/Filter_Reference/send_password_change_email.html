Description[edit]

send_password_change_email filters whether to send the password change email. It is part of the wp_update_user() function.

Parameters[edit]

$send
(bool) (required) Whether to send the email.
Default: None
$user
(array) (optional) The original user array.
Default: None
$user_data
(array) (optional) The updated user array.
Default: None

Examples[edit]

To disable the notice of changed password sent by the wp_update_user() function, simply use the filter hook to return false: 

add_filter( 'send_password_change_email', '__return_false' );
