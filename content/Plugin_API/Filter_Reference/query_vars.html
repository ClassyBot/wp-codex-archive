Description[edit]

Allows query vars to be added, removed, or changed prior to executing the query.

Example[edit]

function themeslug_query_vars( $qvars ) {
  $qvars[] = 'custom_query_var';
  return $qvars;
}
add_filter( 'query_vars', 'themeslug_query_vars' );
