Description[edit]

Fires immediately after a new user is created. Either when a user self-registers, or a Super Admin creates a new user. Hook to 'wpmu_new_user' for events that should affect all new users, but only on Multisite (otherwise use 'user_register').


Parameters[edit]

This hook is passed one parameter:

$user_id
The user ID.
