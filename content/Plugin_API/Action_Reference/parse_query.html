Description[edit]

parse_query is an action triggered after WP_Query->parse_query() has set up query variables (such as the various is_ variables used for conditional tags).

Parameters[edit]

A registered action function is passed the following parameters.

$wp_query
(WP_Query) (required) The query object that parsed the query.
Default: None

Usage[edit]

add_action('parse_query', 'set_custom_isvars');
function set_custom_isvars( $wp_query ) {
    $wp_query->is_foo = ...
}

Change Log[edit]

  • In wp-includes/classes.php since 1.5
  • Moved from wp-includes/classes.php to wp-includes/query.php in 2.1

Source File[edit]

Triggered by WP_Query->parse_query() in wp-includes/query.php.
