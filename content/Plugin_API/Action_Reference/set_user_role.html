Description[edit]

The 'set_user_role' action hook triggers any time a users role is changed.

Parameters[edit]

$user_id
(int) (required) The user's ID
Default: None
$role
(string) (required) The user's "new" role
Default: None
$old_roles
(array) (optional) The user's previous role(s). Added in v3.6.
Default: None
