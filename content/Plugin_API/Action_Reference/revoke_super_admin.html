Description[edit]

The 'revoke_super_admin' action hook triggers any time a users "Super Admin" status is changed.

A Note About Capabilities[edit]

Super Admin is 'not' an actual role object in WordPress (unlike Subscriber, Contributor, Editor, etc), but a special "user status". As such, in order to give Super Admins new Capabilities you must add the capabilities directly to the users, not the Super Admin role (which does not exist).

A list of WordPress Super Admins is stored in your WordPress install's 'site_admins' site options record.

Parameters[edit]

$user_id
(int) (required) The user's ID
Default: None

Source File[edit]

'revoke_super_admin' is location in wp-admin/includes/ms.php
