Tema nedir?

wordpress tema sistemi bizim web günlüğümüze giden sadece bir yoldur. WordPress Tema bir dizi dosya ile birlikte çalışan bir web günlüğü için temel birleştirici tasarımlı bir grafik arayüzü üretmektedir. Bu dosyalara şablon nedir. Tema, temel yazılım biçimini değiştirmeden sitenin görüntülenme biçimini değiştirir. Temalar şablon dosyaları, görüntü dosyaları (*. jpg, *. gif), stil (*. css), özel sayfaları yanı sıra, gerekli kod dosyaları (*. php) özelleştirilmiş içerebilir. Kullanmış olduğun şablonları özelleştirebilirsiniz. bu özelliği size wordpress sunmaktadır.

Yeni tema nasıl edinebilirim ?

wordpress.org üzerinde yeni temalar edinebilirsiniz. adresi ise http://wordpress.org/extend/themes/" href="http://wordpress.org/extend/themes/ WordPress Theme Directory

wordpress ilk kurulduğu zaman iki tema ile sizlerle buluşuyor. biri classic teması birde default teması vardır. Temaları yönetebilmek için yönetim paneli / Görünüm / tema kısmından ulaşabilirsiniz.

Temanın yüklenmesi

  1. Öncelikle beğenmiş olduğumuz temayı bilgisayarımıza indirelim
  2. ister bir ftp programı ile bağlanılm istersek yeni tema ekle alanından göz at tıklayıp yeni temamızı seçip yükleyelim
  3. eğer bir ftp programı kullanacaksak dosyaları zipten çıkartıp dosyaları wp-contents/themes dizinine yüklemeliyiz.
  4. Ve yine üstteki resimde gözüktüğü yerden yüklemiş olduğumuz temayı etkinleştir seçeneğinden etkinleştirebiliriz.

aktif Temayı seçme

  1. admin panel e giriş yapın
  2. girişteki tema değiştir linkine tıklayın
  3. açılan sayfadan istediğimiz temayı etkinleştire tıklayarak ana temamızı seçmiş oluyoruz.

Tema oluşturma

Öncelikle kendi sitemize özgü bir tema tasarlamak istiyorsak bu biraz zaman alır özellikle css kısmı eğer ki bilgimiz yok ise hiç bitmez. en iyisi hazır temalar bulup bu temaları en iyi duruma getirip özelleştirerek kullanmaktır.

Tema dosyaları;

  1. http://codex.wordpress.org/Creating_an_Error_404_Page 404 Template = 404.php (sayfa bulunamadı sayfası)
  2. Archive Template = archive.php (arşiv sayfası)
  3. http://codex.wordpress.org/Creating_an_Archive_Index Archive Index Page = archives.php (arşiv index sayfası)
  4. Comments Template = comments.php (yorum sayfası)
  5. Footer Template = footer.php (alt kısım)
  6. http://codex.wordpress.org/Designing_Headers Header Template = header.php (üst kısım)
  7. Links = links.php (linklerin bulunduğu sayfa)
  8. Main Template = index.php (anasayfa)
  9. http://codex.wordpress.org/Pages Page Template > = page.php (sayfaların gösterildiği sayfa)
  10. Popup Comments Template = comments-popup.php
  11. Post Template = single.php (yazıların gösterildiği sayfa)
  12. Search Form = searchform.php (arama formu)
  13. http://codex.wordpress.org/Creating_a_Search_Page Search Template = search.php (arama sonuçları sayfası)
  14. http://codex.wordpress.org/Customizing_Your_Sidebar Sidebar Template = sidebar.php (kenar çubuğu)
  15. Stylesheet = style.css (stil dosyamız)
