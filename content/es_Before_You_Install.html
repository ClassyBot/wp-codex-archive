Antes de instalar WordPress, compruebe que su alojamiento u hospedaje Web reune las condiciones necesarias (tranquilo, la mayoría las cumplen), y que usted posee los conocimientos técnicos necesarios (tranquilo, no es complicado). Ninguno, de los dos requisitos debería ser un problema pero está bien comprobarlo antes.

Para ver la lista de requerimientos que debe tener su alojamiento u hospedaje Web, vea Hospedando WordPress. Otro sitio dónde poder comprobarlo es en: La Página Oficial de Descargas.

¡Deje que WordPress le sea Instalado Gratuitamente![edit]

Si está interesado en usar WordPress en su propio sitio pero no tiene los conocimientos técnicos o no se siente capacitado para realizar la instalación usted mismo, puede pedir que la instalación le sea realizada por el equipo de WordPress, Wordpress.

Necesitará saber como usar un editor de texto para editar el archivo principal de configuración. Si es usuario de Windows puede usar el Bloc de notas que viene incluído. Si quiere, podrá editar los Archivos de Plantilla (ver Plantillas para más referencias). Es posible hacerlo a través del Panel de Administración de Wordpress pero es MUY RECOMENDABLE usar un "BUEN" Editor de Textos (también puede descargar editores de texto gratuitos como Notepad++ o PSPad, ambos programas gratuitos). Para mas información ver: Editando Archivos.

Podrá usar cualquier programa de FTP para subir archivos y cambiar los permisos en los archivos WordPress a su Sitio Web remoto y cambiar los permisos en los archivos (opcional). Puede usar FileZilla para esto.

¡Y eso es todo! Ahora ya está preparado para continuar con la Instalación.
