Antes de instalar WordPress, necesita verificar que su proveedor de hospedaje web cumpla las condiciones necesarias (¡no tenga pánico! La mayoría lo cumple) y que posea las habilidades requeridas (¡no tenga pánico! No es complicado). No poseer alguna de las dos puede ser un problema, pero es prudente que verifique primero.

Su servidor web[edit]

Por una lista de requerimientos en su servidor web, vea Hospedando WordPress. Otro lugar para verificar es el sitio oficial de descargas (en inglés).

Usted mismo[edit]

  • Necesitará saber como utilizar un editor de texto para editar el archivo principal de configuración. Si es un usuario de Windows, el Bloc de Notas puede hacerlo.
  • Después, talvez quiera editar los Archivos de plantilla. Puede hacer esto a través del Panel de Administración de Wordpress, pero usar un buen editor de textos es altamente recomendado. Para más información, lea Como editar archivos
  • Necesitará ser capaz de usar un programa de FTP para subir archivos y ajustar permisos de archivo (opcional). Puede utilizar FileZilla para esta tarea.
