Açıklama[edit]

Mevcut yorumun ID'sini gösterir. Etiket The Loop içerisinde veya bir yorum döngüsü içerisinde bulunmalıdır.

Değeri döndürmek için get_comment_ID kullanın.

Kullanım[edit]

 <?php comment_ID(); ?> 

Parametreler[edit]

Bu fonksiyon hiçbir parametre almaz.

Dönen Değerler[edit]

Dönen değeri görmek için get_comment_ID() inceleyebilirsiniz.

Örnekler[edit]

Varsayılan kullanım[edit]

<p>Bu <?php comment_ID(); ?> nolu yorum tüm yorumlar için id döndürür.</p>

Comment ID as Anchor ID[edit]

Uses the comment ID as an anchor id for a comment. 

<div id="comment-<?php comment_ID() ?>">
<?php comment_author() ?> tarafından</div>
<div class="comment-text"><?php comment_text() ?></div>

Notlar[edit]

Değişim Kaydı[edit]

Başlangıç: 1.5.0

Kaynak Dosya[edit]

comment_ID() fonksiyonu wp-includes/comment-template.php içersinde yer almaktadır.
