Começando com aWordPress versão 2.7, a capacidade de usar os atalhos do teclado para navegar e moderar comentários foi introduzida. Esses atalhos de teclado são projetados para economizar tempo, permitindo-lhe navegar rapidamente e executar ações em comentários. Se o seu blog recebe um grande número de comentários, você vai achar esses atalhos especialmente úteis.

Ativando atalhos de teclado[edit]

Os atalhos de teclado são habilitados na configuração de cada usuário e pode ser ativado ao visitar o perfil em Painel > Usuários> Seu Perfil. Marque a caixa da opção Atalhos de teclado: Ativar atalhos de teclado para moderação de comentários. para ativar atalhos de teclado na moderação de comentários.

Por favor note, os atalhos de teclado são projetados para usar as duas mãos, simultaneamente, em um layout de teclado estilo QWERTY.

O Significado de "Selecionado"[edit]

Os atalhos de teclado são usados para navegação e para ações. Para qualquer ato que afete um comentário, ele deve estar selecionadoem primeiro lugar. O comentário que é considerado selecionadoé indicado com um fundo azul claro. Por padrão, nenhum comentário está selecionado, assim, você precisa pressionar a tecla j, ou k, para começar a navegar.

Usando Atalhos de Teclado para Navegar[edit]

Navegar entre comentários é feita com j, ou k. Quando você visitar a página de comentários, nenhum comentário está selecionado, assim, primeiro pressione j ou k, para selecionar o primeiro comentário. Após isso, basta pressionar j para mover a seleção para o próximo comentário.

  • Pressionando J move a seleção atual (fundo azul) para baixo.
  • Pressionando K move a seleção atual (fundo azul) para cima.

Veja que se você chegar ao fim de uma página de comentários e pressionar j, novamente, você será levado para a próxima página e o primeiro comentário da próxima página será selecionado. Da mesma forma, pressionando kna parte superior de uma página comentário o levará para a página anterior, selecionando o comentário no final da página anterior. Isto torna muito fácil navegar através de uma longa lista de comentários para executar as ações necessárias.

Usando Atalhos de Teclado para Executar Ações[edit]

Para uma dessas ações para afetar um comentário, certifique-se que o comentário está selecionado (veja acima).

  • Pressionando a aprova o comentário selecionado.
  • Pressionando s marca o comentário atual como spam.
  • Pressionando d move o comentário para o lixo (WordPress 2.9) ou excluir o comentário atual.
  • Pressionando z' restaura o comentário atual do lixo ou ativa Desfazer quando essa linha é selecionada.

Note-se que as primeiras três ações (as ações mais comuns) usam as três teclas mais à esquerda de um teclado QWERTY. É recomendável que você altere sua mão esquerda para que o dedo indicador esteja em d em vez de f, como é normal.

  • Pressionando u desaprova o comentário selecionado, colocando-o de volta para moderação.
  • Pressionando r nicia uma resposta ao comentário inline atual (você pode pressionar Esc para cancelar a resposta).
  • Pressionando q ativa a Edição Rápida que permite a edição rápida do comentário atual.

Ações em massa[edit]

Você também pode executar uma ação em vários comentários de uma vez. Primeiro, você precisa selecionar os comentários para a ação em massa, usando x para verificar a caixa de seleção do comentário selecionado. Shift-x pode ser usado para alternar entre as caixas, invertendo o seu estado. Depois de ter verificado alguns comentários:

  • Pressionando Shift+a aprova as observações selecionadas.
  • Pressionando Shift+s marca os comentários selecionados como spam.
  • Pressionando Shift+dapaga as observações selecionadas.
  • Pressionando Shift+u desaprova as observações selecionadas.
  • Pressionando Shift+t move as observações selecionadas para o lixo.
  • Pressionando Shift+z recupera os comentários selecionados do lixo.
