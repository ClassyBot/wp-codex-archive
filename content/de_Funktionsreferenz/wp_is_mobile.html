Beschreibung[edit]

Dieser Conditional Tag überprüft, ob der Besucher die Seite mit einem Mobiltelefon besucht. Das ist eine boolsche Funktion, daher wird entweder TRUE oder FALSE zurückgegeben. Die Überprüfung erfolgt durch das Auslesen des User Agent Strings des Browsers ($_SERVER['HTTP_USER_AGENT'])

Verwendung[edit]

<?php wp_is_mobile(); ?>

Parameter[edit]

Diese Funktion benötigt keine Parameter

Beispiele[edit]

<?php if ( wp_is_mobile() ) {
	/* Zeige hier Inhalte an, die für mobile Endgeräte gedacht sind*/
} ?>

Änderungsprotokoll[edit]

  • 4.0 -- Erfolgt durch Durchsuchen von HTTP_USER_AGENT auf: [ Mobile, Android, Silk/, Kindle, BlackBerry, Opera Mini, OperaMobi ]. Auflösung, Fenstergröße oder andere Faktoren werden nicht beachtet.
  • Seit: 3.4

Quelltext[edit]

wp_is_mobile() ist enthalten in wp-includes/vars.php.
