Beschreibung[edit]

Holt die Revisionen des aktuellen Posts oder eines angegebenen Posts

Verwendung[edit]

<?php wp_get_post_revision( $post ); ?>

Parameter[edit]

$post
(int|object) (required) Post ID oder das Post Objekt
Default: None

Rückgabewerte[edit]

(array) 
Ein Array von Revisionen oder ein leeres Array wenn es keine Revision gibt

Anmerkung[edit]

  • Verwendet get_post() um den Post zu holen

Änderungsprotokoll[edit]

Seit: 2.6

Quelltext[edit]

wp_get_post_revision() ist enthalten in wp-includes/revision.php

Ähnlich[edit]

  • wp_is_post_revision() - Gibt an ob der übergebene Post eine Revision ist.
