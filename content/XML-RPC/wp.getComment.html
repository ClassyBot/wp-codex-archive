wp.getComment[edit]

Get comment data

Availability[edit]

WordPress 2.7+

Parameters[edit]

  • (string) blog_id
  • (string) username
  • (string) password
  • (int) comment_id

Response[edit]

  • (dateTime.iso8601) date_created_gmt
  • (int) user_id
  • (int) comment_id
  • (int) parent
  • (string) status (approve, hold, spam)
  • (string) content
  • (string) link
  • (int) post_id
  • (string) post_title
  • (string) author
  • (string) author_url
  • (string) author_email
  • (string) author_ip
  • (string) type (<empty>, trackback, pingback)
