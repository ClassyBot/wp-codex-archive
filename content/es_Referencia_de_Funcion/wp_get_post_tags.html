Descripción[edit]

Devuelve una lista de tag para un post específico.

Uso[edit]

<?php wp_get_post_tags( $post_id, $args ) ?>

Parámetros[edit]

$post_id
(entero) (opcional) El ID del Post.
Default: 0
$args
(array) (opcional) Sobrescribe los valores por defecto. Ver wp_get_object_terms() para obtener un listado de las opciones validas que pueden ser pasadas.
Default: array

Valores Retornados[edit]

(array) 
Listado de tags

Ejemplos[edit]

Para un post con los siguientes tags tag2, tag5 y tag6, el código 

$t = wp_get_post_tags($post->ID);
print_r($t);

Muestra 

Array
(
   [0] => stdClass Object
       (
           [term_id] => 4
           [name] => tag2
           [slug] => tag2
           [term_group] => 0
           [term_taxonomy_id] => 4
           [taxonomy] => post_tag
           [description] => 
           [parent] => 0
           [count] => 7
       )

   [1] => stdClass Object
       (
           [term_id] => 7
           [name] => tag5
           [slug] => tag5
           [term_group] => 0
           [term_taxonomy_id] => 7
           [taxonomy] => post_tag
           [description] => 
           [parent] => 0
           [count] => 6
       )

   [2] => stdClass Object
       (
           [term_id] => 16
           [name] => tag6
           [slug] => tag6
           [term_group] => 0
           [term_taxonomy_id] => 16
           [taxonomy] => post_tag
           [description] => 
           [parent] => 0
           [count] => 2
       )

)


Para tomar una lista solo de los ID de los tag para un post en particular: 

global $post;
$tag_ids = wp_get_post_tags( $post->ID, array( 'fields' => 'ids' ) );

and assuming the same dataset as the first example, $tag_ids would contain 

[4, 7, 16]
