Description[edit]

Redirects the users browser to one of the following in the order below...

  1. $_GET['ref']
  2. $_POST['ref']
  3. $_SERVER['HTTP_REFERER']
  4. $_GET['redirect']
  5. $_POST['redirect']
  6. $url

Parameters[edit]

$url
(string) (optional) Redirect URL
Default: [blank string]

Return Values[edit]

(null)

Usage[edit]

<?php wpmu_admin_do_redirect($url); ?>
