Description[edit]

Gets an active blog for user, either primary blog or from blogs list.

Parameters[edit]

$user_id
(cast) (required) The user id to look up.
Default: None

Return Values[edit]

(object) 
Returns an object holding the details of the blog.

Usage[edit]

<?php get_active_blog_for_user($user_id); ?>
