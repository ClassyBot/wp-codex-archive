Description[edit]

Deletes the specified blog option from the database.

Parameters[edit]

$id
(integer) (required) The id of the blog to delete from.
Default: None
$key
(string) (required) The key of the blog option to be deleted.
Default: None

Return Values[edit]

(null)

Usage[edit]

<?php delete_blog_option($id, $key); ?>
