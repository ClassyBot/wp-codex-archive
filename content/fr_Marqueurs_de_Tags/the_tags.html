Note: This function is only available in WordPress 2.3 or newer.

Description[edit]

Ce marqueur de modèle est apparu avec WordPress Version 2.3. the_tags affiche un lien vers le ou les tags auquel appartient un article. Ce marqueur doit être utilisé dans La Boucle.

Utilisation[edit]

 <?php the_tags('before', 'separator', 'after'); ?> 

Exemples[edit]

Affiche une liste de tags avec un retour à ligne après chacun d'eux.  <?php the_tags('Tags:', ',', '<br />'); ?> 

Utilisation par défaut[edit]

L'utilisation par défaut faire apparaître les tags sous forme de liste, avec la virgule (,) comme spéarateur et précédée du texte : Tags: . 

<p><?php the_tags(); ?></p>
Tags: WordPress, Computers, Blogging

Séparer avec une Flèche[edit]

Pour afficher les liens des tags avec une flèche (>) séparant les tags et le texte Social tagging: au début 

<?php the_tags('Social tagging: ',' > '); ?>
Social tagging: WordPress > Computers > Blogging

Séparer avec une puce[edit]

Pour afficher les liens des tags avec une puce(•) séparant les tags, le texte Tagged with: au début et un retour à la ligne à la fin 

<?php the_tags('Tagged with: ',' &bull; ','<br />'); ?>
Tagged with: WordPress • Computers • Blogging

Paramètres[edit]

before 
(string) Texte à afficher avant la liste des tags. Par défaut à Tags:
separator 
(string) Texte ou caractère à afficher entre chaque lien de tag. Par défaut, une virgule (,) entre chaque tag.
after 
(string) Texte à afficher avant la liste des tags. Aucun affichage par défaut.
