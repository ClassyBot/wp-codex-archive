Antes de instalar Wordpress, é preciso verificar se o seu servidor oferece todas as condições necessárias (a maioria oferece mas é bom verificar!) e que você também tenha alguns conhecimentos básicos necessários.

Para uma lista de verificação do que é necessário ser oferecido pelo servidor, veja Hosting WordPress. Outra lista é Página de Requerimentos Oficial.

Com foi dito, você também precisa ter algum conhecimento. Isso quer dizer, um conhecimento básico de como usar um editor de texto para editar o arquivo de configuração, por exemplo. Se você é um usuário Windos, O Bloco de Notas já é suficiente.

Mais tarde, você pode precisar os arquivos de Template do seu site, (veja Templates). Pode ser através do editor do Wordpress Administration Panels, mas pode ser melhor usar um editor com mais funcionalidades. Veja Editing Files para mais informações.

Você precisa saber como usar um programa de FTP para enviar arquivos e mudar suas permissões de acesso. Um aplicativo gratuito para isso é o FileZilla.

E é isso! Agora que está tudo devidamente verificado, é só partir para a Instalação.
