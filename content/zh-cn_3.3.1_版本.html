2012 年 1 月 3 日，WordPress 3.3.1 公开发布。这是一个维护与安全性更新。

3.3.1 版本的 资料库版本号（wp_options 中的 db_version）依然是 19470。

安装/更新信息[edit]

如果希望更新為 WordPress 3.3.1，可以从网站后台的 "控制台" > "更新" 选单进行自动下载与更新。或是前往 http://wordpress.org/download/release-archive/ 手动进行下载与更新。

安装与更新 WordPress 的步骤说明：

  • 升级 WordPress

如果你是 WordPress 新手，我们建议你可以先阅读：

  • WordPress 新手 - 如何开始
  • 使用 WorssPress 的第一步 或 升级 WordPress 的延伸说明
  • WordPress 教学

摘要[edit]

来自 发佈网誌的文章：「这个维护版本修正了 WordPress 3.3 版本的 15 个问题，以及影响 3.3 版本的一个跨网站脚本漏洞。感谢 Joshua H.、Hoang T.、Stefan Zimmerman、Chris K.以及 Go Daddy 安全团队，发现了这个错误并回报给我们的安全团队。」

3.3.1 版本的完整修改纪录可以在 http://core.trac.wordpress.org/changeset?new=19669%40branches%2F3.3&old=19590%40trunk 找到
