Descripción[edit]

Esta etiqueta de plantilla retorna la dirección que permite al usuario loguearse en el sitio.

Uso[edit]

 <?php echo wp_login_url( $redirect ); ?> 

Parameters[edit]

$redirect
(string) (optional) URL to redirect to on login.
Default: None

Examples[edit]

Uso predeterminado[edit]

<a href="<?php echo wp_login_url(); ?>" title="Login">Login</a>

Loguearse y redireccionar a la página anterior[edit]

<a href="<?php echo wp_login_url( get_permalink() ); ?>" title="Login">Login</a>

Loguearse y redireccionar a la página principal[edit]

<a href="<?php echo wp_login_url( get_bloginfo('url') ); ?>" title="Loguearse">Loguearse</a>

Change Log[edit]

Desde: 2.7.0

Archivo fuente[edit]

wp_login_url() está localizado en wp-includes/general-template.php.
