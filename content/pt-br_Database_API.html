Sobre[edit]

Esta página lista todas as páginas relacionadas a Database API. Cada cobertura das funções envolvidas e a utilização de uma conjunto de funcionalidades. Juntas elas formam o que poderia ser chamado de "WordPress Database API", que é a interface para plugin/theme/add-on criada por todo projeto WordPress em relação aos acesso dos dados como valores armazenados na camada de banco de dados.

Se você ler tudo isso, você deve ter um bom senso de como estender o WordPress através de Plugins que acessam o banco de dados de maneira fácil.
