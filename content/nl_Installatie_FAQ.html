Heb ik een database nodig?[edit]

WordPress vereist dat je MySQL database database hebt om informatie op te slaan. Dus ja, je hebt een database nodig.

Je kan een database aanmaken wanneer:

  1. Je er nog geen hebt.
  2. Je een aparte database voor je blog wilt en je een aardige webhost hebt die je toestaat om meerdere databases te hebben.

Het is niet essentieel om voor elke WordPress weblog een nieuwe database te hebben.

  • Als je dezelfde database voor meerdere WordPress installaties gebruikt, zorg er dan voor dat er in elk 'wp-config.php' bestand een unieke prefix staat.
  • Als je een nieuwe database opzet voor een blog, verander dan het 'wp-config.php' bestand zodat de database gegevens juist zijn.

Wat houdt de 403 error in die ik krijg als ik mijn site bezoek?[edit]

Symptomen: Je kan niet op je adminaccount inloggen nadat je de juiste gebruikersnaam en wachtwoord hebt gebruikt. Je krijgt een foutbericht zoals: 

 You are not authorized to view this page
 You might not have permission to view this
 directory or page using the credentials you 
 supplied. If you believe you should be able 
 to view this directory or page, please try 
 to contact the Web site by using any e-mail 
 address or phone number that may be listed 
 on the personal.fredsmith.com home page. 
 You can click Search to look for information
 on the Internet. HTTP Error 403 - Forbidden''

Oplossing: Als de webhost een Windows server is, kan er een probleem zijn met de 'Directory Indexes'. De standaard configuratie van de Windows server is om index.htm, index.html, default.htm, default.html of default.asp te laden.

WordPress is gemaakt in PHP, de indexpagina is index.php. Dit is dus geen onderdeel van de standaard configuratie van de server, we moeten het dus toevoegen. Om dit te doen klik je op Web Options in het Control Panel. Vervolgens scroll je naar de Directory Indexes sectie om index.php aan de Directory Indexes toe te voegen.

Neem eventueel contact op met je webhost voor hulp.

Waarom is mijn uploadmap "C:apachehtdocswordpress"?[edit]

Toen je het uploadpad opgaf, gebruikte je 'backslashes'.

Gebruik in plaats daarvan de voorwaartse slashes "/" om het uploadpad op te geven.
