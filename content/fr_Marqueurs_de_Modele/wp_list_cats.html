Description[edit]

Affiche une liste des Catégories (en anglais) sous forme de liens. Quand un de ces liens est cliqué, tous les articles de la catégorie correspondante s'affiche dans le Modèle de Catégorie (en anglais) approprié défini par les règles de Hiérarchie des Modèles (en anglais). Ce marqueur fonctionne comme le marqueur list_cats, à l'exception du fait qu'il accepte une chaîne de requête quand list_cats emploie une liste de paramètres.

Utilisation[edit]

 <?php wp_list_cats('arguments'); ?> 

Exemples[edit]

Utilisation par Défaut[edit]

Par défaut, le marqueur :

  • optionall - N'affiche pas de liens vers l'ensemble des catégories
  • all - Texte à afficher pour le lien vers l'ensemble des catégories
  • sort_column - Effectue un tri des catégories par ID
  • sort_order - Effectue un tri ascendants des catégories
  • file - Affiche les catégories en utilisant le modèle index.php
  • list - Affiche les catégories dans une liste non-ordonnée (<ul><li>)
  • optioncount - N'affiche pas le décompte des articles
  • hide_empty - N'affiche pas les catégories vides
  • use_desc_for_title - Utilise la description comme titre des liens
  • children - Affiche les catégories enfants (sous-catégories) des catégories listées
  • hierarchical - Affiche les catégories enfants (sous-catégories) sous leur catégorie parente, dans une liste hiérarchique

 <?php wp_list_cats(); ?> 

Catégories avec Exclusions[edit]

Affiche les liens des catégories triés par nom, en affichant le nombre d'articles pour chaque catégorie et en excluant les catégories 10 et 15.

 <ul>
 <?php wp_list_cats('sort_column=name&optioncount=1&exclude=10, 15'); ?>
 </ul> 

Afficher Uniquement les Enfants d'une Catégorie[edit]

Affiche les liens des catégories triés par ID (sort_column=id), sans afficher le nombre d'articles par catégorie (optioncount=0, en affichant la description comme titre des liens (use_desc_for_title=0), uniquement les catégories enfants de la catégorie 8 (child_of=8).

 <?php wp_list_cats('sort_column=id&optioncount=0&use_desc_for_title=0&child_of=8'); ?> 

Note : S'il n'y a pas d'article dans une catégorie parent, elle ne s'affichera pas.

Catégories avec Liens vers les Fils RSS[edit]

Affiche les liens des catégories triés par nom, sans afficher le nombre d'article par catégorie et en affichant les liens vers les fils RSS de chaque catégorie.

 <?php wp_list_cats('sort_column=name&optioncount=0&feed=RSS'); ?> 

Notes sur l'Utilisation[edit]

Quand le paramètre 'list' est défini pour une liste non-ordonnée, le marqueur wp_list_cats() nécessite des marqueurs UL d'ouverture et de fermeture, mais crée automatique les marqueurs LI pour chaque élément de la liste.

Paramètres[edit]

optionall 
(booléen) Définit s'il faut affiche un lien vers l'ensemble des catégories. Note : Cette foncionnalité n'est plus supportée dans WordPress 1.5.x et 2.0 mais est sensée être rajoutée dans la version 2.1. Les options sont :
  • 1 (Vrai)
  • 0 (Faux) (Défaut)
all 
(chaîne) Si optionall est vrai, ce paramètre définit le texte à afficher comme lien vers l'ensemble des catégories. Note : Cette fonctionnalité n'est plus supportée par WordPress 1.5.x et 2.0 mais est sensée être rajoutée dans la version 2.1. Défini par défaut à 'Toutes'.
sort_column 
(chaîne) Clé de tri des catégories. Les options sont :
  • 'ID' (Défaut)
  • 'name'
sort_order 
(chaîne) Ordre de tri des catégories. Les options sont :
  • 'asc' (Ascendant) (Défaut)
  • 'desc' (Descendant)
file 
(chaîne) Le fichier php vers lequel le lien de catégorie doit pointer. Défini par défaut à index.php.
list 
(booléen) Définit si les catégories sont incluses dans une liste non-ordonnée (<ul><li>). Les options sont :
  • 1 (Vrai) (Défaut)
  • 0 (Faux)
optiondates 
(booléen) Définit s'il faut affiche la date du dernier article pour chaque catégorie. Les options sont :
  • 1 (Vrai)
  • 0 (Faux) (Défaut)
optioncount 
(booléen) Définit s'il faut affiche le décompte des articles pour chaque catégorie. Les options sont :
  • 1 (Vrai)
  • 0 (Faux) (Défaut)
hide_empty 
(booléen) Définit s'il faut cacher les catégories sans article. Les options sont :
  • 1 (Vrai) (Défaut)
  • 0 (Faux)
use_desc_for_title 
(booléen) Définit s'il faut utiliser la description des catégorie comme titre dfes liens (i.e. <a title="Category Description" href="...). Les options sont :
  • 1 (Vrai) (Défaut)
  • 0 (Faux)
children 
(booléen) Définit s'il faut afficher les catégories enfants (sous-catégories). Les options sont :
  • 1 (Vrai) (Défaut)
  • 0 (Faux)
child_of 
(entier) Affiche uniquement les catégories enfant de cette catégorie (définie par son ID). Pas de valeur par défaut. Si ce paramètre est utilisé, hide_empty est défini à faux.
feed 
(chaîne) Texte à afficher pour le lien du fil RSS2 (en anglais) de chaque catégorie. Par défaut, aucun texte n'est défini et le fil n'est pas affiché.
feed_image 
(chaîne) Chemin / Nom de fichier d'un image à utiliser comme link vers le fil RSS2 de chaque catégorie. Surcharge le paramètre feed.
exclude 
(chaîne) Définit les catégories à exclure (par leur ID), séparées par une virgule s'il y en a plusieurs (ex: '1, 2, 3').
hierarchical 
(booléen) Définit s'il faut afficher les catégories enfants (sous-catégories) dans une liste hiérarchique. Les options sont :
  • TRUE (Vrai) (Défaut)
  • FALSE (Faux)
Note : Le paramètre hierarchical n'est pas disponible dans les version antérieures à 1.5 de WordPress.
