Description[edit]

Displays links associated with a numeric link category ID. This tag uses the settings you specify in the Links Manager. For control over the formatting and display of your links within the tag's parameters, see get_links(). Affiche les liens associés à un identifiant numérique (ID) de catégorie de liens. Ce marqueur utilise les paramètres que vous aviez spécifié dans le Gestionnaire de Liens (en anglais). Pour un contrôle sur le formattage et l'affichage de vos liens avec les paramètres du marqueur, consultez get_links().

Utilisation[edit]

 <?php wp_get_links(category); ?> 

Exemples[edit]

Liens par Catégorie[edit]

Affiche les liens pour la catégorie de lien 1.

 <?php wp_get_links(1); ?> 

Inclure Textes "Avant" et "Après"[edit]

Simule le comportement de get_links_list() mais respecte les paramètres "Avant", "Entre" et "Après" définis pour les Catégories de Liens dans le Gestionnaire de liens. 

<ul>
 <?php
 $link_cats = $wpdb->get_results("SELECT cat_id, cat_name FROM $wpdb->linkcategories");
 foreach ($link_cats as $link_cat) {
 ?>
  <li id="linkcat-<?php echo $link_cat->cat_id; ?>"><h2><?php echo $link_cat->cat_name; ?></h2>
   <ul>
    <?php wp_get_links($link_cat->cat_id); ?>
   </ul>
  </li>
 <?php } ?>
</ul>

Paramètres[edit]

category 
(entier) Identifiant numérique de la catégorie dont les liens vont être affichés. Vous devez sélectionner une catégorie de liens, si aucune n'est spécifiée, la fonction génère une erreur.
