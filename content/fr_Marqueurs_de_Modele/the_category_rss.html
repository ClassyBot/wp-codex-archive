Description[edit]

Affiche le nom d'une ou plusieurs catégories auxquelles appartient l'article au format RSS (en anglais). Ce marqueur doit être utilisé à l'intérieur de la Boucle. Il inclut par défaut l'intégralité des catégories et mots clés associé à l'article.

Utilisation[edit]

 <?php the_category_rss('type') ?> 

Exemple[edit]

Extrait d'un fil RSS2.

 <?php the_category_rss() ?>
<guid><?php the_permalink($id); ?></guid> 

Paramètres[edit]

type 
(chaîne) Le type de fil à afficher. Les valeurs acceptées sont :
  • 'rss' (Défaut)
  • 'rdf'


Modification du contenu de the_category_rss[edit]

Fonction à utiliser 
Placez le code dans le fichier functions.php de votre thème

add_filter('the_category_rss', 'nom-de-la-fonction');
function nom-de-la-fonction($type = null) {$the_list = xxx;}

Remplacez le contenu xxx de $the_list par la valeur désirée
