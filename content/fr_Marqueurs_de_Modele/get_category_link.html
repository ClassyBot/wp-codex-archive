Description[edit]

Retourne l'URL d'une catégorie.

Exemple[edit]

Cet exemple affiche une liste de liens vers les sous-catégories de la catégorie 3 : <?php $categories=  get_categories('hierarchical=1&child_of=3'); ?>
<?php foreach ($categories as $cat) { ?>
<a href='<?php echo get_category_link(3); ?>'> 
<?php echo anyfont_replace_perso($cat->cat_name,'smenu actu'); ?>    
</a>
<?php } ?>
