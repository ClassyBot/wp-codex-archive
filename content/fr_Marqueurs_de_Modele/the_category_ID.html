Description[edit]

Affiche ou retourne l'identifiant numérique (ID) d'une catégorie auquelle appartient un article. Ce marqueur doit être utilisé à l'intérieur de la Boucle.

Cette fonction a été rendue obsolète. Ce qui signifie qu'elle a été remplacée par une nouvelle fonction ou qu'elle n'est plus supportée et qu'elle peut être retirée dans une prochaine version. Toute programmation l'utilisant doit être convertie par un équivalent s'il en existe.

Utilisation[edit]

 <?php the_category_ID(echo); ?> 

Exemple[edit]

Affiche une image correspondate pour chaque catégorie.

 <img src="<?php the_category_ID(); ?>.gif" /> 

Paramètres[edit]

echo 
(booléen) Si vrai (TRUE), affiche l'ID de la catégorie. Si faux (FALSE), retourne la valeur à PHP. TRUE par défaut.

Remplacé Par[edit]

This tag was deprecated when multiple categories were added to WordPress, and there is no one-to-one correspondence with another tag. This PHP code block provides an example for how you can replace it: Ce marqueur a été rendu obsolète quand les catégories multiples ont été ajoutées à WordPress, et il n'y a pas de correspondance avec un autre marqueur. Cet extrait de programmation fournit un exemple par lequel vous pouvez le remplacer.

 <?php 
 foreach((get_the_category()) as $category) { 
     echo $category->cat_ID . ' '; 
 } ?>
