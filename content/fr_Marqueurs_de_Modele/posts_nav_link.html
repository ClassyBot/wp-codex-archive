Description[edit]

Affiche des liens pour les pages précédentes et suivantes. Utile pour fournir une navigation paginée des pages de sommaire, de catégorie et d'archive.

Pour afficher les liens de navigation précédents et suivants avec des articles individuels, consultez next_post_link() et previous_post_link().

Utilisation[edit]

 <?php posts_nav_link('sep', 'prelabel', 'nxtlabel'); ?> 

Note that since weblog posts are traditionally listed in reverse chronological order (with most recent posts at the top), there is some ambiguity in the definition of "next page". WordPress defines "next page" as the "next page toward the past". In WordPress 1.5, the default Kubrick theme addresses this ambiguity by labeling the "next page" link as "previous entries". See Example: Kubrick Theme Format. Notez que puisque les articles de blog sont traditionnellement listés dans l'ordre chronologique inverse (avec les articles les plus récents en premier), il y a une certaine ambiguité dans la définition de la "page suivante". WordPress définit la "page suivante" comme la "page suivante" vers le passé. Dans WordPress 1.5, les thème par défaut Kubrick (en anglais) corrigeait cette ambiguité en nommant le lien "page suivante" en "entrées précédentes". Consultez Exemple: Format du Thème Kubrick.

Exemples[edit]

Utilisation par Défaut[edit]

Par défaut, posts_nav_link ressemble à ça :

« Page Précédente — Page Suivante »

 <?php posts_nav_link(); ?> 

Dans un DIV Centré[edit]

Affiche les liens vers les pages précédentes et suivante ("page précédente · page suivante") centrées sur la page. 

<div style="text-align:center;">
 <?php posts_nav_link(' &#183; ', 'page précédente', 'page suivante'); ?>
 </div>

En Utilisant des Images[edit]

<?php posts_nav_link(' ', '<img src="images/prev.jpg" />', '<img src="images/next.jpg" />'); ?>

Format du Thème Kubrick[edit]

Le format du thème Kubrick (en anglais) pour la navigation des articles. 

<div class="navigation">
 <div class="alignleft"><?php posts_nav_link('','','&laquo; Entrées Précédentes') ?></div>
 <div class="alignright"><?php posts_nav_link('','Entrées Suivantes &raquo;','') ?></div>
 </div>

Personnaliser le Texte du Lien[edit]

Vous pouvez changer le texte dans chacun des liens et dans le texte enter les liens.

Vous pouvez retourner à la page précédente ou vous pouvez avancer à la page suivante.
<p><?php posts_nav_link(' ou ', 'Vous pouvez retourner à la page précédent', 'vous pouvez avancer à la page suivante'); ?>.</p>

Paramètres[edit]

sep 
(chaîne) Texte affiché entre les liens.
  • ' :: ' par défaut dans la version 1.2.x.
  • ' — ' par défaut dans la version 1.5.
prelabel 
(chaîne) Texte du lien pour la page précédente.
  • '<< Page Précédente' par défaut dans la version 1.2.x.
  • '« Page Précédente' dans la version 1.5.
nxtlabel 
(string) Texte du lien pour la page suivante.
  • 'Page Suivante >>' par défaut dans la version 1.2.x.
  • 'Page Suivante »' par défaut dans la version 1.5
