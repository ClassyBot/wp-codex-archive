Description[edit]

Retourne une liste des parents d'une catégorie, y compris la catégorie elle-même, triée par identifiant numérique (ID).

Utilisation[edit]

 <?php get_category_parents(category, display_link, separator, nice_name); ?> 

Exemple[edit]

Affiche les catégories parentes de la catégorie en cours avec les liens séparés par '»'.

 <?php echo(get_category_parents($cat, TRUE, ' &amp;raquo; ')); ?>

Paramètres[edit]

category 
(entier) ID de la catégorie pour laquelle renvoyer les parents. Défini par défaut à la catégorie en cours, si elle existe.
display_link 
(booléen) Définit s'il faut créer un lien pour chaque catégorie affichée. Les options sont :
  • TRUE (Vrai)
  • FALSE (Faux) (Défaut)
separator 
(chaîne) Texte à utiliser pour séparer les catégories. Barre oblique (/) par défaut.
nice_name 
(booléen) Définit s'il faut utiliser l'identifiant comme nom de la catégorie. Les options sont :
  • TRUE (Vrai)
  • FALSE (Faux) (Défaut)
