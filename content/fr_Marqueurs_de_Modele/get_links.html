Description[edit]

Comme [[fr:Marqueurs_de_Modele/wp_get_links|wp_get_links()], ce marqueur affiche les liens du Gestionnaire de Liens (en anglais), mais autorise l'utilisateur à contrôler comment ils seront affichés grâce à ses paramètres, plutôt qu'au travers de l'interface d'administration de WordPres (utile lorsque vous affichez les liens sur plus d'un modèle). Ce marqueur ignore tout lien invisible.

Utilisation[edit]

 <?php get_links(category, 'before', 'after',
'between', show_images, 'order', 
show_description,show_rating, limit, 
show_updated, echo); ?> 

Exemples[edit]

Utilisation par Défaut[edit]

Par défaut, utiliser ce marqueur :

  • Affiche tous les liens
  • Ajoute une nouvelle ligne après chaque lien
  • Utilise une image s'il elle est disponible
  • Ajoute un espace entre l'image et le texte
  • Trie la liste par nom
  • Montre la description du lien
  • Ne montre pas la notation
  • Si aucune limite n'est définie, affiche tous les liens
  • Affiche les liens comme liens et pas comme texte

 <?php get_links(); ?> 

Catégorie Spécifique Triée par URL[edit]

Affiche les liens de la catégorie 2 entre des marqueurs span, utilise les images des liens, n'affiche pas les descriptions, trie par URL.

 <?php get_links(2, '&lt;span&gt;', '&lt;/span&gt;', , TRUE, 'url', FALSE); ?> 

Notation et Horodatage[edit]

Affiche tous les liens dans une liste ordonnée avec description sur une nouvelle ligne, n'utilise pas les image pour les liens, trie par identifiant de lien, affiche les notations et la date de la dernière mise à jour.

 <ol>
 <?php get_links('-1', '&lt;li&gt;', '&lt;/li&gt;', '&lt;br /&gt;', FALSE, 'id', TRUE, 
 TRUE, -1, TRUE); ?>
 </ol> 

Paramètres[edit]

category 
(entier) Identifiant numérique (ID) de la catégorie de liens dont les liens seront affichés. Si aucune n'est spécifiée, tous les liens seront affichés. -1 par défaut (tous les liens).
before 
(chaîne) Texte à placer avant chaque lien. Pas de valeur par défaut.
after 
(chaîne) Texte à placer après chaque lien. <br /> par défaut.
between 
(chaîne) Texte à placer entre chaque lien / image et sa description. ' ' (espace) par défaut.
show_images 
(booléen) Les images des liens devraient être affichées (TRUE - Vrai) ou pas (FALSE - Faux). Vrai par défaut.
order 
(chaîne) Valeur de tri des liens. Utilise 'name' par défaut, sauf si vous passez la valeur '' et c'est alors 'id' qui est pris en compte. Les options sont :
  • 'id'
  • 'url'
  • 'name'
  • 'target'
  • 'category'
  • 'description'
  • 'owner' - l'utilisateur qui a ajouté le lien via le Gestionnaire de Liens.
  • 'rating'
  • 'updated'
  • 'rel' - relation du lien (XFN).
  • 'notes'
  • 'rss'
  • 'length' - Longueur du nom du lien, du plus court au plus long.
Préfixez n'importe laquelle des options ci-dessus avec un caractère de soulignement (ex: '_id') pour effectuer le tri dans l'ordre inverse.
  • 'rand' - Affiche les liens dans un ordre aléatoire.
show_description 
(booléen) La description devrait être affichée (TRUE - Vrai) ou pas (FALSE - Faux). Valide quand show_images est faux, ou quand aucune image n'a été définie pour le lien. Vrai par défaut.
show_rating 
(booléen) La notation devraient être affichée (TRUE - Vrai) ou pas (FALSE - Faux). Faux par défaut.
limit 
(entier) Nombre maximum de liens à afficher. -1 par défaut (tous les liens).
show_updated 
(booléen) La date de dernière mise à jour devrait être affichée (TRUE - Vrai) ou pas (FALSE - Faux). Faux par défaut.
echo 
(booléen) Affiche les liens (TRUE - Vrai) ou les retourne pour les utiliser avec PHP (FALSE - Faux). Vrai par défaut.
