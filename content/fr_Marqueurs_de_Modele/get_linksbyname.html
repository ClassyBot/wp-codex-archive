Description[edit]

Comme wp_get_linksbyname(), ce marqueur affiche les liens du Gestionnaire de Liens (en anglais), mais permet à l'utilisateur de contrôler l'affichage des liens grâce à ses paramètres, plutôt qu'au travers de l'interface d'administration de WordPress (utile si vous affichez les liens sur plusieurs modèles différents).

Utilisation[edit]

 <?php get_linksbyname('cat_name', 'before', 
'after', 'between', show_images, 'orderby',
show_description, show_rating, limit, 
show_updated); ?> 

Exemples[edit]

Utilisation par Défaut[edit]

Par défaut, ce marqueur :

  • Affiche toutes les catégories si aucune n'est spécifiée
  • Affiche tous les liens
  • Ajoute une nouvelle ligne après chaque lien
  • Ajoue un espace entre l'image et le lien (si une image a été définie)
  • Affiche les images des liens si disponibles
  • Trie la liste par ID
  • Affiche la description des liens
  • N'affiche pas la notation
  • Sans limite définie, affiche tous les liens
  • N'affiche pas la date de dernière mise à jour

 <?php get_linksbyname(); ?> 

Catégorie Spécifique Triée par Nom[edit]

Affiche les liens de la catégorie "Amis" dans une liste non-ordonnée avec les descriptions sur une nouvelle, triés par nom.

 <ul>
 <?php get_linksbyname('Amis', '<li>', '</li>', '<br />', FALSE, 'name', TRUE); ?>
 </ul> 

Avec les Images et la Notation, sans la Description[edit]

Affiche tous les liens chacun sur une ligne sans description, en utilisant les image, triés par nom, et en affichant la notation.

 <nowiki><?php get_linksbyname(, , '<br />', , TRUE, 'name', FALSE, TRUE); ?></nowiki> 

Comme une Liste de Définitions avec Image et Description dans des Marqueurs Différents[edit]

Affiche tous les liens dans une liste de définitions, les images placées dans le <dt>, les descriptions dans le <dd>, et triés par notation mais sans l'afficher.

 <dl>
   <?php get_linksbyname('Portfolio', '<dt>',
        '</dd>','</dt><dd>', TRUE, 'rating',
        TRUE, FALSE, -1, FALSE); ?>
 </dl> 

Pour voir des exemples de stylisation de ce marqueur dans une disposition multi-colonnes fluide et élastique avec des effets de survolement, consultez Manipuler les Listes de Définition pour le Plaisir et le Travail (en anglais).

Paramètres[edit]

cat_name 
(chaîne) Nom de la catégorie de lien dont les liens devront être affichés. Si aucune n'est spécifiée, toutes les catégories de liens seront affichées. 'noname' par défaut (tous les liens).
before 
(chaîne) Texte à placer avant chaque lien. Pas de valeur par défaut.
after 
(chaîne) Texte à placer après chaque lien. <br /> par défaut.
between 
(chaîne) Texte à placer entre chaque lien / image et sa description. ' ' (espace) par défaut.
show_images 
(booléen) Les images des liens devraient être affichées (TRUE - Vrai) ou pas (FALSE - Faux). Vrai par défaut.
orderby 
(chaîne) Valeur de tri des liens. 'id' par défaut. Les options sont :
  • 'id'
  • 'url'
  • 'name'
  • 'target'
  • 'category'
  • 'description'
  • 'owner' - utilisateur qui a ajouté le lien grâce au Gestionnaire de Liens
  • 'rating'
  • 'updated'
  • 'rel' - relation avec le lien (XFN).
  • 'notes'
  • 'rss'
  • 'length' - longueur du nom du lien, du plus court au plus long
Préfixez n'importe laquelle des options ci-dessus avec un caractère de soulignement (ex: '_id') pour effectuer un tri dans l'ordre inverse.
  • 'rand' - Affiche les liens dans un ordre aléatoire
show_description 
(booléen) Affiche la description (TRUE - Vrai) ou pas (FALSE - Faux). Valide si show_images est faux ou si aucune image n'a été définie pour le lien. Vrai par défaut.
show_rating 
(booléen) Affiche la notation (TRUE - Vrai) ou pas (FALSE - Faux). Faux par défaut.
limit 
(entier) Nombre maximum de liens à afficher. -1 par défaut (tous les liens).
show_updated 
(booléen) Affiche la date de dernière mise à jour (TRUE - Vrai) ou pas (FALSE - Faux). Faux par défaut.
