Description[edit]

Affiche une liste des Catégories (en anglais) sous forme de liens. Quand un de ces liens est cliqué, tous les articles de la catégorie correspondante s'affiche dans le Modèle de Catégorie (en anglais) approprié défini par les règles de Hiérarchie des Modèles (en anglais). Ce marqueur fonctionne comme le marqueur wp_list_cats, à l'exception du fait qu'il accepte une liste de paramètres quand wp_list_cats emploie une chaîne de requête.

Utilisation[edit]

 <?php list_cats(optionall, 'all', 'sort_column',
                'sort_order', 'file', list, optiondates, 
                optioncount, hide_empty, use_desc_for_title, 
                children, child_of, 'Categories', recurse, 
                'feed', 'feed_img', 'exclude', 
                hierarchical); ?> 

Exemples[edit]

Liste par Défaut[edit]

Affiche la liste des catégories en utilisant les paramètres par défaut :

 <?php list_cats(); ?> 

Triée par Nom de Catégorie[edit]

Affiche la liste des catégories, sans lien vers l'ensemble des catégories, et triée par nom :

 <?php list_cats(FALSE, ' ', 'name'); ?> 

Liste Personnalisée avec Catégories Exclues[edit]

Configure la liste pour ne pas afficher de lien vers l'ensemble des catégories (basées sur les paramètres suivants), triée par ID dans l'ordre ascendant et dans une liste non-ordonnée (<ul><li>) sans date ni décompte d'article, sans cacher les catégories vides, en utilisant la "description" des catégories comme titre des liens, sans affiche les catégories enfants, et en excluant les catégorie 1 et 33 :

 <?php list_cats(FALSE, , 'ID',
                'asc', , TRUE, FALSE, 
                FALSE, FALSE, TRUE, 
                FALSE, FALSE, , FALSE, 
                , , '1,33', 
                TRUE); ?> 

Notes sur l'Utilisation[edit]

Quand le paramètre 'list' est défini pour une liste non-ordonnée, le marqueur list_cats() commence et termine la liste avec UL et chaque élément par LI

Paramètres[edit]

optionall 
(booléen) Définit s'il faut affiche un lien vers l'ensemble des catégories. Note : Cette foncionnalité n'est plus supportée dans WordPress 1.5.x et 2.0 mais est sensée être rajoutée dans la version 2.1. Les options sont :
  • TRUE (Vrai) (Défaut)
  • FALSE (Faux)
all 
(chaîne) Si optionall est vrai, ce paramètre définit le texte à afficher comme lien vers l'ensemble des catégories. Note : Cette fonctionnalité n'est plus supportée par WordPress 1.5.x et 2.0 mais est sensée être rajoutée dans la version 2.1. Défini par défaut à 'Toutes'.
sort_column 
(chaîne) Clé de tri des catégories. Les options sont :
  • 'ID' (Défaut)
  • 'name'
sort_order 
(chaîne) Ordre de tri des catégories. Les options sont :
  • 'asc' (Ascendant) (Défaut)
  • 'desc' (Descendant)
file 
(chaîne) Le fichier php vers lequel le lien de catégorie doit pointer. Défini par défaut à index.php.
list 
(booléen) Définit si les catégories sont incluses dans une liste non-ordonnée (<ul><li>). Les options sont :
  • TRUE (Vrai) (Défaut)
  • FALSE (Faux)
optiondates 
(booléen) Définit s'il faut affiche la date du dernier article pour chaque catégorie. Les options sont :
  • TRUE (Vrai)
  • FALSE (Faux) (Défaut)
optioncount 
(booléen) Définit s'il faut affiche le décompte des articles pour chaque catégorie. Les options sont :
  • TRUE (Vrai)
  • FALSE (Faux) (Défaut)
hide_empty 
(booléen) Définit s'il faut cacher les catégories sans article. Les options sont :
  • TRUE (Vrai) (Défaut)
  • FALSE (Faux)
use_desc_for_title 
(booléen) Définit s'il faut utiliser la description des catégorie comme titre dfes liens (i.e. <a title="Category Description" href="...). Les options sont :
  • TRUE (Vrai) (Défaut)
  • FALSE (Faux)
children 
(booléen) Définit s'il faut afficher les catégories enfants (sous-catégories). Les options sont :
  • TRUE (Vrai)
  • FALSE (Faux) (Défaut)
child_of 
(entier) Affiche uniquement les catégories enfant de cette catégorie (définie par son ID). Pas de valeur par défaut.
Categories 
(entier) Ce paramètre devrait être défini à 0 (zéro) à l'appel de ce marqueur de modèle. (Pour les curieux, les autres valeurs sont utilisées de façon interne uniquement par le marqueur pendant la génération d'une liste hiérarchique.)
recurse 
(booléen) Si faux (FALSE), affiche la liste. Si vrai (TRUE), retourne le résultat à PHP. Faux par défaut.
feed 
(chaîne) Texte à afficher pour le lien du fil RSS2 (en anglais) de chaque catégorie. Par défaut, aucun texte n'est défini et le fil n'est pas affiché.
feed_image 
(chaîne) Chemin / Nom de fichier d'un image à utiliser comme link vers le fil RSS2 de chaque catégorie. Surcharge le paramètre feed.
exclude 
(chaîne) Définit les catégories à exclure (par leur ID), séparées par une virgule s'il y en a plusieurs (ex: '1, 2, 3').
hierarchical 
(booléen) Définit s'il faut afficher les catégories enfants (sous-catégories) dans une liste hiérarchique. Les options sont :
  • TRUE (Vrai) (Défaut)
  • FALSE (Faux)
Note : Le paramètre hierarchical n'est pas disponible dans les version antérieures à 1.5 de WordPress.
