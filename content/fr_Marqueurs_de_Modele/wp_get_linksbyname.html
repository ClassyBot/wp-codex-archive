Description[edit]

Affiche les liens associés avec la catégorie de liens nommée. Ce marqueurs utilise les paramètres que vous avez spécifiés dans le Gestionnaire de Liens (en anglais). Pour un contrôle sur le formattage et l'affichage de vos liens grâce aux paramètres du marqueurs consultez get_linksbyname().

Utilisation[edit]

 <?php wp_get_linksbyname('category','arguments'); ?> 

Exemples[edit]

Utilisation par Défaut[edit]

Par défaut, ce marqueur :

  • Affiche toutes les catégories si aucune n'est spécifiée
  • Affiche tous les liens
  • Ajoute une nouvelle ligne après chaque lien
  • Ajoute un espace entre l'image et le lien (si une image a été définie)
  • Affiche les images des liens si disponibles
  • Trie la liste par ID
  • Affiche la description des liens
  • N'affiche pas la notation
  • Sans limite définie, affiche tous les liens
  • N'affiche pas la date de dernière mie à jour

 <?php wp_get_linksbyname(); ?> 

Affiche tous les liens de Catégorie de Liens appelée "Favoris".

 <?php wp_get_linksbyname('Favoris') ?> 

Affiche tous les liens de la Catégorie de Liens Blogroll, ordonnés par nom, sans leur description et avec la date de dernière mise à jour.

 <?php wp_get_linksbyname('Blogroll','orderby=name&show_description=0&show_updated=1') ?> 

Paramètres[edit]

category[edit]

category 
(chaîne) Nom de la catégorie dont les liens doivent être affichés. Pas de valeur par défaut. Si aucune catégorie n'est spécifiée, toutes les catégories de liens sont affichées.

arguments[edit]

before 
(chaîne) Texte à placer avant chaque lien. Pas de valeur par défaut.
after 
(chaîne) Texte à placer après chaque lien. <br /> par défaut.
between 
(chaîne) Texte à placer entre chaque lien / image et sa description. ' ' (espace) par défaut.
show_images 
(booléen) Détermine si les images doivent être utilisées.
  • 1 (Vrai - défaut)
  • 0 (Faux)
orderby 
(chaîne) Valeur de tri des liens. Les options sont :
  • 'id' (défaut)
  • 'url'
  • 'name'
  • 'target'
  • 'category'
  • 'description'
  • 'owner' - utilisateur qui a ajouté le lien grâce au Gestionnaire de Liens.
  • 'rating'
  • 'updated'
  • 'rel' - relation avec le lien (XFN).
  • 'notes'
  • 'rss'
  • 'length' - longueur du nom du lien, du plus court au plus long.
Préfixez n'importe laquelle des valeur ci-dessous avec un caractère de soulignement (ex: '_id') pour effectuer un tri dans l'ordre inverse.
  • 'rand' - Affiche les liens dans un ordre aléatoire.
show_description 
(booléen) Détermine s'il faut afficher la description. Valide si show_images est faux ou si aucune image n'a été définie pour le lien.
  • 1 (Vrai - défaut)
  • 0 (Faux)
show_rating 
(booléen) Détermine s'il faut afficher la notation.
  • 1 (Vrai)
  • 0 (Faux - défaut)
limit 
(entier) Nombre maximum de liens à afficher. -1 par défaut (tous les liens).
show_updated 
(booléen) Détermine s'il faut afficher la date de dernière mise à jour.
  • 1 (Vrai)
  • 0 (Faux - défaut)
echo 
(booléen) Affiche le lien (1 - vrai) or retourne la liste pour l'utiliser avec PHP (0 - faux). Vrai par défaut.
