Description[edit]

Produit une ancre [[permalien (en anglais) utilisant un identifiant ou un ID (<a id="....) pour un article. Cet utile pour lié un article spécifique sur une page en affichant plusieurs, comme une page d'archives. Ce marqueur doit être utilisé dans la Boucle (en anglais).

Utilisation[edit]

 <?php permalink_anchor('type'); ?> 

Exemple[edit]

Insère l'ancre permalien près du titre d'un article.

 <h3><?php permalink_anchor(); ?><?php the_title(); ?></h3> 

Paramètres[edit]

type 
(chaîne) Type d'ancre à produire. Les options sont :
  • 'id' - L'ancre utiliser l'identifiant numérique (ID). C'est l'option par défaut.
  • 'title' - L'ancre utilise le nom de l'article, également appelé post slug (en anglais).

En relation[edit]

permalink_anchor, get_permalink, the_permalink, permalink_single_rss
