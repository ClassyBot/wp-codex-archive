Description[edit]

Affiche l'extrait de l'article en cours. Si vous ne fournissez pas d'extrait explicite pour un article (dans le champ Extrait optionnel de l'éditeur d'article), les 55 premiers mots du contenu de l'article sont retournés. Dans le même temps, les marqueurs et les images HTML (en anglais) sont retirés du contenu de l'extrait. Ce marqueur doit être utilisé à l'intérieur de la Boucle (en anglais).

Utilisation[edit]

 <?php the_excerpt(); ?> 

Exemples[edit]

Utilisation par Défaut[edit]

Affiche l'extrait de l'article. Utilisé sur une page autre que celle d'un article seul ou d'un permalien comme remplaçant de the_content() pour forcer les extraits à s'afficher à l'intérieur de la Boucle.

 <?php the_excerpt(); ?> 

Utilisé avec des Marqueurs Conditionnels[edit]

Remplace le marqueur the_content() avec the_excerpt sur les pages d'archives (testées avec is_archive()) ou de catégorie (is_category()).

Les deux exemples suivants fonctionnent avec les versions 1.5 et ultérieures.

 <?php if(is_category() || is_archive()) {
     the_excerpt();
 } else {
     the_content();
 } ?> 

Pour les versions de WordPress inférieures à 1.5, seul cet exemple fonctionnera :

 <?php if($cat || $m) {
     the_excerpt();
 } else {
     the_content();
 } ?> 

Paramètres[edit]

Ce marqueur n'accepte pas de paramètre.
