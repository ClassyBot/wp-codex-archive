Description[edit]

Affiche une liste non-ordonnée HTML (en anglais) de tous les liens définis dans les Gestionnaire de Liens (en anglais), triés sous les Catégories de Liens (en anglais). Note : les titres des Catégories de Liens sont générées automatiquement dans des en-tête <h2>.

Note : Ce marqueur ne respecte pas les paramètres "Avant", "Entre" et "Après" définis pour les Catégories de Liens dans Gestionnaire de Liens (en anglais), mais respecte les autres Options de Catégorie. Pour contourner cete limitation, consultez l'Exemple 2 de wp_get_links().

Utilisation[edit]

 <?php get_links_list('order'); ?> 

Réside dans[edit]

wp-includes/links.php

Exemple[edit]

Affiche les liens triés par identifiant numérique (ID) de catégorie de liens.

 <?php get_links_list('id'); ?> 

Genère automatiquement le <li> avec l'ID de la Catégorie de Lien englobé dans un en-tête <tt><h2>. Cela ressemble à ceci : 

<li id="linkcat-1"><h2>Blogroll</h2>
   <ul>
      <li><a href="http://exemple1.com/">Lien Blogroll 1</a></li>
      <li><a href="http://exemple2.com/">Lien Blogroll 2</a></li>
      <li><a href="http://exemple3.com/">Lien Blogroll 3</a></li>
    </ul>
</li>

Paramètres[edit]

order 
(chaîne) Clé de tri des Catégories de Liens. Les options sont :
  • 'name' (Défaut)
  • 'id'
Préfixer les options ci-dessus avec un caractère de soulignement (ex: '_id') effectue le tri dans l'ordre inverse.
