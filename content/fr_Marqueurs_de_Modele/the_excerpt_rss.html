Description[edit]

Affiche l'extrait de l'article en cours formatté pour un fil RSS (en anglais). Si vous ne fournissez pas d'extrait explicite pour un article (dans le champ Extrait optionnel de l'éditeur d'article), les 55 premiers mots du contenu de l'article sont retournés. Ce marqueur doit être utilisé à l'intérieur de la Boucle (en anglais).

Pour un contrôle accru sur le résultat, consultez the_content_rss().

Utilisation[edit]

 <?php the_excerpt_rss(); ?> 

Consultez Paramètres pour la Version 1.2 pour les paramètres disponibles dans cette version.

Exemple[edit]

Affiche l'extrait de l'article, ou les 120 premiers mots du contenu de l'article quand aucun extrait n'existe, formatté pour l'aggrégation RSS.

 <description><?php the_excerpt_rss(); ?></description> 

Paramètres[edit]

Ce marqueur n'accepte pas de paramètre.

Paramètres pour la Version 1.2[edit]

WordPress versions 1.5 et ultérieures ne supporte pas de paramètre pour ce marqueur. Les informations qui suivent sont conservées pour les utilisateurs de la version 1.2.

Utilisation :  <?php the_excerpt_rss(cut, encode_html); ?> 

Paramètres :

cut 
(entier) Nombre de mots à afficher avant de terminer l'extrait. Peut être n'importe quelle valeur numérique inférieur ou égale à la valeur par défaut.
encode_html 
(integer) Defines html tag filtering and special character (e.g. '&') encoding. Options are:
  • 0 - (Default) Parses out links for numbered "url footnotes".
  • 1 - Filters through the PHP function htmlspecialchars(), but also sets cut to 0, so is not recommended when using the cut parameter.
  • 2 - Strips html tags, and replaces '&' with HTML entity equivalent (&amp;). This is the default when using the cut parameter.
