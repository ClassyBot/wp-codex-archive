Page d'accueil du Codex en français - Télécharger WordPress en français
Les utilisateurs francophones se retrouvent sur le site WordPress-Francophone, notamment sur son forum d'entraide.

Add Link Attribute 
Insérez vos propres attributs HTML dans (prétendument) les modèle de générateurs de liens, sans le besoin de récrire ces fonctions directement. Facultativement les attributs des tags <img>.
Another Title 
Réécrit les titres de vos publications à la volée, enlève les redondances telles que 'Blog Archive'.
Break Out Of Frames 
Ajoute un petit bout de code Javascript dans l'entête de chaque page web, afin qu'elles puisse sortir des cadres étrangés. Compatible avec les fonction Live Preview des éditeurs de WP2.
Dashboard Options 
Ajoute des fonctions très utiles à votre Tableau de bord (page d'accueil admin) de Wordpress. Active/Désactive les flus RSS WordPress Development + WordPress Other News + Technorati; affichage de messages personalisés; traite du texte en tant que script php (si vous connaissez le PHP) ou ajoute des flux RSS de votre choix.
Dean's URL Rewriting for WordPress under IIS 
Ce filtre ISAPI enlève index.php du Permalien de Wordpress sur IIS afin de le rendre plus joli et compatible référenceurs (SEO friendly).
German Permalinks 
Convertit les trémas allemands en permaliens.
Leftycats 
Par défaut, utiliser wp_list_cats avec une image RSS optionnelle, on obtient l'affichage de ces dernières à droite du nom des catégories. wp-leftycats fait la même chose que wp_list_cats, mais place les images RSS à gauche du nom des catégories.
Mime Config 
Ce plugin vous permet d'étendre la liste des types de fichiers (mime-types) supporté par l'uploader dans WordPress 2.0
Optimal Title 
Reflète exactement la fonction wp_title(), mais déplace la position du 'séparateur' après le titre plutôt qu'avant. Cela permet d'avoir le nom de votre blog à la fin du titre de la page plutôt qu'en premier.
Underscore Permalinks 
Utilise les "_" (souligné) plutôt que les "-" (tirets) dans les permaliens.
Url Encoded Permalinks 
Utilise l'encodage d'url - urlenncode() - pour encoder les caractères non-anglais pour les permaliens.
