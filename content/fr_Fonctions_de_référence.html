Les fichiers de WordPress définissent nombre de fonctions PHP utiles. Certaines de ces fonctions, connues comme Marqueurs de Modèle, sont définies spécifiquement pour être utilisées dans les thèmes WordPress. Il existe également un bon nombre de fonctions liées aux actions et filtres (le Plugin API), qui sont donc principalement utilisées pour le développement de plugins. Les autres fonctions sont utilisées pour créer les fonctionnalités du cœur de WordPress.

Beaucoup de fonctions du cœur de WordPress sont utiles pour les développeurs de plugins et de thèmes. Cet article liste donc ces fonctions de manière non exhaustive, les Marqueurs de Modèle n'étant pas inclus. Au pied de cette page se trouve une section listant les autres ressources pour trouver des informations sur les fonctions propres à WordPress. En plus de ces informations, le site WordPress phpdoc détaille toutes les fonctions des différentes version de WordPress depuis la version 2.6.1.

Vous pouvez aider à compléter cette page !

Voici comment vous pouvez aider :

  • Ajouter des sous-pages traitant des fonctions non détaillées sur cette page, ou ajouter des exemples sur les pages existantes Marqueurs de Modèle.
  • Ajouter des fonctions en suivant le modèle de la page.
  • Corriger les erreurs des contributions précédentes, il y en aura, nous sommes tous humains :)

Lire Contributing to WordPress pour obtenir des informations sur la manière dont vous pouvez compléter cette page.

Références officielles et non-officielles[edit]

  • WordPress.org PHPDoc - Nearly all the functions in WordPress have "PHPDoc" headers. This site shows all the PHPDoc.
  • Trac Browser - WordPress source code. Choose 'Tags' to find a specific version source code, or 'Trunk' to see the latest version source code.
  • PHPXref for WordPress - Cross reference of WordPress files, functions, and variables, which seems to point to the latest released version. This redalt.com PHPXref site has the bleeding-edge Trunk version, as well as 1.5 and 2.0, but the Trunk version may not be updating any more. So, there is now a new Trunk Xref site on ftwr.co.uk available.
  • WordPress Function History Table - List each WordPress function with including the version it was introduced in or dropped in. PlanetOzh recommends: "see related WordPress Functions Implementation History Tool for more information (and for a more usable tool)"
  • WordPress Variable Report - A list of WordPress functions, listing what version of PHP every function requires (and if there's a compatible function available), what varables/arguments in functions are unused, and what functions/variables each class contains (and their default values).
  • WordPress Hooks Table - A list of WordPress actions and filters, cross-referenced to the files they are in
