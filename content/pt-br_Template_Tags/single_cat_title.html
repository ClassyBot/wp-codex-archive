Descrição[edit]

Exibe ou retorna o título da categoria para a página atual. A função somente pode ser usada fora do loop (The Loop).

Sintaxe[edit]

 <?php single_cat_title( $prefix, $display ); ?> 

Sintaxe padrão[edit]

 <?php single_cat_title(
    'prefix'   => , 
    'display'  => true ); ?> 

Parâmetros[edit]

$prefix
(string) (optional) Texto a ser exibido antes do título.
Default: None
$display
(boolean) (optional) Determina se o título será mostrado quando (TRUE) ou returnado para uso em variáveis PHP (FALSE).
Default: TRUE

Exemplos[edit]

O exemplo mostra o texto "Categoria atual: " seguido do título da categoria. 

<p><?php single_cat_title('Categoria atual: '); ?>.</p>


O exemplo a seguir define para a variável $categoria o título da atual categoria para posterior uso no PHP. 

<?php $categoria = single_cat_title("", false); ?>
