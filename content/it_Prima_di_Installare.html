Prima di Installare WordPress, devi controllare che il tuo provider soddisfi le condizioni necessarie (niente panico! la maggior parte lo fa) e che tu stesso possegga le capacità richieste (niente panico! non è complicato). Nessuna delle due cose dovrebbe essere un problema, ma è una scelta saggia verificarlo prima.

Per una lista dei requisiti per il tuo web host, guarda Hostare WordPress. Un'altra pagina da controllare è quella dei requisiti ufficiali.

Dovrai saper usare un editor di testo per modificare il file di configurazione. Se utilizzi Windows, andrà bene Notepad. Dopo, probabilmente, vorrai modificare i tuoi File dei Template (vedi Templates per alcune raccomandazioni). Puoi farlo tramite il Pannello di Amministrazione, ma usare un buon editor di testo è altamente consigliato. Per maggiori informazioni in proposito, vedi Modificare i File.

Dovrai saper usare un programma FTP per caricare i file e configurare i permessi dei file (opzionale). Potresti usare FileZilla per questo.

Questo è tutto! Ora sei pronto per passare all'Installazione.
