Descrição[edit]

Esta Tag Condicional verifica se o página inicial do WordPress é um blog início ou uma Página estática. Esta é uma função booleana, isso quer dizer que retorna TRUE ou FALSE. Ela retorna TRUE quando a página inicial do blog está sendo exibida e o Configurações->Leitura->A página inicial mostra está definida como "Seus posts recentes", ou quando definido como "Uma página estática" e o valor "Página Inicial" Páginas está sendo exibido.

Uso[edit]

<?php is_front_page(); ?>

Parâmetros[edit]

Esta tag não aceita parâmetros

Valores Retornados[edit]

(boolean) 
True em sucesso, false em falha.

Exemplos[edit]

Se você estiver utilizando uma página estática como página inicial, isso será útil:
<title>
<?php bloginfo('name'); ?> &raquo; <?php is_front_page() ? bloginfo('description') : wp_title(); ?>
</title>

Notas[edit]

  • Veja também: is_home()

Desde: 2.5.0

Source File[edit]

is_front_page() está localizado em wp-includes/query.php.
