Descrição[edit]

Registra um único 'Menu de navegação personalizado' (Custom Navigation Menu). Permite incorporar o menu criado via Painel Administrativo (Aparência → Menus) ao tema em uso.

Veja register_nav_menus() para criar múltiplos menus.

Uso[edit]

 <?php register_nav_menu( $location, $description ); ?> 

Parâmetros[edit]

$location
(string) (required) Identificador de localização do menu, como um slug.
Default: None
$description
(string) (required) Não possui valor como padrão. Deverá ser especificado mesmo se vazio. Servirá para rotular o menu criado.
Default: None

Valores de Retorno[edit]

Nenhum.

Exemplos[edit]

if ( function_exists( 'register_nav_menu' ) ) {
	register_nav_menu( 'meu-menu', 'Meu Menu' );
}

Notas[edit]

  • A função automaticamente registra o suporte a 'Menu Personalizado' para o tema, por isso NÃO é preciso utilizar add_theme_support( 'menus' );

Change Log[edit]

  • Desde: 3.0.0
