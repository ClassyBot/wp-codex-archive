Descrição[edit]

Determina se o suporte ao Multisite está habilitado.

Uso[edit]

 <?php if ( is_multisite() ) { echo 'Multisite está habilitado'; } ?> 

Parâmetros[edit]

Esta função não aceita nenhum parâmetro.

Retornos[edit]

(BOOLEAN) 
True se o Multisite estiver habilitado. Caso contrário, false

Registro de alterações[edit]

Desde: WordPress 3.0.0

Código fonte[edit]

is_multisite() está localizado em wp-includes/load.php.
