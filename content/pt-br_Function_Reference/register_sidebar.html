Descrição[edit]

Constrói a definição para uma única Sidebar e retorna seu ID.

Uso[edit]

<?php register_sidebar( $args ); ?>

Argumentos padrão[edit]

<?php $args = array(
	'name'          => sprintf(__('Sidebar %d'), $i ),
	'id'            => 'sidebar-$i',
	'description'   => '',
	'before_widget' => '<li id="%1$s" class="widget %2$s">',
	'after_widget'  => '</li>',
	'before_title'  => '<h2 class="widgettitle">',
	'after_title'   => '</h2>' ); ?>

Parâmetros[edit]

args
(string/array) (opcional) Constrói a Sidebar baseado nos valores de 'name' e 'id'.
Default: None
  • name - Nome da Sidebar.
  • id - Identificador (ID) da Sidebar.
  • description - Descrição
  • before_widget - Texto a ser inserido antes de todo widget.
  • after_widget - Texto a ser inserido depois de todo widget.
  • before_title - Texto a ser inserido antes de todo título.
  • after_title - Texto a ser inserido depois de todo título.
