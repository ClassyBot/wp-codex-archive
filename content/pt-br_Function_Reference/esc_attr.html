Descrição[edit]

Codifica < > & " ' (sinal de menor, sinal de maior, 'e' comercial, aspas duplas, aspas simples). Não codificará as entidades HTML duas vezes.

Uso[edit]

<?php esc_attr( $texto ) ?>

Parâmetros[edit]

$texto
(string) (required) O texto a ser codificado.
Default: None

Valores de Retorno[edit]

(string) 
O texto codificado com entidades HTML.

Exemplos[edit]

<h1>
  <a href='<?php bloginfo( 'url' ); ?>' title='<?php esc_attr( bloginfo( 'name' ) ); ?>'>
    <?php bloginfo( 'name' ); ?>
  </a>
</h1>

Exibe como atributo do link o nome do blog codificado.

Arquivo Fonte[edit]

esc_attr() está localizada em wp-includes/formatting.php.
