Description[edit]

Updates the comment count for the post. It is used by wp_update_comment_count().

Usage[edit]

<?php wp_update_comment_count_now( $post_id ); ?>

Parameters[edit]

$post_id
(integer) (required) The post ID
Default: None

Return Values[edit]

(boolean) 
False if $post_id is not valid. True on success.

Examples[edit]

Notes[edit]

  • Uses: $wpdb - To update the 'comment_count' field for the post in the wp_posts table.
  • Uses: do_action():
    • Calls 'wp_update_comment_count' hook with $post_id, the new count, and the old count.
    • Calls 'edit_post' hook with $post_id and the post object.
