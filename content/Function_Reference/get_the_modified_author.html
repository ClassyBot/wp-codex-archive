Description[edit]

Retrieve the author who last edited the current post.

Usage[edit]

<?php echo get_the_modified_author(); ?>

Return Values[edit]

(string) 
The author's display name.

Source File[edit]

get_the_modified_author() is located in wp-includes/author-template.php.
