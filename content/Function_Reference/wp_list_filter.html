Description[edit]

Filters a list of objects, based on a set of key => value arguments.

Usage[edit]

<?php wp_list_filter( $list, $args, $operator ); ?>

Parameters[edit]

$list
(array) (required) An array of objects to filter
Default: None
$args
(array) (optional) An array of key => value arguments to match against each object
Default: array()
$operator
(string) (optional) The logical operation to perform: 'AND' means all elements from the array must match; 'OR' means only one element needs to match; 'NOT' means no elements may match. The default is 'AND'.
Default: AND

Return Values[edit]

(array)
