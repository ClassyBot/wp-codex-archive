Description[edit]

Update a blog details field.

Usage[edit]

<?php update_blog_status( $blog_id, $pref, $value, $refresh ) ?>

Parameters[edit]

$blog_id
(integer) (required) Blog ID
Default: None
$pref
(string) (required) A field name
Default: None
$value
(string) (required) Value for $pref
Default: None
$refresh
(boolean) (optional) Whether to refresh the blog details cache. Default is true. Deprecated in version 3.1.
Default: true

Return Values[edit]

(void) 
This function does not return a value.

  • Uses global: (object) $wpdb
