Description[edit]

Saves nav menu items

Parameters[edit]

$nav_menu_selected_id
(int or string) (required) id, slug, or name of the currently-selected menu.
Default: None
$nav_menu_selected_title
(string) (required) Title of the currently-selected menu.
Default: None

Return[edit]

(array) 
The menu updated message.

Usage[edit]

 <?php wp_nav_menu_update_menu_items( $nav_menu_selected_id, $nav_menu_selected_title ); ?> 

Example[edit]

wp_nav_menu_update_menu_items( $nav_menu_selected_id, $nav_menu_selected_title );

Change Log[edit]

  • Since: 3.6.0
