Description[edit]

This function is essentially the same as update_option() but works network wide when using WP Multisite.

Usage[edit]

<?php update_site_option( $option, $value ); ?>

Parameters[edit]

$option
(string) (required) Name of option. Expected to not be SQL-escaped.
Default: None
$value
(mixed) (required) Option value. Expected to not be SQL-escaped.
Default: None

Return Values[edit]

(boolean) 
False if value was not updated and true if value was updated.

Examples[edit]

Notes[edit]

  • Uses: apply_filters() - Calls 'pre_update_site_option_$option' hook to allow overwriting the option value to be stored.
  • Uses: do_action() - Calls 'update_site_option_$option' and 'update_site_option' hooks on success.

Change Log[edit]

  • 3.4.0 - update_site_option() was moved from wp-includes/functions.php to wp-includes/option.php.
  • Since: 2.8.0

Source File[edit]

update_site_option() is located in wp-includes/option.php.
