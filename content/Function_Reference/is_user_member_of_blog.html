Description[edit]

Find out whether a user is a member of a given blog.

Usage[edit]

<?php is_user_member_of_blog( $user_id, $blog_id ); ?>

Parameters[edit]

$user_id
(integer) (optional) User ID to check membership
Default: current user
$blog_id
(integer) (optional) Blog ID to check membership
Default: current blog

Return Value[edit]

(boolean) 
True if member, False if not

Examples[edit]

Check if the current user is a member of blog 5: 

<?php

$user_id = get_current_user_id();
$blog_id = 5;

if ( is_user_member_of_blog( $user_id, $blog_id ) ) {

   /*
    * The user is a member of the blog with ID 5.
    * We could do stuff here, like show them a link, etc.
    */
}

?>
