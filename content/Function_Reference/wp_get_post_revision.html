Description[edit]

Retrieve a post revision

Usage[edit]

<?php wp_get_post_revision( $post ); ?>

Parameters[edit]

$post
(int|object) (required) ID or Object of the revision we want to retrieve.
Default: None

Return Values[edit]

(array) 
A post object with the revision, or NULL on failure/if the revision doesnt exist.

Notes[edit]

  • Uses: get_post() To retrieve the post.

Change Log[edit]

Since: 2.6

Source File[edit]

wp_get_post_revision() is located in wp-includes/revision.php

Related[edit]

  • wp_is_post_revision() - Determines if the specified post is a revision.
  • wp_get_posts_revisions() - Returns all revisions of specified post.
