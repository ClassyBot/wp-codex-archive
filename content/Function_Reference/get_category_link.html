Description[edit]

Returns the correct url for a given Category ID.

In a Plugin or Theme, it can be used as early as the setup_theme Action. Any earlier usage, including plugins_loaded, generates a Fatal Error.

Usage[edit]

<?php get_category_link( $category_id ); ?> 

Parameters[edit]

$category_id
(integer) (required) The category ID whose link will be retrieved.
Default: None

Return Values[edit]

(URL) 
(string) URL with a link to the category.

Examples[edit]

Category Link[edit]

<?php
    // Get the ID of a given category
    $category_id = get_cat_ID( 'Category Name' );

    // Get the URL of this category
    $category_link = get_category_link( $category_id );
?>

<!-- Print a link to this category -->
<a href="<?php echo esc_url( $category_link ); ?>" title="Category Name">Category Name</a>

Notes[edit]

  • Uses: apply_filters() Calls 'category_link' filter on category link and category ID.
  • Uses global: (unknown) $wp_rewrite
  • Using a nonexistent Category ID will return a link to the parent directory. (eg: http://example.com/baseurl/archives/category/)

Change Log[edit]

Since: 1.0.0
