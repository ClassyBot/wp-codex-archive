Description[edit]

Adds a screen option to the current page.

Usage[edit]

<?php
  add_screen_option( 
      'per_page',
      array('label' => _x( 'Comments', 'comments per page (screen options)' )) );
?>

Parameters[edit]

$option
(string) (required) An option name.
Default: None
$args
(array) (optional) Option-dependent arguments.
Default: array()

Examples[edit]

Change default columns for network dashboard[edit]

<?php
add_action('wp_network_dashboard_setup', function() {
    add_screen_option('layout_columns', ['default' => 2]);
});
?>
