Description #Description

\n\t

When a theme adds ‘post-thumbnail’ support, a special ‘post-thumbnail’ image size is registered, which differs from the ‘thumbnail’ image size managed via the Settings > Media screen.

\n

When using the_post_thumbnail() or related functions, the ‘post-thumbnail’ image size is used by default, though a different size can be specified instead as needed.

\n\n\t\t

    Styling Post Thumbnails
    \nPost Thumbnails are given a class “wp-post-image”. They also get a class depending on the size of the thumbnail being displayed. You can style the output with these CSS selectors:

    \n
    \n\timg.wp-post-image\n\timg.attachment-thumbnail\n\timg.attachment-medium\n\timg.attachment-large\n\timg.attachment-full\n
    \n

    You can also give Post Thumbnails their own class:

    \n
    \n// Give the Post Thumbnail a class "alignleft".\necho get_the_post_thumbnail( $post_id, 'thumbnail', array( 'class' => 'alignleft' ) );\n

    Thumbnail Sizes
    \nThe default image sizes of WordPress are “thumbnail”, “medium”, “large” and “full” (the size of the image you uploaded).
    \nThese image sizes can be configured in the WordPress Administration Media panel under Settings > Media.
    \nThemes may also add “post-thumbnail”. This is how you can use these default sizes with get_the_post_thumbnail():

    \n
    \r\n// without parameter -> Post Thumbnail (as set by theme using set_post_thumbnail_size())\r\nget_the_post_thumbnail( $post_id );                   \r\n\r\nget_the_post_thumbnail( $post_id, 'thumbnail' );      // Thumbnail (Note: different to Post Thumbnail)\r\nget_the_post_thumbnail( $post_id, 'medium' );         // Medium resolution\r\nget_the_post_thumbnail( $post_id, 'large' );          // Large resolution\r\nget_the_post_thumbnail( $post_id, 'full' );           // Original resolution\r\n\r\nget_the_post_thumbnail( $post_id, array( 100, 100) ); // Other resolutions\r\n
    \n

    Register new image sizes for Post Thumbnails with: add_image_size().
    \nTo set the default size for Post Thumbnails see: set_post_thumbnail_size().

    Remove width and height attributes from the returning html

    \n

    get_the_post_thumbnail function produces html with img tag which includes width, height, src, class, alt and title attributes. The following code snippet is useful if someone wish to remove width and height attribute.

    \n

    This function provides 'post_thumbnail_html' filter before returning the html.

    \n
    \r\nadd_filter( 'post_thumbnail_html', 'remove_thumbnail_width_height', 10, 5 );\r\n\r\nfunction remove_thumbnail_width_height( $html, $post_id, $post_thumbnail_id, $size, $attr ) {\r\n    $html = preg_replace( '/(width|height)=\\"\\d*\\"\\s/', "", $html );\r\n    return $html;\r\n}\r\n

    Post thumbnails in a listing context are often linked to the content they’re representing. Instead of adding that every time, here is a filter you can add that will do it automatically based on an `$attr` passed in.

    \n
    \n
    \n
    \r\n/**\r\n * Link thumbnails to their posts based on attr\r\n *\r\n * @param $html\r\n * @param int $pid\r\n * @param int $post_thumbnail_id\r\n * @param int $size\r\n * @param array $attr\r\n *\r\n * @return string\r\n */\r\n\r\nfunction wpdev_filter_post_thumbnail_html( $html, $pid, $post_thumbnail_id, $size, $attr ) {\r\n\r\n\t if ( ! empty( $attr[ 'link_thumbnail' ] ) ) {\r\n\r\n\t\t$html = sprintf(\r\n\t\t\t'<a href="%s" title="%s" rel="nofollow">%s</a>',\r\n\t\t\tget_permalink( $pid ),\r\n\t\t\tesc_attr( get_the_title( $pid ) ),\r\n\t\t\t$html\r\n\t\t);\r\n\t }\r\n\r\n\treturn $html;\r\n}\r\n\r\nadd_filter( 'post_thumbnail_html', 'wpdev_filter_post_thumbnail_html', 10, 5 );\r\n
