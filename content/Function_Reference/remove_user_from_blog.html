Description[edit]

Removes a user from a blog.

Accepts an optional $reassign parameter, if you want to reassign the user's blog posts to another user upon removal.

Usage[edit]

<?php remove_user_from_blog( $user_id, $blog_id, $reassign ); ?>

Parameters[edit]

$user_id
(integer) (required) ID of user to remove
Default: None
$blog_id
(integer) (required) ID of blog to remove user from
Default: None
$reassign
(string) (optional) a user to whom to reassign the removed user's posts
Default: empty string

Return Value[edit]

(WP_Error object|boolean) 
WP_Error object on failure, True on success

Examples[edit]

Remove user 243 from blog 3 and reassign their posts to user 2. 

<?php

$user_id = 243;
$blog_id = 3;
$reassign = 2;

remove_user_from_blog( $user_id, $blog_id, $reassign );

?>

Notes[edit]

Use the 'remove_user_from_blog' action to fire an event when users are removed from a blog.
