Descrizione[edit]

Riceve variabili dalla classe WP_Query.

Utilizzo[edit]

<?php get_query_var( $var ) ?>

Parametri[edit]

$var
(stringa) (required) La chiave della variabile da ricevere.
Default: None

Valori di Ritorno[edit]

(mixed) 

Esempi[edit]

Note[edit]

  • Vedi WP_Query::get()
  • Usa global su: (object) $wp_query

Change Log[edit]

Since: 1.5.0

Sorgente[edit]

get_query_var() è situata in wp-includes/query.php.
