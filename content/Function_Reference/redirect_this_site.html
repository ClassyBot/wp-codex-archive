Description[edit]

Ensure that the current site's domain is listed in the allowed redirect host list.

Usage[edit]

<?php redirect_this_site( $deprecated ); ?>

Parameters[edit]

$deprecated
(unknown_type) (optional) no description
Default: empty string

Return Values[edit]

(array) 
The current site's domain

Source File[edit]

redirect_this_site() is located in wp-includes/ms-functions.php
