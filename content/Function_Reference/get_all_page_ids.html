Description[edit]

Get a list of page IDs.

Usage[edit]

<?php get_all_page_ids() ?>

Parameters[edit]

  • None

Return Values[edit]

(array) 
List of page IDs.

Examples[edit]

<?php
$page_ids=get_all_page_ids();
echo '<h3>My Page List :</h3>';
foreach($page_ids as $page)
{
	echo '<br />'.get_the_title($page);
}

 ?>


Notes[edit]

  • Uses: $wpdb

Change Log[edit]

Since: 2.0.0
