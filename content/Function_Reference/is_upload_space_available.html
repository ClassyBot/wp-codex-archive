Description[edit]

Determines if there is any upload space left in the current blog's quota.

Usage[edit]

<?php is_upload_space_available() ?>

Parameters[edit]

This function does not accept any parameters

Return Value[edit]

(boolean) 
True if space is available, false otherwise.
