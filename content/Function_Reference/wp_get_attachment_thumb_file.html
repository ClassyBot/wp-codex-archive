Description[edit]

Retrieve thumbnail for an attachment.

Usage[edit]

<?php wp_get_attachment_thumb_file( $attachment_id ); ?>

Parameters[edit]

$attachment_id
(integer) (optional) Attachment ID.
Default: 0

Return Values[edit]

(string|boolean) 
False on failure. Thumbnail file path on success.
