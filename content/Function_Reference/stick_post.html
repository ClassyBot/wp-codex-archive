Description[edit]

Sticks a post by adding the post id into sticky_posts option in the options table.


Usage[edit]

 <?php stick_post($post_id); ?> 

Parameter[edit]

$post_id
(integer) (required) Id if the post you want to stick
Default: None

Return Value[edit]

Function does not return any value

Example[edit]


$post_id = 1;
stick_post($post_id);
