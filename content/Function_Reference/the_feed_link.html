Description[edit]

Display the permalink for the feed type.

Usage[edit]

<?php the_feed_link( $anchor, $feed ); ?>

Parameters[edit]

$anchor
(string) (required) The link's anchor text.
Default: None
$feed
(string) (optional) Feed type
Default:

Source file[edit]

the_feed_link() is located in wp-includes/link-template.php.
