Description[edit]

Add magic quotes to $_GET, $_POST, $_COOKIE, and $_SERVER.

Also forces $_REQUEST to be $_GET + $_POST. If $_SERVER, $_COOKIE, or $_ENV are needed, use those superglobals directly.

Usage[edit]

<?php wp_magic_quotes(  ); ?>
