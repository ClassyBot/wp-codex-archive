Description[edit]

Returns the post's parent's post_ID.

Usage[edit]

 <?php wp_get_post_parent_id( $post_ID ); ?> 

Parameters[edit]

$post_id
(integer) (required) post ID.
Default: None

Return Value[edit]

(mixed)
Post's parent ID on success. False on error.

Change Log[edit]

Since: 3.1.0

Source File[edit]

wp_get_post_parent_id() is located in wp-includes/post.php.
