Description[edit]

Fixes javascript bugs in browsers.

Converts unicode characters to HTML numbered entities.

Usage[edit]

<?php funky_javascript_fix( $text ) ?>

Parameters[edit]

$text
(string) (required) Text to be made safe.
Default: None

Return Values[edit]

(string) 
Fixed text.
