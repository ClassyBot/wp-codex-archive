Description[edit]

Add a user to the Super admin user list in WordPress Multisite

Usage[edit]

 <?php grant_super_admin( $user_id ); ?> 

Parameters[edit]

$user_id
(integer) (required) The user ID to grant super admin privileges
Default: None

Returns[edit]

(BOOLEAN) 
True if successful, false otherwise

Changelog[edit]

Since: Wordpress 3.0.0

Source Code[edit]

grant_super_admin() is located in wp-admin/includes/ms.php.
