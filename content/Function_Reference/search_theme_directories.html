Description[edit]

Search all registered theme directories for complete and valid themes.

Usage[edit]

<?php search_theme_directories() ?>

Parameters[edit]

None.

Return Values[edit]

(array) 
Valid themes found

Examples[edit]

Notes[edit]

  • Uses global: (unknown type) $wp_theme_directories
  • Uses global: (array) $wp_broken_themes Holds broken themes list.

Change Log[edit]

Since: 2.9.0

Source File[edit]

search_theme_directories() is located in wp-includes/theme.php.
