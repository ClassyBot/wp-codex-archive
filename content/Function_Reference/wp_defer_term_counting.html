Description[edit]

Enable or disable term counting.

Usage[edit]

<?php wp_defer_term_counting( $defer ); ?>

Parameters[edit]

$defer
(boolean) (optional) Optional. Enable if true, disable if false.
Default:

Return Values[edit]

(bool) 
Whether term counting is enabled or disabled.
