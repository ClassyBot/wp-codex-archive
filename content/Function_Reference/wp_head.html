Description[edit]

Fire the 'wp_head' action. Put this template tag immediately before </head> tag in a theme template (ex. header.php, index.php).

Usage[edit]

 <?php wp_head(); ?> 

Parameters[edit]

This function does not accept any parameters.

Return values[edit]

None.

Examples[edit]

In twentyten theme[edit]

wp-content/themes/twentyten/header.php: 

<?php 
 ...
    /* Always have wp_head() just before the closing </head>
     * tag of your theme, or you will break many plugins, which
     * generally use this hook to add elements to <head> such
     * as styles, scripts, and meta tags.
     */
    wp_head();
 ?>
 </head>

Notes[edit]

  • Uses do_action() Calls 'wp_head' hook.

Changelog[edit]

  • Since: 1.2.0

Source File[edit]

wp_head() is located in wp-includes/general-template.php.
