Description[edit]

Adds nonce HTML fields to the signup page.

Usage[edit]

<?php signup_nonce_fields(); ?>

Parameters[edit]

This function does not accept any parameters

Return Value[edit]

This function does not return a value. It echoes out the necessary HTML.
