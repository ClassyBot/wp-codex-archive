Description[edit]

Unregisters a custom navigation menu for a theme location.

Usage[edit]

 <?php unregister_nav_menu( $location ); ?> 

Parameters[edit]

$location
(string) (required) Menu location identifier slug.
Default: None

Return Values[edit]

bool True on success, false on failure.

Example[edit]

 <?php unregister_nav_menu( 'primary' ); ?> 

Changelog[edit]

  • Since: 3.1

Source File[edit]

  • unregister_nav_menu() is located in wp-includes/nav-menu.php.
