Description[edit]

Load the RDF RSS 1.0 Feed template.

Usage[edit]

<?php do_feed_rss() ?>

Parameters[edit]

None.

Return Values[edit]

(void) 
This function does not return a value.

Notes[edit]

  • Uses: load_template() to load feed template.

Since: 2.1.0
