Description[edit]

Displays the date of the post in YYYY-MM-DD format (ex: 2004-09-24). This tag must be used within The Loop.

Usage[edit]

 <?php the_date_xml(); ?> 

Parameters[edit]

This tag does not accept any parameters.

Example[edit]

<p>Date posted: <?php the_date_xml(); ?></p>
Date posted: 2005-05-14
