Description[edit]

Returns null.

Useful for returning null to filters easily.

Usage[edit]

<?php 
// This will add a filter on `example_filter` that returns null
add_filter( 'example_filter', '__return_null' ); 
?>

Return Values[edit]

(null) 
null

Change Log[edit]

Since: 3.4
