Description[edit]

Get a full blog URL, given a domain and a path.

Usage[edit]

<?php get_blogaddress_by_domain( $domain, $path ) ?>

Parameters[edit]

$domain
(string) (required) The domain
Default: None
$path
(string) (required) The path
Default: None

Return Values[edit]

(string) 
The full blog URL
