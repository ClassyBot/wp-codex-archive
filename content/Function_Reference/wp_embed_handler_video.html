Description[edit]

Video embed handler callback. (This function creates filter wp_embed_handler_video.)

Usage[edit]

 <?php wp_embed_handler_video( $matches, $attr, $url, $rawattr ) ?> 

Parameters[edit]

$matches
(array) (required) The regex matches from the provided regex when calling wp_embed_register_handler().
Default: None
$attr
(array) (required) Embed attributes.
Default: None
$url
(string) (required) The original URL that was matched by the regex.
Default: None
$rawattr
(array) (required) The original unmodified attributes.
Default: None

Return[edit]

(string) 
The embed HTML.
