Description[edit]

Retrieves the HTML link of the url of the author of the current comment.

Usage[edit]

 <?php $author = get_comment_author_url_link( $linktext, $before, $after ); ?> 

Parameters[edit]

$linktext
(string) (optional) Text to display instead of the comment author's URL link.
Default: empty
$before
(string) (optional) Text or HTML to display before the URL link.
Default: empty
$after
(string) (optional) Text or HTML to display after the URL link.
Default: empty

Return Values[edit]

(string) 
The HTML link between the $before and $after parameters.

Examples[edit]

Notes[edit]

  • Uses: apply_filters() - Calls 'get_comment_author_url_link' hook on the comment author's URL link.
  • Uses: get_comment_author_url() - Retrieves the comment author's URL without link.
