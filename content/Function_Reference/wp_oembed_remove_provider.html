Description[edit]

Removes an oEmbed provider.

Usage[edit]

 <?php wp_oembed_remove_provider( $format ) ?> 

Parameters[edit]

$format
(string) (required) The URL format of the oEmbed provider you want to remove.
Default: None

Return Values[edit]

(boolean) 
True if provider was removed, False otherwise.

Examples[edit]

To remove a provider, you will need to know the format used in wp-includes/class-oembed.php.

The following example removes the WordPress TV provider: 

<?php wp_oembed_remove_provider( 'http://wordpress.tv/*' ); ?>
