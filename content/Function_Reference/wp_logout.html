Description[edit]

Log the current user out, by destroying the current user session.

Usage[edit]

 <?php wp_logout(); ?> 

Parameters[edit]

This function does not accept any parameters.

Return values[edit]

None.

Example[edit]

<?php wp_logout(); ?>

Change Log[edit]

Since: 2.5.0

Source File[edit]

wp_logout() is located in wp-includes/pluggable.php.
