Description[edit]

Returns an HTML string that makes a tag cloud.

Usage[edit]

<?php wp_generate_tag_cloud( $tags, $args ); ?>

Default Usage[edit]

 <?php $args = array(
    'smallest'  => 8,
    'largest'   => 22,
    'unit'      => 'pt',
    'number'    => 0,
    'format'    => 'flat',
    'separator' => '\n',
    'orderby'   => 'name',
    'order'     => 'ASC',
    'topic_count_text_callback'  => 'default_topic_count_text',
    'topic_count_scale_callback' => 'default_topic_count_scale',
    'filter'    => 1 ); ?>
