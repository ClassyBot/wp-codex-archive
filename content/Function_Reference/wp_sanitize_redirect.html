Description[edit]

Sanitizes a URL for use in a redirect. Removes invalid URL characters in the URL parameter.

Parameters[edit]

$location
(string) (required) URL of the link to redirect to.
Default: None

Return Values[edit]

(string) 
redirect-sanitized URL

Change Log[edit]

Since: 2.3
