Description[edit]

Displays the returned translated text from translate().

Usage[edit]

<?php _e( $text, $domain ) ?>

Parameters[edit]

$text
(string) (required) Text to translate
Default: None
$domain
(string) (optional) Domain to retrieve the translated text
Default: 'default'

Return Values[edit]

(void) 
This function does not return a value.

Examples[edit]

Display some translated text: 

<?php _e( 'Some text to translate and display.', 'textdomain' ); ?>

Notes[edit]

  • Echos returned translate() string.
  • l10n is an abbreviation for localization.

Change Log[edit]

  • Since: 1.2.0

Source File[edit]

_e() is located in wp-includes/l10n.php.
