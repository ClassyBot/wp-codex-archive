Description[edit]

Displays a link that lets users post a comment in reply to a specific comment.

If JavaScript is enabled and the comment-reply.js JavaScript is loaded the link moves the comment form to just below the comment.

Usage[edit]

 <?php comment_reply_link( $args, $comment, $post ); ?> 

Parameters[edit]

$args
(array) (optional) Associative array of options (see below)
Default: array
$comment
(integer) (optional) ID of comment being replied to.
Default: null
$post
(integer) (optional) ID of post that comment is going to be displayed on.
Default: null

Arguments[edit]

All of the following argument can be overridden using the $args parameter.

add_below 
(string) are for the JavaScript addComment.moveForm() method parameters.
Defaults to 'comment'. Concatenated as '{$add_below}-{$id}' i.e. the comment container's id should be 'comment-23'.
respond_id 
(string) are for the JavaScript addComment.moveForm() method parameters. Defaults to 'respond'
reply_text 
(string) only used if they can post a reply. Defaults to 'Reply'.
login_text 
(string) only used if the user must log in or register first before posting a comment. Defaults to 'Log in to leave a comment'.
depth 
(integer)
before 
("string") the html or text to add before the reply link
after 
("string") the html or text to add after the reply link
