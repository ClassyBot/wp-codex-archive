Description[edit]

Check a MIME-Type against a list.

If the wildcard_mime_types parameter is a string, it must be comma separated list. If the real_mime_types is a string, it is also comma separated to create the list.

Usage[edit]

<?php wp_match_mime_types( $wildcard_mime_types, $real_mime_types ); ?>

Parameters[edit]

$wildcard_mime_types
(string|array) (required) e.g. audio/mpeg or image (same as image/*) or flash (same as *flash*).
Default: None
$real_mime_types
(string|array) (required) post_mime_type values
Default: None

Return Values[edit]

(array) 
Description.
