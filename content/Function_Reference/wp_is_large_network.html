Description[edit]

Whether or not we have a large Multisite network. The default criteria for a large network is either more than 10,000 users or more than 10,000 sites.

Parameters[edit]

$using
(string) (required) Sites or Users.
Default: sites

Returns[edit]

(BOOLEAN) 
True if the network meets the criteria for large. False otherwise.

Change Log[edit]

Since: 3.3.0
