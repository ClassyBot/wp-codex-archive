Description[edit]

Switch the interal blog id used to create keys in blog specific groups.

Usage[edit]

 <?php wp_cache_switch_to_blog( $blog_id ) ?> 

Parameters[edit]

$blog_id
(integer) (required) Blog ID.
Default: None
