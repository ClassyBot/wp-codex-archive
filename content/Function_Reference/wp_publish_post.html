Description[edit]

Publish a post by transitioning the post status.

Note: This function does not do anything except transition the post status. If you want to ensure post_name is set, use wp_update_post() instead.

Usage[edit]

<?php wp_publish_post( $post_id ) ?>

Parameters[edit]

$post_id
(integer) (required) Post ID.
Default: None

Return Values[edit]

(null) 

Examples[edit]

Notes[edit]

  • Uses: $wpdb
  • Uses: do_action() to initiate the following action hooks, passing $post->ID and $post (post data) parameters:
    • 'edit_post'
    • 'save_post'
    • 'wp_insert_post'

Change Log[edit]

Since: 2.1.0
