Description[edit]

Display the post content for feeds.

Usage[edit]

<?php the_content_feed( $feed_type ); ?>

Parameters[edit]

$feed_type
(string) (optional) The type of feed. Can be atom, rss, rss2 or rdf.
Default: null

Since: 2.9
