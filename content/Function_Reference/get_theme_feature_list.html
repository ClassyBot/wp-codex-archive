Description[edit]

Retrieve list of WordPress theme features (aka, theme tags).

Usage[edit]

<?php get_theme_feature_list() ?>

Parameters[edit]

None.

Return Values[edit]

(array) 
Array of features keyed by category, with translations keyed by slug.
