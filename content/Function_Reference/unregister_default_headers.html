Description[edit]

Unregister default headers.

This function must be called after register_default_headers() has already added the header you want to remove.

Usage[edit]

<?php unregister_default_headers( $headers ); ?>

Parameters[edit]

$header
(string|array) (required) The header string id (key of array) to remove, or an array thereof.
Default: None

Return Values[edit]

(boolean) 
Whether the headers were removed.

Change Log[edit]

Since: 3.0.0

Source File[edit]

unregister_default_headers() is located in wp-includes/theme.php.
