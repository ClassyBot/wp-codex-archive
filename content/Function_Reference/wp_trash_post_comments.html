Description[edit]

Moves comments for a post to the trash

Usage[edit]

 <?php wp_trash_post_comments( $post_id ); ?>

Parameters[edit]

$post_id
(integer) (optional) Post ID.
Default: 0

Return Values[edit]

(mixed) 
False on failure.

Notes[edit]

  • Uses: do_action() on 'trash_post_comments' before trashing.
  • Uses: do_action() on 'trashed_post_comments' after trashing.
  • Uses: clean_comment_cache()

Change Log[edit]

Since: 2.9.0

Source File[edit]

wp_trash_post_comments() is located in wp-includes/post.php.
