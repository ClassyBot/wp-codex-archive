Description[edit]

Unregister a previously registered embed handler.

Usage[edit]

<?php wp_embed_unregister_handler( $id, $priority ) ?>

Parameters[edit]

$id
(string) (required) The handler ID that should be removed.
Default: None
$priority
(integer) (optional) The priority of the handler to be removed
Default: 10

Return Values[edit]

This function does not return a value.

Examples[edit]

For a handler registered with the default priority: 

wp_embed_unregister_handler( 'forbes' );

With a priority other than the default: 

wp_embed_unregister_handler( 'forbes', 5 );

Notes[edit]

  • Uses: WP_Embed::unregister_handler()
