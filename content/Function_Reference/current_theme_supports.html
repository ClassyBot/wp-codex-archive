Description[edit]

Checks a theme's support for a given theme feature.

Usage[edit]

<?php current_theme_supports( $feature ); ?>

Parameters[edit]

$feature
(string) (required) the feature being checked
Default: None

Features list:

  • 'post-thumbnails'
  • 'post-formats'
  • 'custom-header'
  • 'custom-background'
  • 'menus'
  • 'automatic-feed-links'
  • 'editor-style'
  • 'widgets'
  • 'html5'
  • 'title-tag'

Return Values[edit]

(boolean) 

Examples[edit]

<?php 
if (current_theme_supports('custom-header')) {
  // do something special when custom-header is supported...
}
?>

Notes[edit]

  • Uses global: (unknown type) $_wp_theme_features

Change Log[edit]

  • Since: 2.9.0

Source File[edit]

current_theme_supports() is located in wp-includes/theme.php.
