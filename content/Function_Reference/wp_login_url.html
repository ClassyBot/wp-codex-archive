Description[edit]

This Template Tag returns the URL that allows the user to log in to the site.

Usage[edit]

 <?php echo wp_login_url( $redirect ); ?> 

Parameters[edit]

$redirect
(string) (optional) URL to redirect to on login.
Default: None

Examples[edit]

Default Usage[edit]

<a href="<?php echo wp_login_url(); ?>" title="Login">Login</a>

Login and Redirect to Current Page[edit]

<a href="<?php echo wp_login_url( get_permalink() ); ?>" title="Login">Login</a>

Login and Redirect to Homepage[edit]

<a href="<?php echo wp_login_url( home_url() ); ?>" title="Login">Login</a>

Notes[edit]

  • Uses: apply_filters() Calls login_url hook on HTML link content.
  • $redirect argument must be absolute, such as http://example.com/mypage/. For best results, use site_url( '/mypage/ ' ).

Change Log[edit]
