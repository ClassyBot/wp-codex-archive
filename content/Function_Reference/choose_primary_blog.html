Description[edit]

Displays a form table with fields letting the user choose the primary blog.

Usage[edit]

<?php choose_primary_blog(); ?>

Parameters[edit]

None.

Returns[edit]

This function does not return a value.

Example[edit]

<?php choose_primary_blog(); ?>

Source File[edit]

choose_primary_blog() is located in wp-admin/includes/ms.php.
