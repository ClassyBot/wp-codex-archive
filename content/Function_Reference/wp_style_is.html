Description[edit]

Determine if a style has been registered, queued, printed, or is waiting to be printed.

Usage[edit]

<?php wp_style_is( $handle, $list = 'enqueued' ); ?>

Parameters[edit]

$handle
(string) (required) Name of the style. Lowercase string.
Default: None
$list
(string) (optional) The list to query.
  • registered - style was registered through wp_register_style()
  • enqueued / queue - style was enqueued
  • done - style has been printed
  • to_do - style has not yet been printed
Default: enqueued

Return Values[edit]

(bool) 
True or false.

Change Log[edit]

Since: 2.8

Source File[edit]

wp_style_is() is located in wp-includes/functions.wp-styles.php.
