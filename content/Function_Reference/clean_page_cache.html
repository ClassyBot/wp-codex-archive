Description[edit]

Will clean the page in the cache.

Clean (delete) page from cache that matches $id. Will also clean cache associated with 'all_page_ids' and 'get_pages'.

Usage[edit]

<?php clean_page_cache( $id ) ?>

Parameters[edit]

$id
(integer) (required) Page ID to clean
Default: None

Return Values[edit]

(void) 
This function does not return a value.

Examples[edit]

Notes[edit]

  • Uses: do_action() Calls 'clean_page_cache' on page ID.
  • Uses: clean_post_cache() on page ID.
