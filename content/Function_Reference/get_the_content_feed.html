Description[edit]

Retrieve the post content for feeds.

Usage[edit]

<?php get_the_content_feed( $feed_type ); ?>

Parameters[edit]

$feed_type
(string) (optional) The type of feed. Can be: rss, rss2, atom or rdf.
Default: null

Return Values[edit]

(string) 
The filtered content.

Change Log[edit]

Since: 2.9
