Description[edit]

Create a new import tool

Usage[edit]

<?php register_importer( $id, $name, $description, $callback ) ?>

Parameters[edit]

$id
(string) (required) Importer tag. Used to uniquely identify importer
Default: None
$name
(string) (required) Importer name and title
Default: None
$description
(string) (required) Importer description
Default: None
$callback
(callback) (required) Callback to run
Default: None

Return Values[edit]

WP_Error 
Returns WP_Error when $callback is WP_Error

Source File[edit]

register_importer() is located in wp-admin/includes/import.php.
