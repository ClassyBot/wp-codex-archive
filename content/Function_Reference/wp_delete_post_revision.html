Description[edit]

Deletes a revision.

Deletes the row from the posts table corresponding to the specified revision.

Usage[edit]

<?php wp_delete_post_revision( $revision_id ); ?>

Parameters[edit]

$revision_id
(int|object) (required) Revision ID or revision object.
Default: None

Return Values[edit]

(mixed) 
Null or WP_Error if error, deleted post if success.

Change Log[edit]

Since: 2.6
