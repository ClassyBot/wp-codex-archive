Description[edit]

Save the properties of a menu item or create a new one.

Usage[edit]

<?php $menu_id = wp_update_nav_menu_item($menu_id, $menu_item_db_id, $menu_item_data) ?>

Parameters[edit]

$menu_id
(string) (required) The ID the menu.
Default: None
$menu_item_db_id
(int) (optional) The ID of the menu item, or 0 to create a new menu item.
Default: 0
$menu_item_data
(array) (optional) The menu item's data.
Default: array()

Return Values[edit]

(int|WP_Error) 
The menu item's ID or WP Error Object in case of failure.

Source File[edit]

wp_update_nav_menu_item() is located in wp-includes/nav-menu.php.
