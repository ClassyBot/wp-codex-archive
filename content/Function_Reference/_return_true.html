Description[edit]

Returns true.

Useful for returning true to filters easily.

Usage[edit]

<?php 
// This will add a filter on `example_filter` that returns true
add_filter( 'example_filter', '__return_true' ); 
?>

Return Values[edit]

(boolean) 
true

Change Log[edit]

Since: 3.0

Source File[edit]

__return_true() is located in wp-includes/functions.php
