Description[edit]

Fetch a Role definition.

Usage[edit]

 <?php get_role( $role ); ?> 

Parameters[edit]

role
(string) (required) role name
Default: None

Returns[edit]

(mixed) 
Returns a WP_Role object on success, null on failure.
