Description[edit]

Converts a value to a non-negative integer.

Synonym of abs( intval( $foo ) ).

Usage[edit]

<?php absint( $maybeint ); ?>

Parameters[edit]

$maybeint
(mixed) (required) Data you wish to have converted to a non-negative integer.
Default: None

Return Values[edit]

(int) 
A non-negative integer.

Change Log[edit]

Since: 2.5
