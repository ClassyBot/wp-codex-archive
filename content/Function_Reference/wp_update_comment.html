Description[edit]

Updates an existing comment in the database.

Filters the comment and makes sure certain fields are valid before updating.

Usage[edit]

<?php wp_update_comment( $comment ) ?>

Parameters[edit]

$comment
(array) (required) Contains information on the comment. See get_comment() for a list of valid attributes.
Default: None

Return Values[edit]

(integer) 
Comment was updated if value is 1, or was not updated if value is 0.

Examples[edit]

Unapproving a comment[edit]

<?php
$comment = array();
$comment['comment_ID'] = 123;
$comment['comment_approved'] = 0;
wp_update_comment( $comment );
?>

Notes[edit]

  • Uses: wp_transition_comment_status() Passes new and old comment status along with $comment object
  • Uses: get_comment()
  • Uses: wp_filter_comment()
  • Uses: get_gmt_from_date()
  • Uses: clean_comment_cache()
  • Uses: wp_update_comment_count()
  • Uses: do_action() on 'edit_comment' on $comment_ID.
  • Uses global: (object) $wpdb
