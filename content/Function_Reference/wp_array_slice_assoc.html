Description[edit]

Extract a slice of an array, given a list of keys

Usage[edit]

<?php wp_array_slice_assoc( $array, $keys ) ?>

Parameters[edit]

$array
(array) (required) The original array
Default: None
$keys
(array) (required) The list of keys
Default: None

Return Values[edit]

(array) 
The array slice
