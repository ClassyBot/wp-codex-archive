Description[edit]

The status of a comment by ID.

Usage[edit]

<?php wp_get_comment_status( $comment_id ) ?>

Parameters[edit]

$comment_id
(integer) (required) Comment ID
Default: None

Return Values[edit]

(string|boolean) 
Status might be 'deleted', 'approved', 'unapproved', 'spam' or false on failure.

Examples[edit]

<?php
$status = wp_get_comment_status( $comment_id );
if ( $status == "approved" ) {
  // show the comment
}
?>

Notes[edit]

  • Uses: get_comment()
