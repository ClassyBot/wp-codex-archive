Description[edit]

Do trackbacks for a list of URLs.

Usage[edit]

<?php trackback_url_list( $tb_list, $post_id ) ?>

Parameters[edit]

$tb_list
(string) (required) Comma separated list of URLs
Default: None
$post_id
(integer) (required) Post ID
Default: None

Return Values[edit]

(void) 
This function does not return a value.

Examples[edit]

Notes[edit]

  • Uses: wp_get_single_post() on $post_id.
  • Uses: trackback() on each trackback url.

Change Log[edit]

Since: 1.0.0

Source File[edit]

trackback_url_list() is located in wp-includes/post.php.
