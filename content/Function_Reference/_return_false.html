Description[edit]

Returns false.

Useful for returning false to filters easily.

Usage[edit]

<?php 
// This will add a filter on `example_filter` that returns false 
add_filter('example_filter', '__return_false'); 
?>

Return Values[edit]

(boolean) 
false

Change Log[edit]

Since: 3.0

Source File[edit]

__return_false() is located in wp-includes/functions.php
