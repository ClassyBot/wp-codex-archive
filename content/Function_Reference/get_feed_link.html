Description[edit]

Retrieve the permalink for a feed by feed type.

Usage[edit]

<?php get_feed_link( $feed ); ?>

Parameters[edit]

$feed
(string) (optional) Feed type. Defaults to the default feed if empty.
Default: Empty string

Return Values[edit]

(string) 
The link for the feed.

Examples[edit]

Notes[edit]

  • Uses: global $wp_rewrite to get the feed permastructure.
  • Uses: get_default_feed() to get the default feed if $feed is empty.
  • Uses: apply_filters() - calls 'feed_link' filter to filter the output.

Change Log[edit]

  • Since: 1.5.0

Source file[edit]

get_feed_link() is located in wp-includes/link-template.php.
