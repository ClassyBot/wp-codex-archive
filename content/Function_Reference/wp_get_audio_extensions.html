Description[edit]

Return a filtered list of WP-supported audio formats. (This function is called by wp_audio_shortcode() and wp_maybe_load_embeds(), and creates filter wp_audio_extensions.)

Default WordPress supported audio formats are: mp3, ogg, wma, m4a, wav.

Usage[edit]

 <?php wp_get_audio_extensions() ?> 

Parameters[edit]

This tag has no parameters.

Return[edit]

(array) 
Description.

Example[edit]

Print an array of default WordPress-supported audio extensions: 

$audio_extensions = wp_get_audio_extensions();

Displays as: 

Array
(
    [0] => mp3
    [1] => ogg
    [2] => wma
    [3] => m4a
    [4] => wav
)
