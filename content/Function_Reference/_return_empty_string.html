Description[edit]

Returns an empty string.

Useful for returning an empty string to filters easily.

Usage[edit]

<?php 
// This will add a filter on `example_filter` that returns an empty string
add_filter( 'example_filter', '__return_empty_string' ); 
?>

Return Values[edit]

(string) 

Change Log[edit]

Since: 3.7
