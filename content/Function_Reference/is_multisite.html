Description[edit]

Determine whether Multisite support is enabled.

Usage[edit]

 <?php if ( is_multisite() ) { echo 'Multisite is enabled'; } ?> 

Parameters[edit]

This function does not accept any parameters.

Returns[edit]

(BOOLEAN) 
True if Multisite is enabled, false otherwise

Changelog[edit]

Since: WordPress 3.0.0

Source Code[edit]

is_multisite() is located in wp-includes/load.php.
