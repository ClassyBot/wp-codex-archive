Description[edit]

Sets blog defaults.

This function creates a row in the wp_blogs table.

Usage[edit]

<?php install_blog_defaults($blog_id, $user_id) ?>

Parameters[edit]

$blog_id
(integer) (optional) value is ignored
Default: None
$user_id
(integer) (required) User of the new blog
Default: None

Return Value[edit]

This function does not return a value
