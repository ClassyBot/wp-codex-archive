Description[edit]

Whether to force SSL on content.

Usage[edit]

<?php force_ssl_content( $force ); ?>

Parameters[edit]

$force
(string|bool) (optional)
Default: empty string

Return Values[edit]

(bool) 
True if forced, false if not forced.

Change Log[edit]

Since: 2.8.5

Source File[edit]

force_ssl_content() is located in wp-includes/ms-functions.php
