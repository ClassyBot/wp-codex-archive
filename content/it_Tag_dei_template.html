I tag dei template si usano nei template del proprio sito e forniscono gli strumenti per renderlo unico e interessante dando la possibilità di mostrarvi informazioni dinamicamente o personalizzarlo in altro modo. Di seguito viene presentata una lista di tag disponibili in WordPress, raggruppati in categorie a seconda della loro funzione specifica.

Per maggiori informazioni sui tag dei template e sui template in generale, si leggano i seguenti articoli:

  • Stepping Into Template Tags - un'introduzione ai tag dei template.
  • Anatomy of a Template Tag - dettagli su come inserire i tag nei file template.
  • Come passare i parametri - dettagli su come usare i parametri con i tag.
  • Include Tags - tag utili per includere un file template in un altro.
  • Conditional Tags - tag non mostrati qui, utili per rendere i propri template più flessibili con l'aggiunta di opzioni.
  • Riferimento funzioni - tag relativi alle funzionalità interne di WordPress.
  • Deprecated Functions - tag deprecati.
  • Templates - lista integrale di risorse relative a template, temi e tag.
  • Stepping Into Templates - introduzione ai file template.

Nota: Se si pensa di avere qualcosa di interessante da dire sull'implementazione di un tag dei template, lo si aggiunga alla sua documentazione in modo che sia di beneficio agli altri utenti di WordPress. Tuttavia, per prima cosa si leggano Contributing to WordPress e Codex:Contributing per l'aiuto e guidelines su come modificare le pagine del Codex.

File[edit]

I file dei tag dei template si trovano nella cartella wp-includes. I file hanno suffisso "-template.php" per distinguerli dagli altri file di WordPress. Ci sono 9 file dei tag dei template:
