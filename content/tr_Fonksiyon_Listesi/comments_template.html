Açıklama[edit]

Yorum şablonunu temaya dahil eder. Yazı veSayfalar da görüntülenir.

Kullanım[edit]

<?php comments_template( $file, $separate_comments ); ?>

Parameterler[edit]

$file
(string) (optional) Bulunur:
Default: /comments.php
$separate_comments
(boolean) (optional) Yorumları türüne göre ayırır.
Default: false

Örnekler[edit]

Normal kullanım 

<?php comments_template(); ?>

Alternatif Yorum Şablonu[edit]

Bazı durumlarda farklı bir yorum şablonu göstermek isteyebilirsiniz.Bu durumda şablonunuzun adı farkli-comments.php olabilir. Temaya şu şekilde include edilir: 

<?php comments_template( '/farkli-comments.php' ); ?>

Notlar[edit]

  • Uses global: (array) $comment List of comment objects for the current post
  • Uses: $wpdb
  • Uses: $id
  • Uses: $post
  • Uses: $withcomments Will not try to get the comments if the post has none.
