Açıklama[edit]

Bu Koşul Etiketi ana sayfanın görüntülenip, görüntülenmediğini kontrol eder. Bu bir mantıksal fonksiyondur, yani TRUE veya FALSE döndürür.

Not: Wordpress 2.1 bu fonksiyonu önceki versiyonlardan farklı yorumlar - Bkz. static Front Page. Eğer ana sayfa olarak statik bir sayfa (bkz. is_front_page()) tanımlamışsanız, bu etiket sizin "yazılar sayfanız" (yazıları gösterdiğiniz sayfa) için uygulanacaktır.

Kullanım[edit]

<?php is_home(); ?>

Parametreler[edit]

Bu etiket hiçbir parametre almaz.

Dönen Değerler[edit]

(boolean) 
Ana sayfa gösteriliyorsa True, değilse false.

Örnekler[edit]

Aşağıdaki ana sayfanız görüntülenirken yan menünüzde farklı bir içerik göstermek için kullanılabilir.
<?php
if ( is_home() ) {
    // Bu ana sayfa
} else {
    // Bu ana sayfa değil
}
?>

Notlar[edit]

  • Ayrıca Bakınız: is_front_page()

Değişim Kaydı[edit]

Başlangıç: 1.5.0

Kaynak Dosya[edit]

is_home() fonksiyonu wp-includes/query.php dosyasında yer alır.
