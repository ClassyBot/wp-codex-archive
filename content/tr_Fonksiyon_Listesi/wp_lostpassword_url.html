Açıklama[edit]

Bu tema etiketi, parolasını unutan kullanıcıları yeni parola isteme sayfasına yönlendirir.

Kullanım[edit]

 <?php echo wp_lostpassword_url( $redirect ); ?> 

Parametreler[edit]

$redirect
(string) (İsteğe bağlı.) Kayıp şifre aldıktan sonra bağlantıya yönlendirme.
Default: None

Örnekler[edit]

Normal Kullanım[edit]

<a href="<?php echo wp_lostpassword_url(); ?>" title="Şifremi unuttum">Şifremi unuttum</a>

Yeni şifre istedikten sonra geçerli sayfaya yönlendir[edit]

<a href="<?php echo wp_lostpassword_url( get_permalink() ); ?>" title="Şifremi unuttum">Şifremi Unuttum</a>


Yeni şifre istedikten sonra anasayfaya yönlendir[edit]

<a href="<?php echo wp_lostpassword_url( get_bloginfo('url') ); ?>" title="Şifremi Unuttum">Şifremi Unuttum</a>

Notlar[edit]

  • Kullanıldığı yerler: apply_filters() 'wp_lostpassword_url' i HTML link içeriği bağlamak.

Kaynak Dosya[edit]

wp_lostpassword_url() wp-includes/general-template.php dosyasında yer alır.

Benzerler[edit]

Login Tags: is_user_logged_in(), wp_login_form(), wp_loginout(), wp_logout(), wp_register()
Login URLs: wp_login_url(), wp_logout_url(), wp_lostpassword_url(), wp_registration_url()

See also index of Function Reference and index of Template Tags.
