תאור[edit]

תג מותנה זה בודק אם העמוד הראשי מוצג. זהו תג בוליאני, משמע הוא מחזיר אמת או שקר.

אופן שימוש[edit]

<?php is_home(); ?>

פרמטרים[edit]

תג זה לא מקבל פרמטרים.

ערך חוזר[edit]

(בוליאני) 
אמת בעמוד ראשי, שקר אחרת.

דוגמאות[edit]

ניתן להשתמש בדוגמא הבאה בתפריט הצדדי לצורך הצגת תכנים שונים כאשר הגולש נמצא בעמוד הבית או בעמוד שאינו עמוד הבית. 

<?php
if ( is_home() ) {
	// This is a homepage
} else {
	// This is not a homepage
}
?>

הערות[edit]

  • ראה גם: is_front_page()

תיעוד[edit]

מאז גרסה: 1.5.0

קובץ מקור[edit]

is_home() נמצא ב wp-includes/query.php.
