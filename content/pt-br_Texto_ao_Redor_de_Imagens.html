Há momentos em que você quer usar imagens em seu post ou página jutamente com seu texto, mas na maioria das vezes você quer que sua imagem fique de um ladou ou outro com o texto contornando a imagem.

Quando você insere uma imagem pela janela Inserir mídia, você tem a opção de definir a posição da imagem, então o WordPress vai adicionar o código HTML para a imagem juntamente com a classe CSS para a posição que escolheu.

Consulte Inserindo Imagens em Posts e Páginas para saber mais.

A imagem não flutua[edit]

Quando isso não acontece é porque seu tema provavelmente não possui algumas das classes CSS necessárias para que as imagens em seu site flutuem conforme você definir.

Abra o arquivo style.css do seu tema em um editor de texto e procure por todas as tags img. Verifique se as intruções a seguir existem, se nao existirem, adicione no arquivo: 

img.alignright {float:right; margin:0 0 1em 1em}
img.alignleft {float:left; margin:0 1em 1em 0}
img.aligncenter {display: block; margin-left: auto; margin-right: auto}
a img.alignright {float:right; margin:0 0 1em 1em}
a img.alignleft {float:left; margin:0 1em 1em 0}
a img.aligncenter {display: block; margin-left: auto; margin-right: auto}

Uma vez que tenha estas instruções em seu arquivo de estilos, basta marcar o alinhamento desejado na hora que inserir uma imagem no post ou página na janela Inserir mídia.
