A maioria dos servidores e instalações pessoais devem ser capaz de hospedar o WordPress desde que o básico requerido esteja disponível.

Exigido para o Servidor[edit]

  • PHP versão 5.2 ou mais recente
  • MySQL versão 5.0 ou mais recente
  • (Opcional) Apache módulo mod_rewrite (para habilitar o uso de Links Permanentes em várias configurações)
  • Pelo menso 64 MB de memória para o PHP.

Instalação Recomendada para o Servidor Web[edit]

Recomendamos Linux com Apache ou servidores mais robustos como NGINX, mas qualquer servidor com suporte a PHP e MySQL é suficiente. Se o seu servidor não tem suporte para estas plataformas, além de mod_rewrite, o melhor é procurar por um serviço com estas funcionalidades.

Também é indispensável que o servidor permita conexões remotas, para que muitas das funções do WordPress funcionem. Se o servidor bloquear conexões externas HTTP, muito da capacidade do WordPress será prejudicada.
