概述[edit]

条件判断函数用来检测、判断使用。 如在blog posts index page大量使用。 is_home属于Boolean函数, 返回值为：TRUE 或 FALSE.

注意:从WordPress 2.1版本开始修改了实现：相对低版本修改为直接判断方式 - 请参考static Front Page. 如果你的网站选择采用静态页面作为首页(参考is_front_page()), is_home同样也会应用到你的"日志页面".

用法[edit]

<?php is_home(); ?>

传入参数[edit]

is_home不需要传入任何参数.

返回值[edit]

(boolean) 
真则返回True, 假则返回false.

使用样例[edit]

下面这个例子你可添加到sidebar文件里面，如果在首页或者其他页面，则可输出不同的结果。
<?php
if ( is_home() ) {
    // 这是首页
} else {
    // 这不是首页
}
?>

注意事项[edit]

is_home 使用了全局对象$wp_query WP_Query object. is_home 在Actionparse_query无法使用.

  • 请参考: is_front_page()

变更记录[edit]

起源: 1.5.0版本

源码文件[edit]

is_home() 函数在源码文件wp-includes/query.php中.
