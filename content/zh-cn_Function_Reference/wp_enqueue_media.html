描述 Description[edit]

嵌入所有的脚本(scripts)，样式表(styles)，设置(settings)和使用所有关于附件(media) JavaScript API 所必须的模板(templates)。

该函数应该在 'admin_enqueue_scripts' 动作(action)中或其后调用。

使用 Usage[edit]

<?php wp_enqueue_media( $args ); ?>

参数 Parameters[edit]

$args
(array) (optional) An array of arguments. Currently only one argument is used, 'post', which if passed will be used to set up the scripts for that post. Note: if you do not set this parameter, uploaded files will not be attached to a post.

一个参数数组。但目前该数组中只有一个参数在使用，'post'，如果该参数被传递了，会被用于设定该 post 的脚本。记住：如果你不设定该参数，上传好的文件将不会被附到该 post 中。

Default: array()

返回值 Returns Values[edit]

该函数不返回值。

示例 Examples[edit]

默认用法： 

wp_enqueue_media();

对某个特殊的文章插入 media scripts 

<?php

$args = array( 'post' => 34 );

wp_enqueue_media( $args );

?>

笔记 Notes[edit]

  • 用法： do_action() - 在嵌入该 media 后调用 'wp_enqueue_media'
  • 用法: did_action( 'wp_enqueue_media' ) - 确保你希望的函数只被调用一次。
  • 用法: apply_filters():
    • 调用 'media_upload_tabs' 时，传入一个旧的 thickbox media tabs 的数组以确保插件正确运作。默认的 tabs 将会在之后重设。
    • 调用 'disable_captions' 以检查是否 captions 应该被禁用。
    • 调用 'media_view_settings' 时，传入一个关于设置和 post 对象的数组
    • 调用 'media_view_strings' 时，传入一个字符串组，这些字符串用于 media viewer 和 post 对象。
  • 用法： wp.media (JavaScript) 会调用 media uploader
    • 文档 ： http://www.grafikart.fr/tutoriels/wordpress/wp-media-uploader-403 (French video)
