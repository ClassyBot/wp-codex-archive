概述[edit]

按照顺序坚持内容块是否存在如下HTML标签：<{media type}>, <object>, <embed>, or <iframe>。 如未找到任何HTML标签，则检查日志的第一行查找URL。

用法[edit]

 <?php get_embedded_media( $type, &$content, $remove, $limit )  ?> 

传入参数[edit]

$type
(string) (required) Type of media: audio or video.
Default: None
$content
(string) (required) A string which might contain media data.
Default: None
$remove
(boolean) (optional) Whether to remove the found URL from the passed content.
Default: false
$limit
(integer) (optional) The number of galleries to return.
Default: 0

返回值[edit]

(array) 
A list of found HTML media embeds and possibly a URL by itself.
