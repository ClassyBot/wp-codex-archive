Descrizione[edit]

Verifica se il chiamante si trova nel loop (ossia se il loop è attivo al momento).

Utilizzo[edit]

<?php in_the_loop(); ?>

Parametri[edit]

Nessuno.

Valori restituiti[edit]

(boolean) 
True se il chiamante è all'interno del loop, false se il loop non è iniziato oppure è terminato.

Log delle modifiche[edit]

Da: 2.0.0

File sorgente[edit]

in_the_loop() si trova in wp-includes/query.php.

Si veda anche l'indice di Riferimento funzioni e l'indice di Tag dei template.
