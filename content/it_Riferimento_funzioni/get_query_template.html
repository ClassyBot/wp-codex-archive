Descrizione[edit]

Recupera il percorso del file senza l'uso dell'estensione.

Utilizzata per recuperare velocemente il percorso del file senza includerne l'estensione. Controlla anche il template genitore, se il file esiste, con l'uso di locate_template(). Permette posizioni più generiche di file senza l'uso delle altre funzioni get_*_template().

Può essere utilizzata con include() o require() per il recupero del percorso. 

   if ( '' != get_query_template( '404' ) )
       include( get_query_template( '404' ) );

oppure il medesimo risultato si può ottenere con 

   if ( '' != get_404_template() )
       include( get_404_template() );

Utilizzo[edit]

<?php get_query_template( $type, $templates ); ?>

Parametri[edit]

$type
(string) (richiesto) Nome del file senza estensione.
Default: Nessuno
$templates
(array) (opzionale) Una lista opzionale di template candidati
Default: array()

Valori restituiti[edit]

(string) 
Percorso completo del file.

Esempi[edit]

Note[edit]

  • Utilizza: apply_filters() su {$type}_template sul risultato di locate_template().
  • Utilizza: locate_template() su "{$type}.php".

Log delle modifiche[edit]

Da: 1.5.0

File sorgente[edit]

get_query_template() si trova in wp-includes/template.php.
