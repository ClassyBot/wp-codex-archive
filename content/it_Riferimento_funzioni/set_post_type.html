Descrizione[edit]

Aggiorna il tipo del post di cui viene passato l'ID.

La cache della pagina o dell'articolo viene ripulita.

Utilizzo[edit]

<?php set_post_type( $post_id, $post_type ); ?>

Parametri[edit]

$post_id
(integer) (richiesto) ID del post di cui modificare il tipo.
Default: 0
$post_type
(string) (opzionale) Nuovo tipo del post. Tra i valori supportati ci sono 'post' e 'page'.
Default: post

Valori restituiti[edit]

(integer) 
Numero di righe modificate. 1 in caso di successo, 0 in caso di fallimento.

Esempi[edit]

<?php
$post_id = 15;
if ( set_post_type( $post_id, 'page'  ) ) {
  echo "Il post #$post_id ora è una pagina";
} else {
  echo "Impossibile trasformare questo post in una pagina";
}
?>
