Descrizione[edit]

Fa scattare l'azione 'wp_head'. Questa funzione va collocata subito prima del tag di chiusura </head> nel template di un tema (es. header.php, index.php).

Utilizzo[edit]

 <?php wp_head(); ?> 

Parametri[edit]

La funzione non accetta parametri.

Valori restituiti[edit]

Nessuno.

Esempi[edit]

Nel tema TwentyTen[edit]

wp-content/themes/twentyten/header.php: 

<?php 
 ...
    /* Inserire sempre wp_head() appena prima del tag di chiusura </head>
     * del proprio tema, in caso contrario molti plugin che generalmente
     * utilizzano questo aggancio per aggiungere elementi in <head>
     * come stili, script e meta tag, non funzioneranno.
     */
    wp_head();
 ?>
 </head>

Note[edit]

  • Utilizza do_action() per richiamare l'aggancio 'wp_head'.

Log delle modifiche[edit]

  • Da: 1.2.0

File sorgente[edit]

wp_head() si trova in wp-includes/general-template.php.
