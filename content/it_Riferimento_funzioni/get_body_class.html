Descrizione[edit]

Recupera le classi per l'elemento body sotto forma di array.

Utilizzo[edit]

<?php get_body_class($class); ?>

Parametri[edit]

$class
(string o array) (opzionale) Una stringa o un array di classi da aggiungere al valore restituito.
Default: false

Valori restituiti[edit]

(array) 
Un array con nomi di classi.

Log delle modifiche[edit]

  • Da: 2.8.0
  • 3.1.0 Aggiunto il supporto per le classi di stile basate su ID

File sorgente[edit]

get_body_class() si trova in wp-includes/post-template.php.

Si veda anche l'indice di Riferimento funzioni e l'indice di Tag dei template.
