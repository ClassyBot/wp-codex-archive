Descrizione[edit]

Restituisce null.

Utile per restituire null ai filtri in modo facile.

Utilizzo[edit]

<?php 
// Quanto segue aggiunge un filtro su `esempio_filtro` che restituisce null
add_filter('esempio_filtro', '__return_null'); 
?>

Valori restituiti[edit]

(null) 
null

Log delle modifiche[edit]

Da: 3.4

File sorgente[edit]

__return_null() si trova in wp-includes/functions.php
