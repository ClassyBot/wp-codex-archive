Descrizione[edit]

Rimuove i metadati di un utente corrispondenti a determinati criteri.

I criteri possono riguardare solo la chiave oppure sia la chiave che il valore. La rimozione basata su chiave e valore previene la rimozione di metadati duplicati con la medesima chiave. È anche possibile rimuovere tutti i metadati che hanno una determinata chiave, se necessario.

Utilizzo[edit]

<?php delete_user_meta( $user_id, $meta_key, $meta_value ) ?>

Parametri[edit]

$user_id
(integer) (richiesto) ID dell'utente.
Default: Nessuno
$meta_key
(string) (richiesto) Nome del metadato.
Default: Nessuno
$meta_value
(mixed) (opzionale) Valore del metadato.
Default: ''

Valori restituiti[edit]

(boolean) 
False in caso di fallimento, true in caso di successo.

Esempi[edit]

<?php
if ( ! delete_user_meta($user_id, '_province') ) {
  echo "Ooops! Errore durante la cancellazione di questa informazione!";
}
?>

Note[edit]

  • Utilizza: delete_metadata()
