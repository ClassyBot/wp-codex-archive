Descrizione[edit]

Controlla se l'utente corrente ha una capacità o un ruolo per un dato blog.

Questa funzione è simile a current_user_can(), ma permette di verificare se l'utente corrente ha una capacità per un altro blog sulla rete. Per verificare se l'utente ha la capacità sul blog corrente, si può usare current_user_can().

Utilizzo[edit]

<?php current_user_can_for_blog($blog_id, $capability); ?>

Parametri[edit]

$blog_id
(integer) (richiesto) ID del blog.
Default: Nessuno
$capability
(string) (richiesto) Nome della capacità o del ruolo.
Default: Nessuno

Valori restituiti[edit]

(boolean) 
True in caso affermativo, false altrimenti.

Log delle modifiche[edit]

Da: 3.0

File sorgente[edit]

current_user_can_for_blog() si trova in wp-includes/capabilities.php

Correlati[edit]

User Capabilities: user_can(), current_user_can(), current_user_can_for_blog()

Si veda anche l'indice di Riferimento funzioni e l'indice di Tag dei template.
