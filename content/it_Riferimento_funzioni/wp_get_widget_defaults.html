Descrizione[edit]

Recupera la lista delle sidebar predefinite registrate.

Parametri[edit]

Questa funzione non accetta parametri.

Valori restituiti[edit]

(array) 
Un array di array vuoti indicizzati per ID della sidebar.

Log delle modifiche[edit]

Da: 2.2

File sorgente[edit]

wp_get_widget_defaults() si trova in wp-includes/widgets.php
