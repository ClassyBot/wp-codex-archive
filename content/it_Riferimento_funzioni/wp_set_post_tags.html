Descrizione[edit]

Imposta i tag per un post. Ogni tag che ancora non esiste viene automaticamente creato. Utilizza wp_set_post_terms().

Utilizzo[edit]

<?php wp_set_post_tags( $post_ID, $tags, $append ) ?>

Parametri[edit]

$post_ID
(integer) (richiesto) ID del post.
Default: 0
$tags
(string,array) (opzionale) Lista di tag. Può essere un array o una stringa separata da virgole.
Default: array
$append
(boolean) (opzionale) Se true, i tag verranno aggiunti al post. Se false, i tag rimpiazzeranno quelli esistenti.
Default: false

Valori restituiti[edit]

(boolean|null) 
restituisce false se $post_id non è un integer oppure se è 0. Restituisce null altrimenti.

Esempi[edit]

Aggiungere i tag significato e vita ai tag correnti del post con ID 42: 

wp_set_post_tags( 42, 'significato,vita', true );


Note[edit]

Log delle modifiche[edit]

Da: 2.3.0

File sorgente[edit]

wp_set_post_tags() si trova in wp-includes/post.php.

Correlati[edit]

wp_insert_term wp_delete_term wp_set_post_terms
