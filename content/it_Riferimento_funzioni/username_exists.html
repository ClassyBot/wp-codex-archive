Descrizione[edit]

Restituisce l'ID dell'utente se questo esiste oppure null se non esiste.

Utilizzo[edit]

<?php username_exists( $username ); ?> 

Parametri[edit]

$username
(string) (richiesto) Una stringa che rappresenta lo username di cui verificare l'esistenza.
Default: Nessuno

Valori restituiti[edit]

(mixed) 
Questa funzione restituisce l'ID dell'utente se questo esiste oppure null se non esiste.

Esempi[edit]

Utilizzare username_exists() nei propri script per decidere se il dato username esiste. 

<?php  
       $username = $_POST['username'];
       if ( username_exists( $username ) )
           echo "Username un uso!";
       else
           echo "Username non in uso!";
?>
