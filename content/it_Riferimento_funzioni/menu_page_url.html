Descrizione[edit]

Ottiene l'URL per accedere ad una particolare pagina del menu basandosi sullo slug con cui è stata registrata.

Se lo slug non è stato registrato in modo appropriato, non verrà restituito alcun URL.

Utilizzo[edit]

<?php menu_page_url( $menu_slug, $echo ); ?>

Parametri[edit]

$menu_slug
(string) (richiesto) Lo slug con cui riferirsi a questo menu (deve essere univoco per questo menu)
Default: Nessuno
$echo
(boolean) (opzionale) Indica se stampare o meno l'URL.
Default: true

Valori restituiti[edit]

(string) 
L'URL

Log delle modifiche[edit]

Da: 3.0

File sorgente[edit]

menu_page_url() si trova in wp-admin/includes/plugin.php
