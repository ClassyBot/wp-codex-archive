Descrizione[edit]

Implementa la funzionalità dello shortcode Video per aggiungere (e riprodurre) file video in un post. A seconda delle condizioni, aggiunge il tag HTML "video" (con il media player, ridimesionato a $content_width definito nel tema attivo) oppure altro HTML utile allo scopo.

I formati video supportati da WordPress sono: mp4, m4v, webm, ogv, wmv, flv.

Utilizzo[edit]

 <?php wp_video_shortcode( $attr ) ?> 

Parametri[edit]

$attr
(array) (richiesto) Attributi dello shortcode.
Default: Nessuno

Valori restituiti[edit]

(string) 
Contenuto HTML per aggiungere il video.

Log delle modifiche[edit]

  • Da: 3.6.0

File sorgente[edit]

wp_video_shortcode() si trova in wp-includes/media.php.
