Descrizione[edit]

Recupera la miniatura di un allegato.

Utilizzo[edit]

<?php wp_get_attachment_thumb_file( $post_id ); ?>

Parametri[edit]

$post_id
(integer) (opzionale) ID dell'allegato.
Default: 0

Valori restituiti[edit]

(mixed) 
False in caso di fallimento, percorso del file della miniatura in caso di successo.
