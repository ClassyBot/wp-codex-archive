Descrizione[edit]

Aggiunge metadati al record di un utente.

Utilizzo[edit]

<?php add_user_meta( $user_id, $meta_key, $meta_value, $unique ); ?>

Parametri[edit]

$user_id
(integer) (richiesto) ID dell'utente
Default: Nessuno
$meta_key
(string) (richiesto) Nome del metadato.
Default: Nessuno
$meta_value
(mixed) (richiesto) Valore del metadato.
Default: Nessuno
$unique
(boolean) (opzionale) Indica se una chiave già esistente non deve essere aggiunta.
Default: false

Valori restituiti[edit]

(integer/boolean) 
Chiave primaria in caso di successo, false in caso di fallimento.

Esempi[edit]

<?php
$user_id = 1;
$awesome_level = 1000;
add_user_meta( $user_id, '_level_of_awesomeness', $awesome_level);
?>

Note[edit]

  • Utilizza: add_metadata()

Log delle modifiche[edit]

Da: 3.0.0

File sorgente[edit]
