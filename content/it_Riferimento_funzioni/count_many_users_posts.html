Descrizione[edit]

Restituisce il conteggio degli articoli data una lista di ID utente. Questa è un'operazione O(n), quindi è preferibile a count_user_posts() ogni volta che più di un singolo conteggio è necessario.

Utilizzo[edit]

 <?php count_many_users_posts( $users, $post_type, $public_only ); ?> 

Parametri[edit]

$users
(array) (richiesto) Lista di ID utente di cui recuperare il conteggio degli articoli.
Default: Nessuno
$post_type
(string) (opzionale) Nome del tipo di post.
Default: null
$public_only
(string) (opzionale) Ignora il controllo delle capacità dell'utente corrente (False) oppure restituisce solamente gli articoli che l'utente corrente può vedere (True).
Default: False

Valori restituiti[edit]

(array) 
Quantità di articoli che ogni utente ha scritto.

Esempi[edit]

Utilizzo standard[edit]

La chiamata a count_many_users_posts restituisce il numero di articoli creati da ogni utente. 

<?php
$users = array(1, 3, 9, 10);
$counts = count_many_users_posts($users);
echo 'Articoli creati dall'utente 3: ' . $counts[3];
?>
Articoli creati dall'utente 3: 143

Log delle modifiche[edit]

  • 3.5.0: Aggiunto il parametro public_only
  • Da: 3.0.0

File sorgente[edit]

count_many_users_posts() si trova in wp-includes/user.php.
