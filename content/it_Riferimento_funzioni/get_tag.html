Descrizione[edit]

Recupera i tag di un post attraverso l'ID del tag o un oggetto tag.

Se al parametro $tag si passa un oggetto, che si assume essere la riga recuperata dal database, i dati del tag verranno memorizzati nella cache.

Se al parametro $tag si passa l'ID del tag, allora questo verrà recuperato dal database, se non è già presente nella cache.

Se si guarda la funzione get_term(), si vedrà che ad entrambi i tipi verranno applicati parecchi filtri e infine verranno ripuliti basandosi sul valore del parametro $filter.

Utilizzo[edit]

<?php &get_tag( $tag, $output, $filter ) ?>

Parametri[edit]

$tag
(integer|object) (richiesto)
Default: Nessuno
$output
(string) (opzionale) Costante OBJECT, ARRAY_A, o ARRAY_N
Default: OBJECT
$filter
(string) (opzionale) Il valore predefinito è raw o nessun filtro definito di WordPress verrà applicato.
Default: 'raw'

Valori restituiti[edit]

(object|array) 
Valore restituito sulla base del valore di $output.
