Descrizione[edit]

Rimuove un provider oEmbed.

Utilizzo[edit]

 <?php wp_oembed_remove_provider( $format ) ?> 

Parametri[edit]

$format
(string) (richiesto) Il formato dell'URL del provider oEmbed che si vuole rimuovere.
Default: Nessuno

Valori restituiti[edit]

(boolean) 
True se il provider è stato rimosso, False altrimenti.

Esempi[edit]

Per rimuovere un provider, è necessario conoscerne il formato usato in wp-includes/class-oembed.php.

L'esempio seguente rimuove il provider di WordPress TV: 

<?php wp_oembed_remove_provider( 'http://wordpress.tv/*' ); ?>

Log delle modifiche[edit]

  • Da: 3.5.0

File sorgente[edit]

wp_oembed_remove_provider si trova in wp-includes/media.php
