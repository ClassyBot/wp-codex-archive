Descrizione[edit]

Determina se l'utente è un (super) amministratore della rete. Controlla se l'utente è amministratore anche se la modalità rete è disabilitata.

Utilizzo[edit]

 <?php is_super_admin( $user_id ); ?> 

Parametri[edit]

$user_id
(integer) (opzionale) L'ID di un utente.
Default: Se non indicato viene considerato l'utente corrente.

Valori restituiti[edit]

(boolean) 
True se l'utente è (super) amministratore della rete. Restituisce true anche se la modalità rete è disabilitata e l'utente è un amministratore.

Esempi[edit]

<?php
//Rimuovere il menu "Modifica" per gli utenti che non sono super amministratori di una rete multisito
if ( !is_super_admin() ) {
	add_action( 'admin_init', 'my_remove_edit_menu' );
}
function my_remove_edit_menu() {
	remove_menu_page('edit.php');
}
?>
