Descrizione[edit]

Recupera il nome di una categoria basandosi sul suo ID.

Utilizzo[edit]

<?php get_the_category_by_ID( $cat_ID ); ?>

Parametri[edit]

$cat_ID
(int) (richiesto) ID della categoria.
Default: Nessuno

Valori restituiti[edit]

(string) 
Nome della categoria.

Log delle modifiche[edit]

Da: 0.71

File sorgente[edit]

get_the_category_by_ID() si trova in wp-includes/category-template.php
