Descrizione[edit]

Recupera la lista di tag abbinati ad un post.

Utilizzo[edit]

<?php wp_get_post_tags( $post_id, $args ) ?>

Parametri[edit]

$post_id
(integer) (opzionale) ID del post.
Default: 0
$args
(array) (opzionale) Sovrascrive i valori predefiniti. Si veda wp_get_object_terms() per una lista di opzioni valide che possono essere passate.
Default: array

Valori restituiti[edit]

(array) 
Lista dei tag del post.

Esempi[edit]

Per un post con tag tag2, tag5 e tag6, il codice 

$t = wp_get_post_tags($post->ID);
print_r($t);

stampa 

Array
(
   [0] => stdClass Object
       (
           [term_id] => 4
           [name] => tag2
           [slug] => tag2
           [term_group] => 0
           [term_taxonomy_id] => 4
           [taxonomy] => post_tag
           [description] => 
           [parent] => 0
           [count] => 7
       )

   [1] => stdClass Object
       (
           [term_id] => 7
           [name] => tag5
           [slug] => tag5
           [term_group] => 0
           [term_taxonomy_id] => 7
           [taxonomy] => post_tag
           [description] => 
           [parent] => 0
           [count] => 6
       )

   [2] => stdClass Object
       (
           [term_id] => 16
           [name] => tag6
           [slug] => tag6
           [term_group] => 0
           [term_taxonomy_id] => 16
           [taxonomy] => post_tag
           [description] => 
           [parent] => 0
           [count] => 2
       )

)


Per ottenere la lista degli ID dei tag di un particolare post: 

global $post;
$tag_ids = wp_get_post_tags( $post->ID, array( 'fields' => 'ids' ) );

e assumendo lo stesso insieme di dati del primo esempio, $tag_ids conterrebbe 

[4, 7, 16]

Note[edit]

  • Utilizza: wp_get_post_terms()

Log delle modifiche[edit]

Da: 2.3.0

File sorgente[edit]

wp_get_post_tags() si trova in wp-includes/post.php.

Correlati[edit]

Si veda anche l'indice di Riferimento funzioni e l'indice di Tag dei template.
