Descrizione[edit]

Aggiunge metadati ad un commento.

Utilizzo[edit]

<?php add_comment_meta( $comment_id, $meta_key, $meta_value, $unique ); ?>

Parametri[edit]

$comment_id
(integer) (richiesto) ID del commento.
Default: Nessuno
$meta_key
(string) (richiesto) Nome del metadato.
Default: Nessuno
$meta_value
(mixed) (richiesto) Valore del metadato.
Default: Nessuno
$unique
(boolean) (opzionale) Indica se le chiavi devono essere univoche.
Default: false

Valori restituiti[edit]

(boolean) 
False in caso di fallimento, true in caso di successo.

Esempi[edit]

Aggiungere un valore personalizzato ad ogni nuovo commento[edit]

<?php
function add_custom_comment_field( $comment_id ) {

   add_comment_meta( $comment_id, 'my_custom_comment_field', $_POST['my_custom_comment_field'] );
}
add_action( 'comment_post', 'add_custom_comment_field' );
?>
