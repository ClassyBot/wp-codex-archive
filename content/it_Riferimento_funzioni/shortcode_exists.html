Descrizione[edit]

Controlla se uno specifico shortcode esiste.

Utilizzo[edit]

 <?php if ( shortcode_exists( 'gallery' ) ) { } ?> 

Parametri[edit]

$tag
(string) (richiesto) Lo shortcode da controllare.
Default: Nessuno

Valori restituiti[edit]

(boolean) 
True se lo shortcode è stato registrato, false altrimenti.

Esempio[edit]

<?php

if ( shortcode_exists( 'gallery' ) ) {
    // Lo shortcode [gallery] esiste.
}

?>

Log delle modifiche[edit]

  • Da: 3.6.0

File sorgente[edit]

shortcode_exists() si trova in wp-includes/shortcodes.php.
