Descrizione[edit]

Restituisce il link alla pagina precedente di un elenco di post.

Utilizzo[edit]

 <?php get_previous_posts_link( $label ); ?> 

Parametri[edit]

$label
(string) (opzionale) Testo del link alla pagina precedente.
Default: '« Pagina precedente'

Valori restituiti[edit]

(string) 
Link alla pagina precedente.

Esempi[edit]

Utilizzo standard[edit]

<?php $prev = get_previous_posts_link();
echo $prev;
?>

Log delle modifiche[edit]

Da: 2.7

File sorgente[edit]

get_previous_posts_link() si trova in wp-includes/link-template.php.
