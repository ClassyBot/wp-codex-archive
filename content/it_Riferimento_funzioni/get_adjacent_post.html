Descrizione[edit]

Recupera il post adiacente a quello corrente. Può essere il post precedente o quello successivo.

Utilizzo[edit]

<?php get_adjacent_post( $in_same_cat, $excluded_categories, $previous ) ?>

Parametri[edit]

$in_same_cat
(boolean) (opzionale) Indica se il post deve essere nella stessa categoria di quello corrente.
Default: false
$excluded_categories
(string) (opzionale) ID delle categorie da escludere.
Default: ''
$previous
(boolean) (opzionale) Indica se recuperare il post precedente.
Default: true

Valori restituiti[edit]

  • Oggetto Post in caso di successo.
  • Null se la variabile globale $post non è impostata.
  • Stringa vuota se non esiste un post corrispondente.

Esempi[edit]

Note[edit]

  • Utilizza la variabile globale: (object) $post
  • Utilizza la variabile globale: (object) $wpdb

Filtri[edit]

$adjacent può essere 'previous' (precedente) oppure 'next' (successivo).

  • "get_{$adjacent}_post_join":
    $join, $in_same_cat, $excluded_categories
  • "get_{$adjacent}_post_where":
    $wpdb->prepare("WHERE p.post_date $op %s AND p.post_type = %s AND p.post_status = 'publish' $posts_in_ex_cats_sql", $current_post_date, $post->post_type), $in_same_cat, $excluded_categories
  • "get_{$adjacent}_post_sort":
    "ORDER BY p.post_date $order LIMIT 1"
