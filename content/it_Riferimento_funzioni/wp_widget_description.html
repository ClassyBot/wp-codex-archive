Descrizione[edit]

Recupera la descrizione di un widget.

Quando si registrano i widget, tra le opzioni è possibile anche includere 'description' da mostrare sul pannello di amministrazione del widget o nel tema.

Parametri[edit]

$id
(integer/string) (richiesto) ID del widget.
Default: Nessuno

Valori restituiti[edit]

(string) 
Descrizione del widget, se disponibile. Null in caso di fallimento.

Log delle modifiche[edit]

Da: 2.5.0

File sorgente[edit]

wp_widget_description() si trova in wp-includes/widgets.php

Correlati[edit]

Widgets: is_active_widget(), the_widget(), register_widget(), unregister_widget(), wp_register_widget_control(), wp_unregister_widget_control(), wp_convert_widget_settings(), wp_get_widget_defaults(), wp_widget_description()
