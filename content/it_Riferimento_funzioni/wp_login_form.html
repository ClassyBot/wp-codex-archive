Descrizione[edit]

Fornisce un semplice modulo di autenticazione da utilizzare ovunque in WordPress. Per impostazione predefinita, stampa il codice HTML immediatamente. Passare array('echo' => false) per restituire la stringa invece di stamparla.

Utilizzo[edit]

 <?php wp_login_form( $args ); ?> 

Utilizzo standard[edit]

 <?php $args = array(
        'echo'           => true,
        'redirect'       => site_url( $_SERVER['REQUEST_URI'] ), 
        'form_id'        => 'loginform',
        'label_username' => __( 'Username' ),
        'label_password' => __( 'Password' ),
        'label_remember' => __( 'Remember Me' ),
        'label_log_in'   => __( 'Log In' ),
        'id_username'    => 'user_login',
        'id_password'    => 'user_pass',
        'id_remember'    => 'rememberme',
        'id_submit'      => 'wp-submit',
        'remember'       => true,
        'value_username' => NULL,
        'value_remember' => false
); ?> 

Parametri[edit]

echo
(boolean) (opzionale) Stampa i risultati. Le opzioni sono:
  • 1 (true)
  • 0 (false)
Default: true
redirect
(string) (opzionale) URL a cui redirigere l'utente. Deve essere assoluto (come in http://example.com/mypage/). Raccomandato: site_url( '/mypage/ ' ).
Default: la pagina corrente
form_id
(string) (opzionale) form_id.
Default: loginform
remember
(boolean) (opzionale) Indica se memorizzare i valori.
  • 1 (true)
  • 0 (false)
Default: true
value_username
(string) (opzionale) value_username.
Default: NULL
value_remember
(string) (opzionale)
  • 1 (true)
  • 0 (false)
Default: false

Esempi[edit]

Utilizzo standard[edit]

Questa semplice funzione stampa un modulo di autenticazione: 

<?php wp_login_form(); ?>

Log delle modifiche[edit]

  • Da: 3.0.0

File sorgente[edit]

wp_login_form() si trova in wp-includes/general-template.php.

Correlati[edit]

Login Tags: is_user_logged_in(), wp_login_form(), wp_loginout(), wp_logout(), wp_register()
Login URLs: wp_login_url(), wp_logout_url(), wp_lostpassword_url(), wp_registration_url()

Si veda anche l'indice di Riferimento funzioni e l'indice di Tag dei template.
