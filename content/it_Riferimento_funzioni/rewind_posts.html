Descrizione[edit]

Riavvolge il ciclo (loop) degli articoli.

Utilizzo[edit]

<?php rewind_posts(); ?>

Parametri[edit]

Questa funzione non accetta alcun parametro.

Valori restituiti[edit]

Questa funzione non restituisce alcun valore.

Esempi[edit]

// ciclo principale
<?php if (have_posts()) : while (have_posts()) : the_post(); ?>
<?php the_content(); ?>
<?php endwhile; endif; ?>

// rewind
<?php rewind_posts(); ?>

// nuovo ciclo
<?php while (have_posts()) : the_post(); ?>
<?php the_content(); ?>
<?php endwhile; ?>

File sorgente[edit]
