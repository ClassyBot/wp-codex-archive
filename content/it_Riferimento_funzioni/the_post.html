Descrizione[edit]

Itera l'indice dell'articolo nel loop. Recupera l'articolo successivo, inizializza l'articolo ed imposta la proprietà 'in the loop' (nel ciclo) a true.

Utilizzo[edit]

<?php the_post(); ?>

Parametri[edit]

Questa funzione non accetta alcun paramentro.

Valori restituiti[edit]

Questa funzione non restituisce alcun valore.

Esempi[edit]

<?php
while ( have_posts() ) : the_post();
	echo '<h2>';
	the_title();
	echo '</h2>';
	the_content();
endwhile;
?>

Note[edit]

  • Utilizza: $wp_query
  • Si veda: WP_Query::the_post()

Log delle modifiche[edit]

  • Da: 1.5.0

File sorgente[edit]

the_post() si trova in wp-includes/query.php.

Correlati[edit]

Query Tags: WP_Query (Classe), get_query_var(), query_posts(), have posts(), the_post(), rewind_posts(), wp_reset_postdata(), wp_reset_query()
