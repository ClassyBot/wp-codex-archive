Descrizione[edit]

Imposta le dimensioni dell'immagine in evidenza (nota anche come miniatura del post) predefinita. Per registrare dimensioni aggiuntive per le immagini evidenza, si deve usare: add_image_size().

Nota: Per abilitare le immagini in evidenza, o miniature del post, il tema corrente deve includere add_theme_support( 'post-thumbnails' ); nel suo file functions.php. Si veda anche Immagini in evidenza.

Utilizzo[edit]

 <?php set_post_thumbnail_size( $width, $height, $crop ); ?> 

Parametri[edit]

$width
(int) (opzionale) La larghezza in pixel della miniatura.
Default: 0
$height
(int) (opzionale) L'altezza in pixel della miniatura.
Default: 0
$crop
(boolean) (opzionale) Indica se ritagliare l'immagine o no. False - Modalità di ritaglio proporzionale; True - Modalità di ritaglio netta.
Default: false

Esempi[edit]

Utilizzo standard[edit]

Da usare nel file functions.php del tema corrente. 

if ( function_exists( 'add_theme_support' ) ) {
	add_theme_support( 'post-thumbnails' );
        set_post_thumbnail_size( 150, 150 );
}

Modalità di ritaglio[edit]

Imposta la dimensione dell'immagine in evidenza ridimensionandola proporzionalmente (ossia, senza distorcerla): 

set_post_thumbnail_size( 50, 50 ); // 50 pixel di larghezza per 50 pixel di altezza, ridimensionata

Imposta la dimensione dell'immagine in evidenza ritagliando l'immagine (o dai lati o dalla cima e dal fondo): 

set_post_thumbnail_size( 50, 50, true ); // 50 pixel di larghezza per 50 pixel di altezza, ritagliata

Log delle modifiche[edit]

  • Da 2.9

File sorgente[edit]

set_post_thumbnail_size() si trova in wp-includes/media.php.

Correlati[edit]
