Descrizione[edit]

Recupera il tipo del post corrente o di un particolare post.

Utilizzo[edit]

<?php echo get_post_type( $post ) ?>

Parametri[edit]

$post
(mixed) (opzionale) Oggetto Post o ID del post. Se null, il post corrente verrà utilizzato.
Default: null

Valori restituiti[edit]

(boolean|string) 
tipo del post o false in caso di fallimento.

Esempi[edit]

Stampa il tipo di post. Questo esempio deve essere inserito nel loop. 

<?php echo 'Il tipo del post è: ' . get_post_type( get_the_ID() ); ?>

Note[edit]

  • Utilizza: get_post() per recuperare il post.

Log delle modifiche[edit]

  • 3.5.0: Il valore predefinito di $post ora è null invece di false, e get_post() viene utilizzata invece di accedere direttamente alla variabile globale $post.
  • Da: 2.1.0

File sorgente[edit]

get_post_type() si trova in wp-includes/post.php.
