Descrizione[edit]

Restituisce una lista con tutti gli ID delle pagine.

Utilizzo[edit]

<?php get_all_page_ids() ?>

Parametri[edit]

  • Nessuno

Valori restituiti[edit]

(array) 
Lista con tutti gli ID delle pagine.

Esempi[edit]

Note[edit]

  • Utilizza: $wpdb

Log delle modifiche[edit]

Da: 2.0.0

File sorgente[edit]

get_all_page_ids() si trova in wp-includes/post.php.
