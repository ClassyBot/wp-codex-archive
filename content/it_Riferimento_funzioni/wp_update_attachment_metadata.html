Descrizione[edit]

Aggiorna i metadati di un allegato.

Utilizzo[edit]

<?php wp_update_attachment_metadata( $post_id, $data ); ?>

Parametri[edit]

$post_id
(integer) (richiesto) ID dell'allegato.
Default: Nessuno
$data
(array) (richiesto) Dati dell'allegato.
Default: Nessuno

Valori restituiti[edit]

(boolean) 
Restituisce un valore da update_post_meta().

Note[edit]

  • Utilizza: apply_filters() per aggiungere wp_update_attachment_metadata() su $data e $post_id.

Log delle modifiche[edit]

Da: 2.1.0

File sorgente[edit]

wp_update_attachment_metadata() si trova in wp-includes/post.php.
