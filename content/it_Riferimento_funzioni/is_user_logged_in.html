Descrizione[edit]

Questo tag condizionale controlla se il visitatore corrente ha effettuato il login. È una funzione booleana, ossia restituisce TRUE o FALSE.

Utilizzo[edit]

 <?php if ( is_user_logged_in() ) { ... } ?> 

Parametri[edit]

Questa funzione non accetta parametri.

Valori restituiti[edit]

(boolean) 
TRUE se l’utente è autenticato, altrimenti FALSE.

Esempi[edit]

Stampa differenti output a seconda che l’utente sia autenticato oppure no.

<?php
if ( is_user_logged_in() ) {
    echo 'Benvenuto, utente registrato!';
} else {
    echo 'Benvenuto, visitatore!';
}
?>

File sorgente[edit]
