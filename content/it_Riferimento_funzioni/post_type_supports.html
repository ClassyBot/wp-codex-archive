Descrizione[edit]

Controlla se il tipo di post fornisce il supporto ad una data funzionalità.

Utilizzo[edit]

 <?php post_type_supports( $post_type, $supports ); ?> 

Parametri[edit]

$post_type
(string) (richiesto) Tipo di post. (max. 20 caratteri)
Default: Nessuno
$supports
(string) (richiesto) Funzionalità da rendere disponibili
Default: Nessuno
  • 'title'
  • 'editor' (contenuto)
  • 'author'
  • 'thumbnail' (immagine in evidenza) (il tema corrente deve supportare anche le Immagini in evidenza)
  • 'excerpt'
  • 'trackbacks'
  • 'custom-fields'
  • 'comments' (mostra anche il fumetto con il conteggio dei commenti sulla pagina di modifica)
  • 'revisions' (memorizza le revisioni)
  • 'page-attributes' (template e ordine delle voci di menu) (hierarchical deve essere true)
  • 'post-formats'

Valori restituiti[edit]

(boolean) 
True in caso di successo, false in caso di fallimento.

Log delle modifiche[edit]

  • Da 3.0

File sorgente[edit]

post_type_supports() si trova in wp-includes/post.php.
