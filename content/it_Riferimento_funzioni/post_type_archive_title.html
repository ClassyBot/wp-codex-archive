Descrizione[edit]

Stampa o recupera il titolo per l'archivio di un tipo di post.

Questa funzione è ottimizzata per i file template archive.php e archive-{posttype}.php per stampare il titolo del tipo di post.

Utilizzo[edit]

<?php post_type_archive_title( $prefix, $display ); ?>

Parametri[edit]

$prefix
(string) (opzionale) Cosa mostrare prima del titolo
Default: Nessuno

Nota: L'argomento $prefix è attualmente ignorato se l'argomento $display è false.
Si veda http://core.trac.wordpress.org/ticket/16632

$display
(bool) (opzionale) Indica se stampare o recuperare il titolo per altro uso
Default: true

Valori restituiti[edit]

(string) 
Il titolo quando si richiede di recuperarlo, null quando si richiede di stamparlo o in caso di fallimento.

Esempi[edit]

Stampa il titolo del tipo di post personalizzato: 

<h1><?php post_type_archive_title(); ?></h1>

Note[edit]

Log delle modifiche[edit]

  • Da: 3.1.0

File sorgente[edit]

post_type_archive_title() si trova in wp-includes/general-template.php.
