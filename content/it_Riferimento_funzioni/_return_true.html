Descrizione[edit]

Restituisce true.

Utile per restituire true ai filtri in modo facile.

Utilizzo[edit]

<?php 
// Quanto segue aggiunge un filtro su `esempio_filtro` che restituisce true
add_filter('esempio_filtro', '__return_true'); 
?>

Valori restituiti[edit]

(boolean) 
true

Log delle modifiche[edit]

Da: 3.0

File sorgente[edit]

__return_true() si trova in wp-includes/functions.php
