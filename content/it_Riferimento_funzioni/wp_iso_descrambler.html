Descrizione[edit]

Converte l'oggetto di un'email in ASCII.

Utilizzo[edit]

<?php wp_iso_descrambler( $string ) ?>

Parametri[edit]

$string
(string) (richiesto) Oggetto dell'email.
Default: Nessuno

Valori restituiti[edit]

(string) 
Stringa convertita in ASCII.

Esempi[edit]

Note[edit]

  • "iso" sembra riferirsi al set di caratteri iso-8859-1.
  • C'è una discussione su WordPress trac in qualche modo correlata riguardo la codifica svedese.

Log delle modifiche[edit]

Da: 1.2.0

File sorgente[edit]

wp_iso_descrambler() si trova in wp-includes/formatting.php.
