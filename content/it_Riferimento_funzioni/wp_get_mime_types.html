Descrizione[edit]

Recupera la lista dei tipi mime e delle estensioni dei file.

Utilizzo[edit]

<?php $mime_types = wp_get_mime_types(); ?>

Parametri[edit]

Nessuno

Valori restituiti[edit]

(array) 
Array con i tipi mime in cui le chiavi sono le espressioni regolari delle estensioni dei file corrispondenti a quei tipi.

Esempi[edit]

Note[edit]

Applica il filtro mime_types per restituire il valore, passando l'array dei tipi mime. Questo filtro deve essere utilizzato per aggiungere tipi, non per rimuoverli. Per rimuovere tipi, utilizzare il filtro upload_mimes.

Log delle modifiche[edit]

  • Da: 3.5.0

File sorgente[edit]

wp_get_mime_types() si trova in wp-includes/functions.php
