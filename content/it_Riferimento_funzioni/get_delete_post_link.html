Descrizione[edit]

Restituisce l'URL per spostare un post nel cestino. Può essere usata sia all'interno sia all'esterno del loop per ogni tipo di post: articoli, pagine, allegati, revisioni.

Utilizzo[edit]

 <?php echo get_delete_post_link( $id ); ?> 

Parametri[edit]

$id
(integer) (opzionale) ID del post
Default: Nessuno
$deprecated
(string) (opzionale) Non usato.
Default:
$force_delete
(bool) (opzionale) Indica se aggirare il cestino e forzare la cancellazione.
Default: false

Valori restituiti[edit]

(string) 
Una stringa contenente il permalink.

Log delle modifiche[edit]

  • Da: 2.9.0

File sorgente[edit]

get_delete_post_link() si trova in wp-includes/link-template.php
