Descrizione[edit]

La funzione wp_get_post_categories() recupera la lista di categorie per un dato post.

Livello di compatibilità per temi e plugin. Rappresenta un livello di astrazione dalla complessità del livello tassonomia.

Utilizzo[edit]

<?php wp_get_post_categories( $post_id, $args ); ?>

Parametri[edit]

$post_id
(integer) (opzionale) ID del post.
Default: 0
$args
(array) (opzionale) Sovrascrive i valori predefinti.
Default: array

Gli $args predefiniti sono: 

$defaults = array('fields' => 'ids');

Valori restituiti[edit]

(array) 
Array contenente la lista con gli ID delle categorie.

Esempi[edit]

L'esempio sottostante mostra come le categorie vengono recuperate e come recuperare informazioni aggiuntive per ogni categoria. 

$post_categories = wp_get_post_categories( $post_id );
$cats = array();
	
foreach($post_categories as $c){
	$cat = get_category( $c );
	$cats[] = array( 'name' => $cat->name, 'slug' => $cat->slug );
}

Note[edit]

  • Utilizza: wp_get_object_terms() Recupera le categorie.

Log delle modifiche[edit]

Da: 2.1.0

File sorgente[edit]

wp_get_post_categories() si trova in wp-includes/post.php.
