Descrizione[edit]

Scorre un array e modifica le chiavi rendendole tutte minuscole.

Utilizzo[edit]

<?php wp_kses_array_lc( $inarray ) ?>

Parametri[edit]

$inarray
(array) (richiesto) Array non filtrato.
Default: Nessuno

Valori restituiti[edit]

(array) 
Array sistemato con tutte le chiavi minuscole.

Esempi[edit]

Note[edit]

  • Questa funzione si aspetta un array multi-dimensionale in input.

Log delle modifiche[edit]

Da: 1.0.0

File sorgente[edit]

wp_kses_array_lc() si trova in wp-includes/kses.php.
