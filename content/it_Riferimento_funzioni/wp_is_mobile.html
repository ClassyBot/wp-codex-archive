Descrizione[edit]

Questa funzione controlla se l'utente sta visitando il sito attraverso un dispositivo mobile. La funzione è booleana, ossia restituisce TRUE o FALSE.

Utilizzo[edit]

<?php wp_is_mobile(); ?>

Parametri[edit]

La funzione non accetta parametri.

Esempi[edit]

<?php if ( wp_is_mobile() ) {
	/* Stampa contenuto specifico per dispositivi mobili */
} ?>

Log delle modifiche[edit]

  • Da: 3.4

File sorgente[edit]

wp_is_mobile() si trova in wp-includes/vars.php.

Si veda anche l'indice di Riferimento funzioni e l'indice di Tag dei template.
