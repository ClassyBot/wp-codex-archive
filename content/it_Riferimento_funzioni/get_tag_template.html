Descrizione[edit]

Restituisce il percorso del template del tag nel tema corrente o in quello genitore.

Per prima cosa recupera il nome del tag corrente, ad esempio 'tag-wordpress.php', poi prova con l'ID del tag, ad esempio 'tag-1.php', e infine ripiega sul template 'tag.php' se quei file non esistono.

Utilizzo[edit]

<?php get_tag_template() ?>

Parametri[edit]

Nessuno.

Valori restituiti[edit]

(string) 

Esempi[edit]

Note[edit]

  • Utilizza: apply_filters() - Richiama 'tag_template' sul percorso del file del template del tag.

Filtri[edit]

  • 'tag_template':
    apply_filters('tag_template', $template)

Log delle modifiche[edit]

Da: 2.3.0

File sorgente[edit]

get_tag_template() si trova in wp-includes/theme.php.

Correlati[edit]

Si veda anche l'indice di Riferimento funzioni e l'indice di Tag dei template.
