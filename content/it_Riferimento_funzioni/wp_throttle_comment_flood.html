Descrizione[edit]

Determina se i commenti devono essere bloccati a causa di un eccessivo invio di commenti.

Utilizzo[edit]

<?php wp_throttle_comment_flood( $block, $time_lastcomment, $time_newcomment ) ?>

Parametri[edit]

$block
(boolean) (richiesto) True se il plugin sta bloccando i commenti.
Default: Nessuno
$time_lastcomment
(integer) (richiesto) Timestamp dell'ultimo commento.
Default: Nessuno
$time_newcomment
(integer) (richiesto) Timestamp del nuovo commento.
Default: Nessuno

Valori restituiti[edit]

(boolean) 
Restituisce true se $block è true o se $block è false e $time_newcomment - $time_lastcomment < 15. Restituisce false altrimenti.
