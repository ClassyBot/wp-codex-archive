Descrizione[edit]

Restituisce l'ID dell'utente corrente.

Utilizzo[edit]

 <?php $user_ID = get_current_user_id(); ?> 

Valori restituiti[edit]

(integer) 
L'ID dell'utente, se c'è un utente corrente; altrimenti 0.

Log delle modifiche[edit]

Da: Version 3.0

File sorgente[edit]

get_current_user_id() si trova in wp-includes/user.php.
