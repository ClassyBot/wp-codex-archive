Descrizione[edit]

Controlla se la variabile passata è un'istanza della classe WP_Error o se si tratta di un altro oggetto PHP.

Utilizzo[edit]

<?php is_wp_error( $thing ); ?>

Parametri[edit]

$thing
(mixed) (richiesto) Qualunque variabile esistente di un tipo conosciuto o sconosciuto.
Default: Nessuno

Valori restituiti[edit]

(boolean) 
True se si tratta di un'istanza della classe WP_Error, false altrimenti.

Esempi[edit]

if ( is_wp_error( $result ) ) {
   $error_string = $result->get_error_message();
   echo '<div id="message" class="error"><p>' . $error_string . '</p></div>';
}

Note[edit]

Log delle modifiche[edit]

Da: 2.1.0

File sorgente[edit]

is_wp_error() si trova in wp-includes/class-wp-error.php.
