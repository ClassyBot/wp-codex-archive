Descrizione[edit]

Controlla se un utente ha una capacità o un ruolo. Simile a current_user_can(), ma questa funzione richiede l'ID di un utente o un oggetto come primo parametro.

Utilizzo[edit]

 <?php user_can( $user, $capability ); ?> 

Parametri[edit]

$user
(integer/object) (richiesto) ID dell'utente oppure oggetto utente.
Default: Nessuno
$capability
(string) (richiesto) Nome della capacità o del ruolo.
Default: Nessuno

Valori restituiti[edit]

(boolean) 
True se l'utente ha la capacità o il ruolo, false altrimenti.

Log delle modifiche[edit]

Da: Version 3.1

File sorgente[edit]

user_can() si trova in wp-includes/capabilities.php.

Correlati[edit]

User Capabilities: user_can(), current_user_can(), current_user_can_for_blog()
