Descrizione[edit]

Stampa un link HTML che punta ad un file o una pagina allegato, contenente alternativamente:

  1. un'immagine in dimensione originale o in miniatura nel caso degli allegati di tipo immagine; oppure
  2. il titolo dell'allegato (sotto forma di testo) per gli allegati che non sono immagini

Se non viene trovato alcun allegato, la funzione mosta la stringa Allegato mancante.

Utilizzo[edit]

<?php the_attachment_link( $id, $fullsize, $deprecated, $permalink ); ?>

Utilizzo standard[edit]

<?php the_attachment_link( 7 ); ?>

Parametri[edit]

$id
(integer) (opzionale) ID dell'allegato desiderato.
Default: ID del post corrente, quando usato nel loop (deve trattarsi di un loop che mostra solo gli allegati).
$fullsize
(boolean) (opzionale) Per un allegato di tipo immagine, indica se usare l'immagine originale (true) oppure la miniatura generata da WordPress (false). Se impostato a false e la miniatura non viene trovata, l'immagine originale viene usata al suo posto.
Default: false
$max_dims
(array) (Parametro deprecato a partire dalla versione 2.3) Larghezza e altezza massime per l'immagine o l'icona, applicate agli attributi width e height dell'elemento immagine (img) restituito dalla funzione. Il rapporto tra le dimensioni è mantenuto costante.
Default: non utilizzato
$permalink
(boolean) (opzionale) Indica se il collegamento deve essere direttamente all'immagine o al file allegato (Default), oppure alla pagina dell'allegato.
Default: false

Esempi[edit]

<?php

// Mostra un'immagine in dimensione originale collegata al file allegato
the_attachment_link( 4, true );

// Mostra una miniatura collegata alla pagina dell'allegato
the_attachment_link( 4, false, false, true );

?>

Note[edit]

Utilizzare wp_get_attachment_link() se si desidera ottenere solo il link, senza stamparlo.

Log delle modifiche[edit]

Da: 2.0.0

File sorgente[edit]

the_attachment_link() si trova in wp-includes/post-template.php.
