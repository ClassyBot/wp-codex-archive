Descrizione[edit]

Recupera l'URL della pagina di un autore di cui si passa l'ID.

Utilizzo[edit]

<?php get_author_posts_url( $author_id, $author_nicename ); ?>

Parametri[edit]

$author_id
(integer) (richiesto) ID dell'autore di cui recuperare l'URL.
Default: Nessuno
$author_nicename
(string) (opzionale) Nickname dell'utente.
Default: Nessuno

Valori restituiti[edit]

(string) 
URL della pagina dell'autore.

Esempi[edit]

Stampare il link alla pagina dell'autore del post corrente 

<a href="<?php echo get_author_posts_url( get_the_author_meta( 'ID' ) ); ?>"><?php the_author_meta( 'display_name' ); ?></a>

File sorgente[edit]

get_author_posts_url() si trova in wp-includes/author-template.php
