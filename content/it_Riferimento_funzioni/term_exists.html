Descrizione[edit]

Controlla se un dato termine esiste e restituisce il suo ID, un array di termini o 0 (false) se il termine non esiste.

Utilizzo[edit]

<?php term_exists( $term, $taxonomy, $parent ) ?>

Parametri[edit]

$term
(integer|string) (richiesto) Il termine da controllare
Default: Nessuno
$taxonomy
(string) (opzionale) Il nome della tassonomia da usare
Default: ''
$parent
(integer) (opzionale) ID del termine genitore sotto cui confinare la ricerca di esistenza
Default: 0

Valori restituiti[edit]

(mixed)
  • Restituisce 0 se il termine non esiste.
  • Restituisce l'ID del termine se nessuna tassonomia è stata specificata e il termine esiste.
  • Restituisce un array se la coppia esiste. array('term_id'=>term id, 'term_taxonomy_id'=>taxonomy id)

Esempi[edit]

Controlla se la categoria 'Senza categoria' esiste[edit]

<?php
$term = term_exists('Senza categoria', 'category');
if ($term !== 0 && $term !== null) {
  echo "La categoria 'Senza categoria' esiste!";
}
?>

Controlla se il tag 'Senza tag' esiste[edit]

<?php
$term = term_exists('Senza tag', 'post_tag');
if ($term !== 0 && $term !== null) {
  echo "Il tag 'Senza tag' esiste!";
}
?>

Note[edit]

  • Utilizza la variabile globale: (object) $wpdb
