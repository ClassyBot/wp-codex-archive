Descrizione[edit]

Verifica se è in uso un tema figlio.

Utilizzo[edit]

<?php is_child_theme(); ?>

Valori restituiti[edit]

(bool) 
true se un tema figlio è in uso, false altrimenti.

Log delle modifiche[edit]

Da: 3.0

File sorgente[edit]

is_child_theme() si trova in wp-includes/theme.php

Si veda anche l'indice di Riferimento funzioni e l'indice di Tag dei template.
