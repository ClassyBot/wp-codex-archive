Descrizione[edit]

Recupera i metadati dell'allegato di cui si passa l'ID.

Utilizzo[edit]

<?php wp_get_attachment_metadata( $attachment_id, $unfiltered ); ?>

Parametri[edit]

$attachment_id
(integer) (richiesto) ID dell'allegato.
Default: Nessuno
$unfiltered
(boolean) (opzionale) Se true, i filtri non vengono applicati.
Default: false

Valori restituiti[edit]

(array|boolean) 
Metadati dell'allegato. False in caso di fallimento.

I campi dell'array sono:

width 
(integer) La larghezza dell'allegato
height 
(integer) L'altezza dell'allegato
file 
(string) Il percorso del file relativo a `wp-content/uploads/`
sizes 
(array) Le chiavi sono gli slug delle dimensioni, ogni valore è un array contenente 'file', 'width', 'height' e 'mime-type'
image_meta 
(array)
