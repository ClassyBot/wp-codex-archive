Descrizione[edit]

Recupera lo stato di un commento partendo dal suo ID.

Utilizzo[edit]

<?php wp_get_comment_status( $comment_id ) ?>

Parametri[edit]

$comment_id
(integer) (richiesto) ID del commento.
Default: Nessuno

Valori restituiti[edit]

(string|boolean) 
Lo stato può essere 'deleted', 'approved', 'unapproved', 'spam' oppure false in caso di fallimento.

Esempi[edit]

<?php
$status = wp_get_comment_status( $comment_id );
if ( $status == "approved" ) {
  // mostrare il commento
}
?>

Note[edit]

  • Utilizza: get_comment()

Log delle modifiche[edit]

Da: 1.0.0

File sorgente[edit]

wp_get_comment_status() si trova in wp-includes/comment.php.
