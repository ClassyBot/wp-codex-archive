Descrizione[edit]

Restituisce l'URI completo di un file allegato oppure false in caso di fallimento.


Utilizzo[edit]

<?php wp_get_attachment_url( $id ); ?>

Utilizzo standard[edit]

<?php echo wp_get_attachment_url( 12 ); ?>

Stampa qualcosa come http://example.net/wp-content/uploads/filename

Parametri[edit]

$id
(integer) (richiesto) L'ID dell'allegato desiderato.
Default: Nessuno

wp_get_attachment_image_src

Valori restituiti[edit]

(string/boolean) 
URI dell'allegato caricato oppure "false" in caso di fallimento.

Note[edit]

Si può modificare l'output di questa funzione attraverso il filtro attachment_link.

Se si desidera l'URI per la pagina dell'allegato, e non il vero e proprio file allegato, si può usare get_attachment_link.

Si faccia riferimento anche a: wp_insert_attachment, wp_upload_dir,

Log delle modifiche[edit]

Da: 2.1.0

File sorgente[edit]

wp_get_attachment_url() si trova in wp-includes/post.php.
