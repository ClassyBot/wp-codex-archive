Descrizione[edit]

Questa funzione aggiunge una tassonomia già registrata ad un tipo di post già registrato. Prende come argomento il nome del tipo di oggetto coinvolto. Restituisce true in caso di successo.

Utilizzo[edit]

 <?php register_taxonomy_for_object_type( $taxonomy, $object_type ); ?> 

Parametri[edit]

$taxonomy
(string) (richiesto) Il nome della tassonomia.
Default: Nessuno
$object_type
(string) (richiesto) Il nome del tipo di oggetto da aggiungere alla tassonomia.
Default: Nessuno

Valori restituiti[edit]

(boolean) 
True in caso di successo, false altrimenti.

Esempio[edit]

<?php register_taxonomy_for_object_type( 'category', 'posts' ); ?> 

Log delle modifiche[edit]

  • Da: 3.0.0

File sorgente[edit]

register_taxonomy_for_object_type() si trova in wp-includes/taxonomy.php.
