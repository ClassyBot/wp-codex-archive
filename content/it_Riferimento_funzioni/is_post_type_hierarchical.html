Descrizione[edit]

Questo tag condizionale controlla se il tipo di post è gerarchico. Questa è una funzione booleana, ossia restituisce TRUE o FALSE (Se viene restituito FALSE può anche voler dire che il tipo di post non esiste).

Dapprima controlla se il tipo di post esiste. Poi recupera l'oggetto e infine restituisce il valore 'hierarchical' dell'oggetto.

Utilizzo[edit]

<?php is_post_type_hierarchical( $post_type) ?>

Parametri[edit]

$post_type
(string) (richiesto) Nome del tipo di post
Default: Nessuno

Valori restituiti[edit]

(boolean) 
Se il tipo di post è gerarchico

Esempi[edit]

Note[edit]

  • Utilizza: post_type_exists() Controlla se il tipo di post esiste.
  • Utilizza: get_post_type_object() Utilizzato per recuperare l'oggetto del tipo di post.

Log delle modifiche[edit]

Da: 3.0.0

File sorgente[edit]

is_post_type_hierarchical() si trova in wp-includes/post.php.
