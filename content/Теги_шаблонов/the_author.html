Описание[edit]

Автор записи может выведен с помощью этого тега шаблонов. Этот тег может использоваться в цикле WordPress.

Использование[edit]

 <?php the_author(); ?> 

Примеры[edit]

Вывод публичного имени автора[edit]

Displays the value in the user's Display name publicly as field. 

<p>This post was written by <?php the_author(); ?></p>

Параметры[edit]

Эта функция используется без параметров. (Указывать какие-либо аргументы не рекомендуется.)
