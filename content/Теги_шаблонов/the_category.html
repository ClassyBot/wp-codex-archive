Описание[edit]

Выводит ссылки на рубрики, к которым относится запись. Должна быть внутри цикла Wordpress.

Использование[edit]

 <?php the_category( $separator, $parents, $post_id ); ?> 

Параметры[edit]

separator
(string) (необязательно) Текст или символ, который будет разделять ссылки на рубрики. По умолчанию ссылки отображаются в виде маркированного списка. Если оставить пустую строку — будет тот же результат, что и по умолчанию.
По умолчанию: пустая строка
parents
(string) (необязательно) Способ отображения дочерних рубрик. Варианты:
  • 'multiple' - Display separate links to parent and child categories, exhibiting "parent/child" relationship.
  • 'single' - Display link to child category only, with link text exhibiting "parent/child" relationship.
По умолчанию: пустая строка
Note: Default is a link to the child category, with no relationship exhibited.
post_id
(integer) (необязательно) ID записи, для которой выводится список рубрик.
По умолчанию: false (выводятся рубрики для текущей записи).

Примеры[edit]

Разделение пробелом[edit]

Список рубрик с пробелом в качестве разделителя. 

<p>Рубрики: <?php the_category(' '); ?></p>
Рубрики: WordPress Компьютеры Блогинг


Разделение запятой[edit]

Список рубрик с запятой в качестве разделителя. 

<p>Рубрики: <?php the_category(', '); ?></p>
Рубрики: WordPress, Компьютеры, Блогинг


Разделение стрелкой[edit]

Список рубрик со стрелкой (>) в качестве разделителя. Примечание: Будьте внимательны, используя этот вариант. Некоторые посетители могут интерпретировать рубрики после знака >, как дочерние рубрики. 

<p>Рубрики: <?php the_category(' &gt; '); ?></p>
Рубрики: WordPress > Компьютеры > Блогинг


Разделение круглым маркером[edit]

Список рубрик с круглым маркером (•) в качестве разделителя. 

<p>Рубрики: <?php the_category(' &bull; '); ?></p>
Рубрики: WordPress • Компьютеры • Блогинг


Исходный файл[edit]

  • the_category() объявляется в wp-includes/category-template.php

История изменений[edit]

  • 2.5: добавлен параметр post_id.
