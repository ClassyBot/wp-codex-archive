Описание[edit]

RSS-контейнер для функции bloginfo().

Позволяет получить те же данные, что и при помощи функции get_bloginfo(). При этом данные будут очищены от тегов и выполнена необходимая конвертация символов, когда данные будут извлекаться для использования в RSS-лентах.

Использование[edit]

<?php get_bloginfo_rss( $show ) ?>

Параметры[edit]

$show
(string) (необязательный) Детальная информация о вашем блоге.
Default: 'name'

Допустимые значения:

  • 'name' - Название блога; устанавливается в общих настройках. (По умолчанию)
  • 'description' - Краткое описание блога; устанавливается в общих настройках.
  • 'url' - URL - адрес сайта.
  • 'rdf_url' - URL для RDF/RSS 1.0 - ленты.
  • 'rss_url' - URL для RSS 0.92 - ленты.
  • 'rss2_url' - URL для RSS 2.0 - ленты.
  • 'atom_url' - URL для ленты Atom.
  • 'comments_rss2_url' - URL для комментариев RSS 2.0 - ленты.
  • 'pingback_url' - URL для уведомлений о ссылке на блог (файл XML-RPC).
  • 'admin_email' - email администратора; устанавливается в общих настройках.
  • 'charset' - Кодировка вашего блога; устанавливается в настройках чтения.
  • 'version' - Версия WordPress, используемая вашим блогом.
Следующие значения используются начиная с версии 1.5:
  • 'html_type' - "Content-type" вашего блога.
  • 'wpurl' - URL для установки WordPress.
  • 'template_url' - URL используемого шаблона.
  • 'template_directory' - URL директории шаблона.
  • 'stylesheet_url' - URL главного CSS файла.
  • 'stylesheet_directory' - URL файлов стилей.

Возвращаемое значение[edit]

(string) 

Примеры[edit]

RSS2 URL[edit]

Присваивает URL RSS2-канала вашего блога переменной $rss2_url. 

<?php $rss2_url = get_bloginfo_rss('rss2_url'); ?>

Примечания[edit]

  • Использует: apply_filters() Вызывает 'get_bloginfo_rss' зацепку с двумя параметрами.

История изменений[edit]

Появился, начиная с версии 1.5.1

Исходный код[edit]

Файл get_bloginfo_rss() расположен в wp-includes/feed.php.
