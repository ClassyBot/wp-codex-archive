Описание[edit]

Добавляет шаблон footer.php из директориии, в которой расположены шаблоны текущей темы. Если задан параметр name, то будет добавлен шаблон подвала с именем footer-{name}.php.

Если текущая тема не содержит шаблон footer.php, то будет добавлен подвал из дефолтной темы wp-includes/theme-compat/footer.php.

Использование[edit]

<?php get_footer( $name ); ?>

Параметры[edit]

$name
(string) (не обязательный) Вызов шаблона подвала типа footer-name.php.
Default: None

Примеры[edit]

Простая страница 404 ошибки[edit]

Следующий код - простой пример шаблона для страницы ошибки "HTTP 404: Страница не найдена", который вы можете включить в свою тему как 404.php. 

<?php get_header(); ?>
<h2>Ошибка 404 - Страница не найдена</h2>
<?php get_sidebar(); ?>
<?php get_footer(); ?>

Множественные файлы подвала[edit]

Различные файлы подвала для различных страниц. 

<?php
if ( is_home() ) :
  get_footer('home');
elseif ( is_404() ) :
  get_footer('404');
else :
  get_footer();
endif;
?>

Имена файлов для домашней страницы и страницы ошибки 404 должны быть footer-home.php и footer-404.php соответственно.

Примечания[edit]

  • Используется с: locate_template()
  • Используется с: do_action() Вызывает действие 'get_footer'.

История изменений[edit]

  • Тег добавлен начиная с версии: 1.5.0
  • 2.7.0 : добавлен параметр функции 'name'.

Исходный код[edit]

Код get_footer() находится в файле wp-includes/general-template.php.
