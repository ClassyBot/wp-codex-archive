Описание[edit]

Выводит ID связанной записи. Должен использоваться внутри цикла Wordpress.

Примечание: Эта функция выводит ID записи. Возвращает ID записи другая - get_the_ID().

Использование[edit]

 <?php the_ID(); ?> 

Параметры[edit]

Не имеет параметров.

Примеры[edit]

По умолчанию[edit]

<p>Номер записи: <?php the_ID(); ?></p>

Идентификатор заголовка записи[edit]

Задаёт уникальный идентификатор для заголовка записи: 

<h3 id="post-<?php the_ID(); ?>"><?php the_title(); ?></h3>

Примечание: В XHTML, атрибут id не может начинаться с цифры. Используя для вывода номера записи в атрибут id тег шаблона the_ID, вы должны вставить перед ним хотя бы одну букву, как показано в примере выше.

История изменений[edit]

С версии: 0.71

Исходный файл[edit]

the_ID() объявляется в wp-includes/post-template.php.
