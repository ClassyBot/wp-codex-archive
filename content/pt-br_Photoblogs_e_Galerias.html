Site Abre Aspas com imagens de fundo


O uso de imagens é bastante variado em um site com WordPress, você pode ter imagens para representar os visitantes que comentam em seu site (avatar de usuário), usar temas com imagens no layout, inserir imagens em seu texto, criar galerias em posts ou páginas, usar galerias de outros serviço e criar um site somente com imagens chamados às vezes de photoblogs.

Há muitas opções de como criar um site cujo conteúdo principal são imagens, você pode usar temas criados somente para este propósito, usar plugins que adicionam recursos extras de gerenciamento de galerias ou em alguns casos as duas opções ao mesmo tempo.

  • Consulte também a documentação Imagens no WordPress se estiver em dúvida de como quer usar imagens em seu site WordPress.

PhotoBlogs[edit]

Photoblogs são diferentes dos blogs comuns. Blogs comuns têm mais ênfase em mostrar textos com alguma imagem ocasional. Photoblogs contém apenas fotos.

Galerias[edit]

Galerias são usadas para mostrar várias fotos, seja em uma página ou postagem do Blog. As galerias são ocasionais, ou seja, para blogs comuns que eventualmente querem mostrar fotos.
