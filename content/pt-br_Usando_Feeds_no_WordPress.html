Existem três tipos de usos básico de feed RSS no WordPress:

Widget RSS[edit]

Informe um endereço de feed RSS


O Widget RSS em Aparência > Widgets é usado para mostrar atualizações de outros sites através do feed RSS, ou seja, o site precisa gerar um feed RSS legível.

Além do site de terceiros, você pode informar o endereço do próprio site, por exemplo, de uma categoria em especial, para isso, consulte a documentação Feeds no WordPresspara saber os endereços de feeds que o WordPress gera, há feeds para cada categoria, comentários, tipo de posts etc.

Feed do Site[edit]

Você pode mostrar um gráfico com um link para o feed RSS do seu site WordPress, usando o widget Texto em Aparência > Widgets, este widget permite inserir código, então tudo o que tem a fazer é informar um link para uma imagem e para o feed RSS que quer mostrar: 

<a href="http://seusite/com/feed" ><img src="http://site.com/aimagedofeed.gif" alt="Feed" title="Assine"/></a>

Você pode informar o feed de um categoria só, todo site etc, consulte a documentação Feeds no WordPresspara saber os endereços de feeds que o WordPress gera, há feeds para cada categoria, comentários, tipo de posts etc.

Google FeedBurner[edit]

O serviço Google FeedBurner permite oferecer um formulário de assinatura para os visitantes. Através do feed do seu site, ele ficará sabendo quando publicou algo novo e então o serviço enviará um email para cada assinante automaticamente.

Você pode adicionar o feed geral do seu site ou um em específico consulte a documentação Feeds no WordPresspara saber os endereços de feeds que o WordPress gera.

Você precisa criar uma conta no serviço e então ativar os recursos de assinatura, contador etc, depois é só inserir o código do forumlário oferecido pelo serviço, pode ser em um widget Texto.

Outra forma é usar o plugin Feedburner Form, mas lembre-se que precisa ter uma conta no serviço
