Introdução[edit]

Plugins são ferramentas para estender a funcionalidade do WordPress.

Este artigo contém uma lista de plugins por categoria, e links para outros repositórios de plugins. O núcleo do WordPress é projetado para ser leve, para maximizar a flexibilidade e minimizar o inchaço de código. Plugins oferecem funções personalizadas e funcionalidades para que cada usuário possa personalizar seu site segundo suas necessidades específicas.

Para obter instruções e informações sobre download, instalação, modernização, solução de problemas e gerenciamento de seu WordPress Plugins, verifique a página do plugin no WordPress ou na página do desenvolvedor. Se você quiser desenvolver seu próprio plugin, há uma lista abrangente de recursos em Recursos para Plugins.

Plugin Repositories[edit]

WordPress Plugins estão disponíveis a partir de várias fontes. A fonte "oficial" de Plugins WordPress é:

  • Repositório Oficial de Plugins WordPress

Nem todos os Plugins WordPress estão no repositório oficial. Tente pesquisar na web por "WordPress Plugin" e as palavras-chave para o tipo de funcionalidade que você está procurando.

Plugins Default[edit]

Os seguintes plugins estão incluídos com a distribuição do núcleo WordPress:

Akismet
Akismet verifica seus comentários no serviço Akismet para ver se são spam ou não.
'Olá Dolly'
Quando ativado você vai ver aleatoriamente a letra de "Olá, Dolly" no canto superior direito do seu Painel de Administração em cada página, exceto o painel de Plugins.
