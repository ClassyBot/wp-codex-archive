WordPress 是一個強大的個人發佈平臺，它規劃了許多很棒的功能，讓你體驗作為網路發佈者是多麼極盡簡單、愉悅、魅力十足。我們驕傲地提供給你一個自由分發、符合標準、快速、輕量、免費的個人發佈平臺，擁有合理的預設設定和功能，以及極度客製化的核心。

授權與平台[edit]

  • 授權：WordPress 遵循 GPL 授權。
  • 平臺：需要 PHP（5.2.4 以上）以及 MySQL（5.0.2 以上）。

經營與管理[edit]

本地安裝
WordPress 可以安裝在自己擁有的網站伺服器，或是使用可以讓你完整管理網誌的虛擬主機帳號。與第三方託管服務不同，只要有需要，你能存取並修改網誌上的一切。這代表了，你可以將 WordPress 安裝在自己的桌上型電腦、家用電腦、甚至是網路上。
可攜式核心
在你公開展示的網誌後台，你可以選擇讓 WordPress 相關檔案使用樹狀結構，與網誌放置在同資料夾之下或是不同資料夾。例如，如果你希望網站網址為 http://example.com （public_html - 網站伺服器或虛擬主機帳號的公開"根目錄"），並且希望 WordPress 的相關檔案位於 http://example.com/wordpress （public_html/wordpress），這辦得到！
支援 UTC
WordPress 允許你使用世界標準時間（UTC）的時差來定義自己的時間，因此，所有與時間相關的元素以通用標準的 GMT 值存入資料庫。此外，即使你的伺服器主機位在不同時區，它仍可以讓你的網誌顯示正確的時間。
帳號管理
WordPress 使用帳號等級來控制使用者存取各功能的權限，因此你可以透過建立或改變帳號等級來控制個別帳號修改網誌內容的權限。
帳號個人資料
網誌的每個帳號可以定義自己的個人資料，例如：電子郵件、即時通訊等。使用者也可以控制他們的詳細資料在網誌上的顯示方式。
簡單的安裝與升級
WordPress 著名的五分鐘安裝簡單易用，無人能敵。升級你的網誌到最新版的 WordPress 也同樣簡單，而且需要的時間更少！
動態生成網頁
每當更新網誌或任何方面的更新之後，不需要重新建立網頁。所有的網頁都是每當有觀眾瀏覽頁面時，透過資料庫和範本產生。這代表，更新網誌或設計都是盡可能"快速"，伺服器所需的"儲存空間最小化"。
國際化和當地語系化
你可以建立一個你所選擇語言的網誌與發表內容。WordPress 使用 gettext 全面翻譯與當地語系化。

發表你的作品[edit]

訊息來源（Feed）
WordPress 完整支援 RSS 1.0 (aka RDF)、RSS 2.0、ATOM 格式，而且在網誌的任何頁面都有相關的訊息來源（Feed）可供讀者訂閱——例如：最新文章、分類、迴響，就像我們之前所說的，可以得到任何想要的訊息來源。讀者使用越多種方式保持追蹤你的網誌不同部份，越容易向全世界傳播你的文章。WordPress 也完全支援 RSS 2.0 的 enclosures 元素，因此添加 MP3（就像播客）到 RSS 訊息來源是極其簡單的。
無多餘字串的固定連結 
網誌上所有頁面的連結網址可以讓它遵循一個標準、無多餘字串的系統，讓所有連結具有結構性、有條理，對於人類、機器甚至於搜尋引擎都是易於理解的。清楚的網址對於搜尋引擎優化和改善使用者體驗是不可或缺的。
網誌間的聯繫
在這個相互聯繫日漸緊密的世界，WordPress 早就準備了 PingBack 和 TrackBack 這兩個有效方式來與其他網誌連接，同時他們也可以連接到你的網誌。

客製化設計[edit]

範本驅動設計
WordPress 使用範本來動態生成網頁。你可以使用範本編輯器和範本標籤來編輯範本，從而控制內容的呈現方式。
範本和檔案編輯器
WordPress 安裝後都附有一個檔案編輯器，你可以使用它來編輯範本和其他檔案，這些動作都在瀏覽器中進行，不需要為了編輯而煩惱下載或上傳的問題。
範本標籤
範本標籤 使得設計網誌所顯示的內容與資訊更加容易。你不需要是 PHP 高手就可以設計自己的網誌。
佈景主題
你可以使用現成的佈景主題或風格來包裝你的網誌。也可以創建專屬於你的佈景主題。
外掛
外掛可以擴充網誌的核心功能。許多使用者自己開發的大量外掛可供大眾使用，讓網誌能做到任何你想做的事。

建立內容[edit]

密碼保護
如果你只想分享內容給小部份人，並非所有人，怎麼辦？很簡單，可以在文章上加設密碼。
文章代稱
如果你希望網站上使用乾淨的固定連結，可以使用文章代稱來定義各篇文章的連結名稱。
定時發表
你可以今天寫完一篇文章後，讓文章在未來某一天自動發表。
多頁文章
如果你的文章太長，可以將它分割為多個頁面，你的讀者不用滾動網頁滾到世界末日。
檔案／圖片上傳
你能夠上傳圖片或檔案，並在文章中顯示或連結它們。你也可以針對上傳圖片時產生的縮圖進行設定。
分類
可以將文章分門別類，例如置入某分類，或是子分類、子子分類...
表情符號
WordPress 能夠聰明的將例如 ":)" 的笑臉字元轉換為對應的圖片。
儲存草稿
儲存尚未完成的文章，留待以後完稿之後再發表。
預覽文章
在你按下“發表”按鈕之前，可以先預覽剛剛所寫的文章，檢查有無錯誤。其實，你可以隨時預覽，因為它是“即時的”。
桌面工具
你不一定需要使用瀏覽器來更新網誌，你可以使用任何支援 MetaWeblog 或 Blogger API 的桌面網誌工具。
透過電子郵件寫網誌
你可以使用郵件形式發送文章並發表在你的網誌上。
書籤工具
將 WordPress 提供的 "發表至網誌！" 書籤加入你使用的瀏覽器之後，此瀏覽器就有一個捷徑能快速建立文章並連結到目前瀏覽器的網頁！
側邊欄
如果你不喜歡書籤工具，可以使用我們友好的瀏覽器側邊欄，也有類似的方式。
格式化
把 WordPress 想像成可以讓文字更加流暢、頁面更吸引人的工具。WordPress 內建文字格式化的外掛，能夠整理內容並且加入優美的版式美化你的文章。

彙整與搜尋[edit]

彙整
在你寫網誌一段時間後，如何組織你的網誌顯得非常重要，為此，WordPress 提供了一些現成的彙整顯示方式，囊括了以前所有的文章。你可以從 年、月、周、日、類別、作者 中進行選擇，可以很方便地從網誌主頁（或者任何其他頁面）連結到彙整頁，透過使用範本標籤產生彙整頁的連結。由於 WordPress 動態生成網頁，這些彙整頁不會額外佔用伺服器的任何空間。
搜尋
WordPress 有一個內建的搜尋工具，讓訪客可以針對他們感興趣的項目來進行搜尋，WordPress 的 搜尋關鍵字外掛 可以高亮度標示搜尋字句，能夠更容易找到他們要搜尋的內容。除此之外，這個外掛同樣可以為透過 Google 之類的搜尋引擎進入網誌的訪客提供相同的功能。總而言之，使用 WordPress 搜尋是快樂的。

討論與迴響[edit]

建立社群
WP 不是 YMCA，但是它也透過迴響、Trackback、Pingback 來建立網誌間的聯繫，協助你接觸觀眾並培養友誼。
允許 HTML 標籤
不是所有人都是邪惡的，但是仍要檢查與限制網誌中可能有害的 Html 標籤。WordPress 預設可使用的 Html 標籤是合理選出可以讓人們在迴響中使用的，不需要擔心資料與伺服器的安全。
審查迴響
為了滿足我們的控制欲，WordPress 提供了一系列審查選項。你可以審查：
  • 所有迴響，決定它們是否出現在網誌上
  • 含有特定文字的迴響
  • 由特定 IP 所發表的迴響
  • 內含超過一定數量連結的迴響
所有這些選項能夠將垃圾迴響和搗亂迴響拒之門外。
通知
WordPress 可以每當有新迴響或待審迴響時，使用電子郵件通知你。

建立與管理網誌鏈結[edit]

網誌鏈結（Blogroll）
網誌鏈結是個可以連結到你經常閱讀的網誌的地方——一種推廣優秀網誌的友好方式。WordPress 內建的 連結管理 讓你能毫不費力的添加和管理連結。
書籤工具
一個簡潔的書簽工具能夠讓你輕鬆地新增書籤或我的最愛到瀏覽器裡。新增一個有趣的網誌或網站的連結如同下次前往時點擊書籤或我的最愛按鈕同樣簡單！
歸類
網誌鏈結上的連結可以進行分類並且有條不紊。
匯入
如果你已經擁有一個連結清單的 OPML 檔，可以將它匯入你的 WordPress。
匯出
難道還需要說我們能夠匯出 OPML 檔嗎？
顯示
如同其他功能，你擁有一些簡潔的範本標籤，讓你的網誌可以按照你想要的方式呈現——例如：譬如按字母排序、排名順序、更新日期——一切由你決定。
