Qu'est-ce que phpMyAdmin ?[edit]

Un des outils de l'administrateur, phpMyAdmin est un script PHP destiné à donner aux utilisateurs la possibilité d'interagir avec leurs bases de données MySQL. WordPress stocke toutes ses informations dans la base de données MySQL et interagit avec la base de données pour générer des informations au sein de votre site WordPress. Une vue "brut" des données, des tables et des champs stockés dans la base de données MySQL est accessible via phpMyAdmin .

À quoi sert-il ?[edit]

Le programme phpMyAdmin est pratique pour effectuer des opérations de maintenance sur les tables, la sauvegarde des informations, et la modification des valeurs directement dans le cas où WordPress ne fonctionne pas. Parfois, dans les Support Forums (en anglais), quelqu'un va poster une requête SQL pour une raison ou une autre qui peut être exécutée en utilisant phpMyAdmin. Bien qu'un bon nombre de ces mêmes tâches peuvent être exécutées en ligne de commande MySQL, ce n'est pas une option adoptée par beaucoup de gens.

Où puis-je l'obtenir ?[edit]

Souvent l'accueil des panneaux de contrôle, tels que cPanel et Plesk, ont phpMyAdmin pré-installé ; il n'y a donc rien de spécial à faire pour l'utiliser. Il est généralement liée à partir de la page de base de données. Demandez à votre hébergeur si cette application est disponible.

Vous pouvez télécharger phpMyAdmin vous-même et l'installer à partir de la page principale du projet phpMyAdmin (en anglais).

Avertissement[edit]

Avec un grand pouvoir vient une grande responsabilité. phpMyAdmin permet d'interagir directement avec la base de données : il vous permet également de corrompre la base de données. Il n'y a pas d'"Annuler" ou "Restaurer" dans votre base de données. Soyez toujours prudent lorsque vous travaillez avec la base de données.
