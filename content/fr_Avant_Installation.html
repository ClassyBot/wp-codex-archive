Avant d'installer WordPress[edit]

Vous devez vérifier :

1- Que votre hébergeur remplit les conditions nécessaires (pas de panique, la plupart les remplissent).
Pour une liste des conditions pour votre hébergement, voir Héberger WordPress.
Vous pouvez aussi vérifier sur la page officielle de téléchargement (anglais).

2- Que vous avez les compétences nécessaires (pas de panique, ce n'est pas si compliqué).

Ces deux conditions ne devraient pas être un problème, mais il est plus sage de vérifier avant de commencer.

Avez-vous les outils nécessaires ?[edit]

Navigateurs[edit]

Cela paraît évident, mais il vous faudra un navigateur (Browser en anglais) vous permettant de visualiser votre site et de vous connecter dans l'interface administrateur.
De manière générale nous recommandons plutôt les navigateurs respectueux des standards tels Firefox, Opéra, Safari (Mac), Google Chrome...

Éditeurs de texte[edit]

Vous devez savoir utiliser un éditeur de texte pour modifier les fichiers de configuration principaux. Si vous êtes un utilisateur de Windows, le Bloc-Notes fera l'affaire dans un premier temps. Plus tard vous voudrez probablement modifier les modèles de page (voir Modèles (en anglais) pour les références). Vous pourrez le faire depuis l'interface d'administration de WordPress, mais il est fortement recommandé d'utiliser un bon éditeur de texte. Pour plus d'informations, voir édition de fichiers (en anglais).

Liens :

  • Mac : TextWangler2, Smultron, SubEthaEdit, Taco HTML Edit, ...
  • Windows: ConTEXT, TsWebEditor, Notepad++ ...
  • Linux: Gedit, ...

Clients FTP[edit]

Vous allez avoir besoin d'utiliser un programme de transfert de fichiers FTP pour transférer des fichiers depuis votre ordinateur sur le serveur et éventuellement en changer les autorisations.

Liens :

  • Mac: Cyberduck (libre), RBrowser (freeware), Captain FTP (shareware), Transmit (commercial).
  • Windows : FileZilla, Smart FTP
  • Linux : Gftp


C'est tout ! Maintenant vous êtes prêts pour l'installation.
