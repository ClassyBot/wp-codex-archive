Descrição[edit]

Percorre o index do post no O Loop. Retorna o próximo post, define o post, define a propriedade 'in the loop' para 'true'.

Uso[edit]

<?php the_post(); ?>

Parâmetros[edit]

Esta função não aceita nenhum parâmetro.

Valores Retornados[edit]

Esta função não retorna nenhum valor.

Exemplos[edit]

<?php
if ( have_posts() ) {
	while ( have_posts() ) {

		the_post(); ?>

		<h2><?php the_title(); ?></h2>

		<?php the_content(); ?>

	<?php }
}
?>

Notas[edit]

  • Usa: $wp_query
  • Veja: WP_Query::the_post()

Change Log[edit]

  • Since: 1.5.0

Arquivo de Origem[edit]

the_post() está localizado em wp-includes/query.php.
