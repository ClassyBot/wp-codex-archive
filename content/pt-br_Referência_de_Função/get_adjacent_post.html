Descrição[edit]

Recupera o post adjacente. Pode ser o post anterior ou o próximo post.

Uso[edit]

<?php get_adjacent_post( $in_same_term, $excluded_terms, $previous, $taxonomy ) ?>

Parâmetros[edit]

$in_same_term
(boolean) (opcional) Se o post deve estar no mesmo termo da taxonomia
Default: false
$excluded_terms
(array ou string) (opicional) Array ou lista separada por vírgulas de termos de IDs excluídos.
Default: ' '
$previous
(boolean) (opcional) Se deseja recuperar o post anterior.
Default: true
$taxonomy
(string) (opcional) Nome da taxonomia, se $in_same_term for true.
Default: 'category'

Valor retornado[edit]

  • Objeto do post se bem-sucedido.
  • Null se a global $post não estiver definida.
  • String vazia se não existir post correspondente.

Exemplos[edit]

Link para o post anterior da mesma taxonomia com o título do post como texto. 

 <?php $prev_post = get_adjacent_post( true, '', true, 'taxonomy_slug' ); ?>
 <?php if ( is_a( $prev_post, 'WP_Post' ) ) { ?>
 	<a href="<?php echo get_permalink( $prev_post->ID ); ?>"><?php echo get_the_title( $prev_post->ID ); ?></a>
 <?php } ?>

Link para o próximo post da mesma taxonomia com o título do post como texto. 

 <?php $next_post = get_adjacent_post( true, '', false, 'taxonomy_slug' ); ?>
 <?php if ( is_a( $next_post, 'WP_Post' ) ) {  ?>
 	<a href="<?php echo get_permalink( $next_post->ID ); ?>"><?php echo get_the_title( $next_post->ID ); ?></a>
 <?php } ?>

Notas[edit]

  • Usa global: (objeto) $post
  • Usa global: (objeto) $wpdb

Filtros[edit]

$adjacent é 'anterior' ou 'próximo'.

  • "get_{$adjacent}_post_join":
    $join, $in_same_cat, $excluded_categories
  • "get_{$adjacent}_post_where":
    $wpdb->prepare("WHERE p.post_date $op %s AND p.post_type = %s AND p.post_status = 'publish' $posts_in_ex_cats_sql", $current_post_date, $post->post_type), $in_same_cat, $excluded_categories
  • "get_{$adjacent}_post_sort":
    "ORDER BY p.post_date $order LIMIT 1"
