Представленный в WordPress 2.1.0, API модификации тем состоит из всех функций и хуков, которые являются частью API WordPress, связанные с использованием значений модификации тем. Эти функции могут использоваться авторами тем для сохранения (и извлечения) настроек своих тем для WordPress.

  • get_theme_mods()
  • set_theme_mod()
  • remove_theme_mod()

Хуки
