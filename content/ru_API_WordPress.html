API WordPress расшифровывается как Интерфейс программирования приложений WordPress. Он может быть сгруппирован по разделам, соответствующим разным аспектам API. Каждый раздел покрывает функции, связанные с названной функциональностью (простите за тавтологию). Все они вместе образуют то, что может быть названо API WordPress, который является интерфейсом к плагинам / темам / дополнениям, созданными проектом WordPress.

Если вы прочитали всё это, вы должны иметь хорошее представление, как расширить WordPress посредством его плагинов.
