Mídia → Adicionar Nova[edit]

Você pode fazer upload de arquivos de mídia sem criar um post antes. Isto permite que você faça upload de arquivos para usar depois em posts e páginas e/ou para pegar o web link de um arquivo específico que você pode compartilhar.

Ajuda[edit]

Clique para obter uma breve descrição deste painel.

Painel Eviar Nova Mídia mostrando o Uploader Flash


Enviando Arquivos[edit]

Há duas opções para enviar arquivos: o Uploader Flash e o Upload do Navegador.

Uploader Flash
permite arrastar e soltar vários arquivos de mídia para a janela para enviá-los de uma só vez.
Uploader do navegador
se estiver com problemas para enviar arquivos, use este modo de envio convencional, onde uma janela se abre para escolher o arquivo a se enviar.
Veja o status de envio para cada arquivo



Editando Arquivos de Imagem[edit]

Uma vez enviados os arquivos, há algumas ferramentas para editar imagens, como cortar, redimensionar, inverter etc. Após enviar uma ou mais imagens, clique no link Mostrar na lista de arquivos enviados e então clique no botão Editar Imagem:

Editando imagem - á recursos como cortar, inverter e redimensionar
